VMware服务管理器使用说明
================================

一、项目概述
-----------
这个工具自动管理VMware NAT Service和VMware DHCP Service，实现：
1. 当检测到vmware.exe运行时 → 自动启动服务
2. 当检测到vmware.exe关闭时 → 自动停止服务
3. 减少不必要的性能损耗

二、文件说明
-----------
1. vmware_service_manager.py      - 主监控脚本（Python）
2. vmware_service_launcher.bat    - 启动器（批处理文件）
3. create_vmware_shortcut.ps1     - 快捷方式创建脚本（PowerShell）
4. 一键安装VMware服务管理器.bat  - 一键安装程序

三、安装方法
-----------
方法1：一键安装（推荐）
-----------------------
1. 双击运行 "一键安装VMware服务管理器.bat"
2. 按提示操作，选择Y继续
3. 程序将自动：
   - 创建桌面快捷方式
   - 启动后台监控
   - （可选）设置开机自启动

方法2：手动安装
---------------
1. 创建桌面快捷方式：
   - 运行：python vmware_service_manager.py --install
   - 或运行：create_vmware_shortcut.ps1

2. 启动监控：
   - 后台运行：pythonw.exe vmware_service_manager.py
   - 控制台运行：python.exe vmware_service_manager.py

四、使用方法
-----------
1. 基本使用：
   - 双击桌面 "VMware服务管理器.lnk" 启动后台监控
   - 监控会自动在后台运行，无需人工干预

2. 高级操作（使用启动器）：
   - 运行 "vmware_service_launcher.bat"
   - 选择相应功能：
     1. 启动监控（后台运行）
     2. 启动监控（控制台窗口）
     3. 创建桌面快捷方式
     4. 检查当前状态
     5. 停止所有VMware服务
     6. 退出

五、功能特点
-----------
1. 智能监控：每5秒检查一次VMware状态
2. 自动管理：根据VMware状态自动启停服务
3. 状态显示：实时显示监控状态
4. 低资源占用：后台运行时资源消耗极低
5. 安全可靠：使用系统API管理服务

六、监控的服务
-------------
1. VMware NAT Service    - VMware网络地址转换服务
2. VMnetDHCP             - VMware DHCP服务（内部名称）

七、注意事项
-----------
1. 需要Python环境和psutil库（已自动检查）
2. 需要管理员权限才能管理Windows服务
3. 如果快捷方式创建失败，请以管理员身份运行
4. 要停止后台监控，需要在任务管理器中结束pythonw.exe进程

八、故障排除
-----------
1. 服务无法启动：
   - 检查是否以管理员身份运行
   - 检查VMware是否已正确安装

2. 监控不工作：
   - 检查Python和psutil是否已安装
   - 尝试在控制台模式下运行查看错误信息

3. 快捷方式无法创建：
   - 手动运行create_vmware_shortcut.ps1
   - 或使用vmware_service_launcher.bat中的选项3

九、卸载方法
-----------
1. 停止监控：在任务管理器中结束pythonw.exe进程
2. 删除快捷方式：删除桌面上的"VMware服务管理器.lnk"
3. 删除文件：删除Code_Set目录下的相关文件

十、技术支持
-----------
如有问题，请检查：
1. 确保VMware已正确安装
2. 确保Python环境可用
3. 确保有管理员权限
