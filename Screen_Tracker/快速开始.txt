=== 屏幕使用时长统计工具 - 跨平台快速开始 ===

当前目录：/run/media/tian/软件/VS_Code/Code_Set/Screen_Tracker

【第一步：进入正确目录】
在终端中执行：
cd "/run/media/tian/软件/VS_Code/Code_Set/Screen_Tracker"

【第二步：安装系统工具（Linux）】
# Arch Linux

pip install psutil pillow matplotlib pystray

# 或者使用 pipx（推荐用于应用程序）
pipx install psutil pillow matplotlib pystray

# 也可以使用 --break-system-packages（不推荐，可能破坏系统）

【第四步：运行程序（任选一种方式）】

=== 常见问题 ===

Q: 提示"python: can't open file"
A: 确保在正确目录：/run/media/tian/软件/VS_Code/Code_Set/Screen_Tracker

Q: 系统托盘不显示
A: 程序会在后台运行，数据正常记录。检查桌面环境兼容性。

Q: 如何停止后台程序
A: ps aux | grep ScreenTracker
   kill [进程ID]
   或者：pkill -f ScreenTracker

Q: Linux系统托盘不显示
A: 检查桌面环境：
   - GNOME：安装 libappindicator-gtk3
   - KDE Plasma：通常支持良好
   - XFCE：需要通知守护进程

Q: 程序无法检测活动窗口
A: 检查系统工具：
   - xdotool：用于活动窗口检测
   - xprintidle：用于空闲时间检测
   - wmctrl：用于窗口管理

   sudo systemctl enable screen-tracker.service
   sudo systemctl start screen-tracker.service

【Linux 系统托盘问题解决】

如果系统托盘不显示：

1. 检查桌面环境
   echo $XDG_CURRENT_DESKTOP
   
2. GNOME 用户（重点说明）
   # 安装必要的库
   sudo pacman -S libappindicator-gtk3  # Arch Linux
   sudo apt install gir1.2-appindicator3-0.1  # Ubuntu/Debian
   
   # 启用系统托盘扩展
   # GNOME 40+ 默认不显示系统托盘，需要安装扩展
   sudo pacman -S gnome-shell-extensions  # Arch
   sudo apt install gnome-shell-extensions  # Ubuntu
   
   # 启用 AppIndicator 支持
   gnome-extensions enable appindicatorsupport@rgcjonas.gmail.com
   
   # 或者使用 GNOME Tweaks 工具
   sudo pacman -S gnome-tweaks  # Arch
   sudo apt install gnome-tweaks  # Ubuntu
   # 打开 Tweaks -> Extensions -> 启用 "AppIndicator and KStatusNotifierItem Support"
   
   # 如果使用 GNOME Shell 40+
   # 可能需要安装 TopIcons 或 TopIcons Plus 扩展
   # 从 https://extensions.gnome.org/ 安装
   
   # 重启 GNOME Shell
   # 按 Alt+F2，输入 'r'，然后按 Enter

3. KDE Plasma 用户
   # 通常无需额外配置，如果仍有问题：
   killall plasmashell && kstart5 plasmashell

4. XFCE 用户
   # 确保有通知守护进程
   sudo pacman -S xfce4-notifyd  # Arch
   sudo apt install xfce4-notifyd  # Ubuntu
   # 重启面板
   xfce4-panel --restart

5. 其他桌面环境
   # 程序会在后台运行，不影响数据记录
   # 查看进程：ps aux | grep ScreenTracker
   # 停止进程：pkill -f ScreenTracker
