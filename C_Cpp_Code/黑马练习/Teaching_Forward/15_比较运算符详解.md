# 15 - 比较运算符详解

## 一、什么是比较运算符？

比较运算符用于**比较两个值的大小关系**，比较的结果是一个**布尔值**（`bool`）：
- `true`（真，1）—— 条件成立
- `false`（假，0）—— 条件不成立

## 二、六种比较运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `==` | 等于 | `5 == 5` | `true` |
| `!=` | 不等于 | `5 != 3` | `true` |
| `<` | 小于 | `3 < 5` | `true` |
| `>` | 大于 | `3 > 5` | `false` |
| `<=` | 小于等于 | `5 <= 5` | `true` |
| `>=` | 大于等于 | `3 >= 5` | `false` |

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    cout << "a == b : " << (a == b) << endl;   // 0 (false)
    cout << "a != b : " << (a != b) << endl;   // 1 (true)
    cout << "a <  b : " << (a < b)  << endl;   // 1 (true)
    cout << "a >  b : " << (a > b)  << endl;   // 0 (false)
    cout << "a <= b : " << (a <= b) << endl;   // 1 (true)
    cout << "a >= b : " << (a >= b) << endl;   // 0 (false)

    // 注意：cout 输出 bool 值时，默认显示 0 和 1
    // 想要显示 true/false，可以用 boolalpha
    cout << boolalpha;
    cout << "a == b : " << (a == b) << endl;   // false
    cout << "a != b : " << (a != b) << endl;   // true

    return 0;
}
```

> **重要提醒**：`=` 是赋值，`==` 才是比较！初学者最容易搞混的错误之一。

## 三、比较运算符的结果是 bool 类型

```cpp
bool result1 = (10 > 5);     // true
bool result2 = (10 < 5);     // false
bool result3 = (3 == 3);     // true
bool result4 = (3 != 3);     // false

// 也可以用 bool 变量接收比较结果
int age = 20;
int score = 85;
bool isAdult = (age >= 18);    // true
bool isPass  = (score >= 60);  // true

// bool 值在内存中：true 存为 1，false 存为 0
cout << true << endl;    // 输出 1
cout << false << endl;   // 输出 0

// 非布尔值也可以用于比较，任何非0值转为bool都是true
bool b1 = 42;    // true（非0）
bool b2 = 0;     // false
bool b3 = -1;    // true（非0）
bool b4 = 0.0;   // false
```

## 四、比较运算符在 if 中的使用（最常见的用法）

```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cout << "请输入成绩: ";
    cin >> score;

    // 用比较运算符判断成绩等级
    if (score >= 90) {
        cout << "优秀" << endl;
    } else if (score >= 80) {
        cout << "良好" << endl;
    } else if (score >= 70) {
        cout << "中等" << endl;
    } else if (score >= 60) {
        cout << "及格" << endl;
    } else {
        cout << "不及格" << endl;
    }

    // 也可以用 == 判断特殊情况
    if (score == 100) {
        cout << "满分！太厉害了！" << endl;
    }

    // 用 != 判断不等于的情况
    if (score != 0) {
        cout << "成绩不为零" << endl;
    }

    return 0;
}
```

## 五、浮点数比较的陷阱（非常重要！）

### 5.1 浮点数不能用 == 直接比较

由于浮点数在计算机中是**近似存储**的，直接用 `==` 比较可能得到错误结果。

```cpp
#include <iostream>
using namespace std;

int main() {
    double a = 0.1 + 0.2;
    double b = 0.3;

    // 直接比较：结果可能是 false！
    cout << "a == b : " << (a == b) << endl;
    // 可能输出 0 (false)！因为 0.1 + 0.2 在计算机中不精确等于 0.3

    // 查看实际存储的值（设置高精度输出）
    cout.precision(20);
    cout << "a = " << a << endl;  // 0.30000000000000004441
    cout << "b = " << b << endl;  // 0.29999999999999998890
    // 看到了吧！两个值并不完全相等

    return 0;
}
```

### 5.2 正确做法：用差值比较（epsilon 容差法）

```cpp
#include <iostream>
#include <cmath>    // 用 fabs 求绝对值
using namespace std;

int main() {
    double a = 0.1 + 0.2;
    double b = 0.3;

    // 正确做法：判断两个数的差值是否足够小
    double epsilon = 1e-9;   // 容差值，0.000000001

    if (fabs(a - b) < epsilon) {
        cout << "a 和 b 相等（在容差范围内）" << endl;
    } else {
        cout << "a 和 b 不相等" << endl;
    }
    // 输出：a 和 b 相等（在容差范围内）

    // 封装成函数更方便复用
    // bool isEqual(double x, double y) {
    //     return fabs(x - y) < 1e-9;
    // }

    // 浮点数也不能直接和0比较
    double c = 1.0 / 3.0 * 3.0 - 1.0;  // 理论上应该是0
    // 正确写法
    if (fabs(c) < epsilon) {
        cout << "c 约等于 0" << endl;
    }

    return 0;
}
```

### 5.3 什么时候可以用 == 比较浮点数？

只有在确定值是**精确的整数运算结果**时才可以。

```cpp
double a = 3.0;
double b = 1.0 + 2.0;

// 这种情况下 a == b 是可靠的，因为都是精确值
if (a == b) {
    cout << "相等" << endl;  // 可以正确输出
}
```

## 六、整数和 char 类型的比较完全可靠

整数在计算机中是**精确存储**的，所以可以直接用 `==` 比较。

```cpp
int a = 100;
int b = 50 + 50;
cout << (a == b) << endl;   // 1 (true)，完全正确！
```

`char` 类型本质上是整数（ASCII码），所以也可以比较大小。

```cpp
char ch1 = 'A';   // ASCII码 65
char ch2 = 'B';   // ASCII码 66

cout << (ch1 < ch2) << endl;   // 1 (true)，因为 65 < 66
cout << (ch1 == 'A') << endl;  // 1 (true)

// 判断字符是大写还是小写
char c = 'G';
if (c >= 'A' && c <= 'Z') {
    cout << c << " 是大写字母" << endl;
} else if (c >= 'a' && c <= 'z') {
    cout << c << " 是小写字母" << endl;
}

// 判断字符是否是数字字符
char d = '5';
if (d >= '0' && d <= '9') {
    cout << d << " 是数字字符" << endl;
}
```

## 七、字符串比较

### 7.1 C++ string 类型可以直接比较

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1 = "apple";
    string s2 = "banana";
    string s3 = "apple";

    cout << boolalpha;
    cout << "\"apple\" == \"apple\"  : " << (s1 == s3) << endl;  // true
    cout << "\"apple\" == \"banana\" : " << (s1 == s2) << endl;  // false
    cout << "\"apple\" != \"banana\" : " << (s1 != s2) << endl;  // true

    // 字符串的大小比较是按字典序（逐字符比较ASCII码）
    cout << "\"apple\" < \"banana\"  : " << (s1 < s2) << endl;   // true
    // 'a' 的 ASCII 码 97 < 'b' 的 ASCII 码 98

    cout << "\"abc\" < \"abd\"       : " << ("abc" < "abd") << endl;  // true
    // 前两个字符相同，比较第三个：'c' < 'd'

    cout << "\"abc\" < \"abcd\"      : " << ("abc" < "abcd") << endl; // true
    // 前三个字符相同，但"abc"比"abcd"短

    return 0;
}
```

### 7.2 C风格字符串的比较

```cpp
#include <iostream>
#include <cstring>   // 用于 strcmp
using namespace std;

int main() {
    // C风格字符串（字符数组）不能直接用 == 比较内容！
    char str1[] = "Hello";
    char str2[] = "Hello";

    // 错误！这比较的是地址，不是内容
    // cout << (str1 == str2) << endl;  // 输出0（false），因为地址不同

    // 正确方法：使用 strcmp 函数
    // strcmp 返回值：0表示相等，负数表示s1<s2，正数表示s1>s2
    if (strcmp(str1, str2) == 0) {
        cout << "str1 和 str2 内容相同" << endl;
    }

    return 0;
}
```

## 八、比较运算符的优先级

比较运算符的优先级**低于**算术运算符，所以不用加括号也可以。

```cpp
int a = 5, b = 3, c = 8;

// 算术运算先算，比较运算后算
cout << (a + b > c) << endl;         // 等价于 ((a + b) > c) -> (8 > 8) -> false
cout << (a * 2 == b + 7) << endl;    // 等价于 ((a*2) == (b+7)) -> (10 == 10) -> true

// 但为了可读性，建议加上括号
cout << ((a + b) > c) << endl;
```

注意：`==` 和 `!=` 的优先级低于 `<` `>` `<=` `>=`。

## 九、连续比较的陷阱

```cpp
int a = 5;

// 错误写法：不能像数学中那样连续写比较
// if (1 < a < 10)   // 这不是判断 a 是否在 1~10 之间！
// 实际运算过程：
//   第1步：1 < a  -> 1 < 5 -> true(1)
//   第2步：1 < 10 -> true
// 即使 a = 100，结果仍然是 true！因为 1 < 100 = 1(true)，1 < 10 = true

// 正确写法：用逻辑与 && 连接两个比较
if (a > 1 && a < 10) {
    cout << "a 在 1 到 10 之间" << endl;   // 正确输出
}
```

## 十、signed 和 unsigned 混合比较的陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = -1;
    unsigned int b = 1;

    // 错误的预期：-1 < 1 应该是 true
    // 实际结果：-1 被转为 unsigned int，变成了一个很大的正数
    cout << "-1 < 1u   : " << (a < b) << endl;    // false！（陷阱！）

    // 为什么？因为 -1 转为 unsigned int 后是 4294967295（最大值）
    cout << "(unsigned)-1 = " << (unsigned)a << endl;  // 4294967295

    // 教训：不要把 signed 和 unsigned 混在一起比较！

    return 0;
}
```

## 十一、综合练习

### 练习1：输入两个数，输出较大值

```cpp
#include <iostream>
using namespace std;

int main() {
    double a, b;
    cout << "请输入两个数: ";
    cin >> a >> b;

    if (a > b) {
        cout << "较大值: " << a << endl;
    } else if (a < b) {
        cout << "较大值: " << b << endl;
    } else {
        cout << "两个数相等" << endl;
    }

    return 0;
}
```

### 练习2：判断闰年

```cpp
#include <iostream>
using namespace std;

int main() {
    int year;
    cout << "请输入年份: ";
    cin >> year;

    // 闰年规则：
    // 1. 能被4整除且不能被100整除
    // 2. 或者能被400整除
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        cout << year << " 是闰年" << endl;
    } else {
        cout << year << " 不是闰年" << endl;
    }

    return 0;
}
```

### 练习3：三个数排序

```cpp
#include <iostream>
using namespace std;

int main() {
    int x = 30, y = 10, z = 20;
    int temp;

    // 冒泡排序思想：两两比较交换
    if (x > y) { temp = x; x = y; y = temp; }
    if (x > z) { temp = x; x = z; z = temp; }
    if (y > z) { temp = y; y = z; z = temp; }

    cout << "排序结果: " << x << " " << y << " " << z << endl;
    // 输出：排序结果: 10 20 30

    return 0;
}
```

## 十二、总结

1. 六种比较运算符：`==` `!=` `<` `>` `<=` `>=`
2. 比较结果是 `bool` 类型：`true`(1) 或 `false`(0)
3. 浮点数**不能**用 `==` 直接比较，要用差值法：`fabs(a - b) < epsilon`
4. 整数和 char 类型可以安全使用 `==` 比较
5. C++ string 可以直接用比较运算符；C风格字符串要用 `strcmp`
6. 不要写连续比较 `1 < a < 10`，要用 `a > 1 && a < 10`
7. 注意区分 `=`（赋值）和 `==`（比较）
8. 不要把 signed 和 unsigned 混在一起比较
9. 比较运算符优先级低于算术运算符，高于逻辑运算符
