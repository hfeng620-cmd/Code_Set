# 结构体进阶用法

## 一、结构体数组

当需要存储**多个同类型的结构体**时，可以使用结构体数组。

### 1.1 定义和初始化

```cpp
#include <iostream>
#include <string>
using namespace std;

struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    // 方式一：先定义数组，再逐个赋值
    Student stuArr1[3];
    stuArr1[0] = {"张三", 20, 95.5};
    stuArr1[1] = {"李四", 21, 88.0};
    stuArr1[2] = {"王五", 19, 92.0};

    // 方式二：定义时直接初始化（推荐）
    Student stuArr2[3] = {
        {"张三", 20, 95.5},
        {"李四", 21, 88.0},
        {"王五", 19, 92.0}
    };

    return 0;
}
```

### 1.2 遍历结构体数组

```cpp
#include <iostream>
using namespace std;

struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    Student stuArr[3] = {
        {"张三", 20, 95.5},
        {"李四", 21, 88.0},
        {"王五", 19, 92.0}
    };

    int len = sizeof(stuArr) / sizeof(stuArr[0]);

    cout << "===== 学生信息表 =====" << endl;
    for (int i = 0; i < len; i++)
    {
        cout << "第 " << i + 1 << " 个学生: "
             << stuArr[i].name << ", "
             << stuArr[i].age << " 岁, "
             << stuArr[i].score << " 分" << endl;
    }

    return 0;
}
```

## 二、结构体指针

通过**指针**指向结构体变量，用 **`->`** 运算符访问成员。

### 2.1 基本用法

```cpp
#include <iostream>
using namespace std;

struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    Student s = {"张三", 20, 95.5};

    // 定义结构体指针，指向 s
    Student *p = &s;

    // 通过指针访问成员（用 -> 运算符）
    cout << "姓名: " << p->name << endl;
    cout << "年龄: " << p->age << endl;
    cout << "成绩: " << p->score << endl;

    // 通过指针修改成员
    p->age = 21;
    p->score = 98.0;
    cout << "修改后: " << p->age << ", " << p->score << endl;

    return 0;
}
```

### 2.2 `.` 和 `->` 的区别

```cpp
Student s = {"张三", 20, 95.5};
Student *p = &s;

// .  用于结构体变量（对象本身）
cout << s.name << endl;

// -> 用于结构体指针
cout << p->name << endl;

// 等价关系：p->name 等价于 (*p).name
cout << (*p).name << endl;
```

| 运算符 | 使用场景 | 示例 |
|--------|----------|------|
| `.` | 结构体变量 | `s.name` |
| `->` | 结构体指针 | `p->name` |

## 三、嵌套结构体

结构体的成员可以是**另一个结构体**。

```cpp
#include <iostream>
using namespace std;

struct Address
{
    string city;
    string street;
    int zipCode;
};

struct Student
{
    string name;
    int age;
    Address address;
};

int main()
{
    Student s = {
        "张三", 20,
        {"北京市", "长安街1号", 100000}
    };

    cout << "姓名: " << s.name << endl;
    cout << "城市: " << s.address.city << endl;
    cout << "街道: " << s.address.street << endl;
    cout << "邮编: " << s.address.zipCode << endl;

    s.address.city = "上海市";
    cout << "修改后城市: " << s.address.city << endl;

    return 0;
}
```

## 四、结构体做函数参数

### 4.1 值传递（不改原数据）

```cpp
void printStudent(Student s)
{
    cout << s.name << ", " << s.age << endl;
    s.age = 100;  // 不影响原变量
}
```

### 4.2 地址传递（改原数据）

```cpp
void addAge(Student *s, int years)
{
    s->age += years;
}
```

### 4.3 引用传递（推荐）

```cpp
void printStudent(const Student &s)
{
    cout << s.name << ", " << s.age << endl;
}

void addAge(Student &s, int years)
{
    s.age += years;
}
```

### 4.4 三种传递方式对比

| 方式 | 语法 | 是否修改原变量 | 效率 | 推荐度 |
|------|------|---------------|------|--------|
| 值传递 | `void func(Student s)` | 不能 | 低 | 只读小结构体 |
| 地址传递 | `void func(Student *s)` | 能 | 高 | C 风格 |
| 引用传递 | `void func(Student &s)` | 能 | 高 | 推荐 |
| const引用 | `void func(const Student &s)` | 不能 | 高 | 只读推荐 |

## 五、案例：冒泡排序结构体数组（按年龄）

```cpp
#include <iostream>
using namespace std;

struct Student
{
    string name;
    int age;
    float score;
};

void printStudents(Student arr[], int len)
{
    for (int i = 0; i < len; i++)
    {
        cout << arr[i].name << "	"
             << arr[i].age << " 岁	"
             << arr[i].score << " 分" << endl;
    }
    cout << endl;
}

void bubbleSortByAge(Student arr[], int len)
{
    for (int i = 0; i < len - 1; i++)
    {
        for (int j = 0; j < len - 1 - i; j++)
        {
            if (arr[j].age > arr[j + 1].age)
            {
                Student temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

int main()
{
    Student stuArr[5] = {
        {"孙悟空", 500, 99.0},
        {"猪八戒", 300, 60.0},
        {"沙僧",   280, 75.0},
        {"唐僧",   40,  85.0},
        {"白龙马", 200, 70.0}
    };

    cout << "排序前:" << endl;
    printStudents(stuArr, 5);

    bubbleSortByAge(stuArr, 5);

    cout << "按年龄排序后（从小到大）:" << endl;
    printStudents(stuArr, 5);

    return 0;
}
```

## 六、案例：查找最高分学生

```cpp
#include <iostream>
using namespace std;

struct Student
{
    string name;
    int age;
    float score;
};

Student& findTop(Student arr[], int len)
{
    int maxIndex = 0;
    for (int i = 1; i < len; i++)
    {
        if (arr[i].score > arr[maxIndex].score)
            maxIndex = i;
    }
    return arr[maxIndex];
}

int main()
{
    Student stuArr[3] = {
        {"张三", 20, 95.5},
        {"李四", 21, 88.0},
        {"王五", 19, 99.0}
    };

    Student &top = findTop(stuArr, 3);
    cout << "最高分: " << top.name << ", " << top.score << endl;

    top.score = 100.0;
    cout << "加分后: " << stuArr[2].name << ", " << stuArr[2].score << endl;

    return 0;
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 结构体数组 | 存储多个同类型结构体，初始化列表赋值 |
| 结构体指针 | 用 `->` 访问成员 |
| `.` vs `->` | 变量用 `.`，指针用 `->` |
| 嵌套结构体 | 用多个 `.` 逐层访问 |
| 值传递 | 拷贝整个结构体，不改原数据 |
| 地址传递 | 传指针，高效，能改原数据 |
| 引用传递 | 语法简洁，推荐使用 |
| const 引用 | 只读保护，推荐用于不改数据的函数 |
