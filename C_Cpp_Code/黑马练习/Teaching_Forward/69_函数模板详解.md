# 函数模板详解

## 1. 为什么需要函数模板

在C++中，如果我们想写一个交换两个变量值的函数，针对不同的数据类型需要写多个版本：

```cpp
// 交换两个整数
void swapInt(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

// 交换两个双精度浮点数
void swapDouble(double &a, double &b) {
    double temp = a;
    a = b;
    b = temp;
}

// 交换两个字符
void swapChar(char &a, char &b) {
    char temp = a;
    a = b;
    b = temp;
}
```

**问题**：这些函数的逻辑完全一样，只是类型不同。如果有10种类型就要写10个函数，代码重复度极高。

**解决方案**：使用函数模板，让编译器帮我们自动生成对应类型的函数。

---

## 2. 函数模板的基本语法

```cpp
template<typename T>  // 声明模板，告诉编译器后面代码中T是一个通用类型
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
}
```

### 2.1 template 关键字

`template` 是声明模板的关键字，后面跟一对尖括号 `<>`，里面写模板参数。

### 2.2 typename 和 class 的区别

```cpp
// 以下两种写法完全等价，没有任何区别
template<typename T>
void func1(T a) { }

template<class T>
void func2(T a) { }
```

**注意**：
- `typename` 和 `class` 在模板参数声明中**完全等价**
- 推荐使用 `typename`，语义更清晰（表示"类型名"）
- 你在网上会看到两种写法混用，不要困惑

### 2.3 类型参数 T

```cpp
// T 只是一个名字，可以用任何合法标识符
template<class MyType>
void print(MyType value) {
    cout << value << endl;
}

// 但约定俗成用 T（Type的首字母），不要标新立异
template<class T>
void print(T value) {
    cout << value << endl;
}
```

### 2.4 多个类型参数

```cpp
// 可以有多个类型参数，用逗号分隔
template<class T1, class T2>
void printTwo(T1 a, T2 b) {
    cout << "第一个值: " << a << endl;
    cout << "第二个值: " << b << endl;
}

int main() {
    printTwo(10, 3.14);       // T1=int, T2=double
    printTwo("Hello", 100);   // T1=const char*, T2=int
    return 0;
}
```

## 3. 函数模板的两种调用方式

### 3.1 自动类型推导

```cpp
#include <iostream>
using namespace std;

template<class T>
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
}

int main() {
    int a = 10, b = 20;

    // 自动类型推导：编译器根据实参 a 和 b 的类型，推导出 T = int
    mySwap(a, b);
    cout << "a = " << a << ", b = " << b << endl;  // a = 20, b = 10

    double c = 1.1, d = 2.2;
    mySwap(c, d);  // 推导出 T = double
    cout << "c = " << c << ", d = " << d << endl;  // c = 2.2, d = 1.1

    return 0;
}
```

### 3.2 显式指定类型

```cpp
#include <iostream>
using namespace std;

template<class T>
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
}

int main() {
    int a = 10, b = 20;

    // 显式指定类型：明确告诉编译器 T 是 int
    mySwap<int>(a, b);
    cout << "a = " << a << ", b = " << b << endl;

    return 0;
}
```

## 4. 自动类型推导的限制

### 4.1 必须能推导出一致的类型

```cpp
template<class T>
void myFunc(T a, T b) {
    cout << a << " " << b << endl;
}

int main() {
    myFunc(10, 20);      // OK，T 推导为 int
    myFunc(10, 3.14);    // 错误！第一个参数推导 T=int，第二个推导 T=double，不一致
    // 解决方案1：显式指定类型
    myFunc<double>(10, 3.14);  // OK，T=double，10会被转换为10.0
    // 解决方案2：修改模板参数
    return 0;
}
```

### 4.2 自动推导不能做隐式类型转换

```cpp
template<class T>
void myFunc(T a) {
    cout << a << endl;
}

int main() {
    // 自动推导时，不会进行隐式类型转换
    // myFunc('A');  // T 推导为 char，不会转为 int

    // 但显式指定时可以
    myFunc<int>('A');  // T=int，'A'(65) 被转换为 int
    return 0;
}
```

### 4.3 推导失败的情况

```cpp
template<class T>
void myFunc(T a, T b) {
    cout << a << " " << b << endl;
}

int main() {
    int a = 10;
    // myFunc(a);  // 错误！模板要求两个参数，只传了一个

    // myFunc(a, "hello");  // 错误！int 和 const char* 无法统一推导
    return 0;
}
```

## 5. 显式指定类型时的隐式转换

```cpp
#include <iostream>
using namespace std;

template<class T>
void printValue(T a) {
    cout << "值: " << a << ", 类型大小: " << sizeof(a) << endl;
}

int main() {
    // 自动推导：精确匹配类型
    printValue(10);        // T = int
    printValue(3.14);      // T = double
    printValue('A');       // T = char

    // 显式指定：会进行隐式类型转换
    printValue<int>(3.14);     // T = int, 3.14 转为 3
    printValue<double>('A');   // T = double, 'A' 转为 65.0
    printValue<char>(65);      // T = char, 65 转为 'A'

    return 0;
}
```

## 6. 函数模板与普通函数的调用规则

### 6.1 普通函数优先匹配

```cpp
#include <iostream>
using namespace std;

// 普通函数
int add(int a, int b) {
    cout << "调用普通函数" << endl;
    return a + b;
}

// 函数模板
template<class T>
T add(T a, T b) {
    cout << "调用函数模板" << endl;
    return a + b;
}

int main() {
    int a = 10, b = 20;

    // 普通函数和模板都能匹配时，优先调用普通函数
    add(a, b);  // 输出：调用普通函数

    return 0;
}
```

### 6.2 模板产生更好匹配时调用模板

```cpp
#include <iostream>
using namespace std;

// 普通函数
int add(int a, int b) {
    cout << "调用普通函数" << endl;
    return a + b;
}

// 函数模板
template<class T>
T add(T a, T b) {
    cout << "调用函数模板" << endl;
    return a + b;
}

int main() {
    // 普通函数需要类型转换（char->int），模板直接匹配 char
    // 模板产生更好的匹配，调用模板
    add('A', 'B');  // 输出：调用函数模板

    return 0;
}
```

### 6.3 空模板参数列表强制调用模板

```cpp
#include <iostream>
using namespace std;

// 普通函数
void print(int a) {
    cout << "普通函数: " << a << endl;
}

// 函数模板
template<class T>
void print(T a) {
    cout << "函数模板: " << a << endl;
}

int main() {
    int a = 10;

    print(a);      // 调用普通函数
    print<>(a);    // 空模板参数列表，强制调用函数模板
    print<int>(a); // 显式指定类型，也是调用函数模板

    return 0;
}
```

## 7. 函数模板的重载

函数模板可以有多个版本，通过参数个数不同来重载：

```cpp
#include <iostream>
using namespace std;

// 版本1：两个参数的交换
template<class T>
void mySwap(T &a, T &b) {
    cout << "调用两个参数版本" << endl;
    T temp = a;
    a = b;
    b = temp;
}

// 版本2：三个参数的交换（重载）
template<class T>
void mySwap(T &a, T &b, T &c) {
    cout << "调用三个参数版本" << endl;
    T temp = a;
    a = b;
    b = c;
    c = temp;
}

int main() {
    int a = 1, b = 2, c = 3;

    mySwap(a, b);       // 调用两个参数版本
    mySwap(a, b, c);    // 调用三个参数版本

    cout << "a=" << a << " b=" << b << " c=" << c << endl;
    // 输出：a=2 b=3 c=1

    return 0;
}
```

## 8. 函数模板的局限性

函数模板对于自定义类型（如结构体、类）可能无法直接工作：

```cpp
#include <iostream>
using namespace std;

class Person {
public:
    string name;
    int age;
    Person(string n, int a) : name(n), age(a) {}
};

template<class T>
bool myCompare(T &a, T &b) {
    return a == b;  // 问题：Person 类型没有定义 == 运算符！
}

int main() {
    int x = 10, y = 20;
    cout << myCompare(x, y) << endl;  // OK，int 支持 ==

    Person p1("张三", 20);
    Person p2("张三", 20);
    // cout << myCompare(p1, p2) << endl;  // 错误！Person 没有 == 运算符

    return 0;
}
```

**解决方案**：使用模板的具体化（特化）。

## 9. 模板的具体化（特化）

当模板无法处理某些类型时，可以为特定类型提供专门的实现：

### 9.1 基本语法

```cpp
#include <iostream>
using namespace std;

class Person {
public:
    string name;
    int age;
    Person(string n, int a) : name(n), age(a) {}
};

// 通用版本的函数模板
template<class T>
bool myCompare(T &a, T &b) {
    cout << "调用通用模板" << endl;
    return a == b;
}

// 具体化版本：专门为 Person 类型提供实现
// template<> 表示这是模板的具体化
// 参数类型写 Person& 表示这是针对 Person 类型的特化
template<> bool myCompare(Person &a, Person &b) {
    cout << "调用具体化模板" << endl;
    // 自定义比较逻辑：比较 name 和 age
    return (a.name == b.name) && (a.age == b.age);
}

int main() {
    int x = 10, y = 10;
    cout << myCompare(x, y) << endl;  // 调用通用模板，输出 1

    Person p1("张三", 20);
    Person p2("张三", 20);
    cout << myCompare(p1, p2) << endl;  // 调用具体化模板，输出 1

    Person p3("李四", 25);
    cout << myCompare(p1, p3) << endl;  // 调用具体化模板，输出 0

    return 0;
}
```

### 9.2 多个类型参数的具体化

```cpp
#include <iostream>
using namespace std;

// 通用模板
template<class T1, class T2>
void printPair(T1 a, T2 b) {
    cout << "通用模板: " << a << ", " << b << endl;
}

// 具体化：当两个参数都是 string 时的特殊处理
template<> void printPair(string a, string b) {
    cout << "具体化模板(双string): [" << a << "] 和 [" << b << "]" << endl;
}

int main() {
    printPair(10, 3.14);           // 调用通用模板
    printPair("hello", "world");   // 调用通用模板（const char*）
    printPair(string("hello"), string("world"));  // 调用具体化模板

    return 0;
}
```

## 10. 综合示例

### 示例1：通用排序函数模板

```cpp
#include <iostream>
using namespace std;

// 通用排序函数模板（冒泡排序）
template<class T>
void bubbleSort(T arr[], int len) {
    for (int i = 0; i < len - 1; i++) {
        for (int j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                T temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

// 打印数组的函数模板
template<class T>
void printArray(T arr[], int len) {
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    // 整型数组排序
    int intArr[] = {5, 2, 8, 1, 9, 3};
    int len = sizeof(intArr) / sizeof(intArr[0]);
    bubbleSort(intArr, len);
    cout << "排序后的整型数组: ";
    printArray(intArr, len);

    // 字符数组排序
    char charArr[] = {'d', 'a', 'c', 'b', 'e'};
    len = sizeof(charArr) / sizeof(charArr[0]);
    bubbleSort(charArr, len);
    cout << "排序后的字符数组: ";
    printArray(charArr, len);

    // 双精度数组排序
    double doubleArr[] = {3.14, 1.41, 2.72, 0.58};
    len = sizeof(doubleArr) / sizeof(doubleArr[0]);
    bubbleSort(doubleArr, len);
    cout << "排序后的双精度数组: ";
    printArray(doubleArr, len);

    return 0;
}
```

### 示例2：自定义类型的具体化比较

```cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
public:
    string name;
    int score;

    Student(string n, int s) : name(n), score(s) {}
};

// 通用模板：找最大值
template<class T>
T getMax(T &a, T &b) {
    return (a > b) ? a : b;
}

// 具体化：Student 按分数比较
template<> Student getMax(Student &a, Student &b) {
    return (a.score > b.score) ? a : b;
}

// 通用模板：打印
template<class T>
void printValue(T &val) {
    cout << val << endl;
}

// 具体化：Student 的打印
template<> void printValue(Student &s) {
    cout << "姓名: " << s.name << ", 分数: " << s.score << endl;
}

int main() {
    Student s1("张三", 85);
    Student s2("李四", 92);

    Student best = getMax(s1, s2);
    cout << "分数较高的学生: ";
    printValue(best);

    return 0;
}
```

## 11. 常见错误和注意事项

### 错误1：忘记 template 声明

```cpp
// 错误写法
void mySwap(T &a, T &b) {  // 编译错误：T 未定义
    T temp = a;
    a = b;
    b = temp;
}

// 正确写法
template<class T>
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
}
```

### 错误2：自动推导时类型不一致

```cpp
template<class T>
void func(T a, T b) { }

int main() {
    // func(10, 3.14);  // 错误！T 无法同时是 int 和 double

    // 解决方案
    func<double>(10, 3.14);  // 显式指定 T = double
    return 0;
}
```

### 错误3：模板定义放在 .cpp 文件中

```
// 函数模板通常要把声明和实现都放在头文件(.h)中
// 因为模板是在编译时实例化的，分开写会导致链接错误
```

### 错误4：忘记具体化的 template<>

```cpp
// 错误：这不是具体化，而是函数重载
bool myCompare(Person &a, Person &b) {
    return a.name == b.name;
}

// 正确：具体化必须有 template<>
template<> bool myCompare(Person &a, Person &b) {
    return a.name == b.name;
}
```

## 12. 总结

| 概念 | 说明 |
|------|------|
| template | 声明模板的关键字 |
| typename/class | 声明类型参数，两者等价 |
| T | 类型参数名，可以是任意标识符 |
| 自动类型推导 | 编译器根据实参推导 T 的类型 |
| 显式指定 | func<Type>(args)，明确指定 T 的类型 |
| 具体化/特化 | template<> 为特定类型提供专门实现 |
| 普通函数优先 | 普通函数和模板都能匹配时，优先调用普通函数 |
| 空参数列表 | func<>(args) 强制调用模板版本 |

**核心思想**：函数模板让我们写出与类型无关的通用代码，编译器在需要时自动生成特定类型的函数，大大减少代码重复。
