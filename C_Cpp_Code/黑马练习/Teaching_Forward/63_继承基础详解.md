# 继承基础详解

## 一、继承的概念

继承是面向对象三大特性之一（封装、继承、多态）。

**继承的核心目的：代码复用。** 子类继承父类后，自动拥有父类的成员变量和成员函数，
不需要重复编写，只需要在子类中添加新的功能即可。

```cpp
// 举个例子：
// 猫和狗都是动物，它们有共同的属性（名字、年龄）和行为（吃饭、睡觉）
// 我们可以把共同部分放在 Animal 类中，Cat 和 Dog 继承 Animal
// 这样就不需要在 Cat 和 Dog 中重复写相同的代码
```

---

## 二、继承的基本语法

```cpp
class 子类名 : 继承方式 父类名 {
    // 子类新增的成员
};
```

**继承方式**有三种：`public`、`protected`、`private`，最常用的是 `public`。

```cpp
#include <iostream>
#include <string>
using namespace std;

// 父类（基类）
class Animal {
public:
    string m_Name;
    int m_Age;

    void eat() {
        cout << m_Name << " 正在吃饭" << endl;
    }

    void sleep() {
        cout << m_Name << " 正在睡觉" << endl;
    }
};

// 子类（派生类）继承父类
class Cat : public Animal {
public:
    // 子类可以新增自己的成员
    void climb() {
        cout << m_Name << " 正在爬树" << endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        cout << m_Name << " 正在汪汪叫" << endl;
    }
};

int main() {
    Cat cat;
    cat.m_Name = "小花";       // 继承自 Animal 的成员
    cat.m_Age = 2;             // 继承自 Animal 的成员
    cat.eat();                  // 继承自 Animal 的函数
    cat.sleep();                // 继承自 Animal 的函数
    cat.climb();                // Cat 自己新增的函数

    cout << endl;

    Dog dog;
    dog.m_Name = "大黄";
    dog.m_Age = 3;
    dog.eat();
    dog.bark();

    return 0;
}
```

输出：
```
小花 正在吃饭
小花 正在睡觉
小花 正在爬树

大黄 正在吃饭
大黄 正在汪汪叫
```

## 三、三种继承方式

### 3.1 继承方式对成员访问权限的影响

继承方式会改变父类成员在子类中的访问权限：

| 父类成员权限 | public继承 | protected继承 | private继承 |
|-------------|-----------|--------------|------------|
| public      | public    | protected    | private    |
| protected   | protected | protected    | private    |
| private     | 不可访问   | 不可访问      | 不可访问    |

**记忆口诀：** 继承后的权限 = min(父类成员权限, 继承方式)

### 3.2 public 继承（最常用）

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    int m_Public;       // 公有成员
protected:
    int m_Protected;    // 保护成员
private:
    int m_Private;      // 私有成员

public:
    Father() : m_Public(10), m_Protected(20), m_Private(30) {}
};

class Son : public Father {
public:
    void show() {
        cout << m_Public << endl;       // OK：public继承后仍为public
        cout << m_Protected << endl;    // OK：public继承后仍为protected
        // cout << m_Private << endl;   // 错误：private成员永远不能被子类访问
    }
};

int main() {
    Son son;
    cout << son.m_Public << endl;       // OK：外部可以访问public成员
    // cout << son.m_Protected << endl; // 错误：protected成员外部不能访问
    // cout << son.m_Private << endl;   // 错误：private成员外部不能访问

    return 0;
}
```

### 3.3 protected 继承

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    int m_Public;
protected:
    int m_Protected;
private:
    int m_Private;

public:
    Father() : m_Public(10), m_Protected(20), m_Private(30) {}
};

class Son : protected Father {
public:
    void show() {
        cout << m_Public << endl;       // OK：protected继承后变为protected
        cout << m_Protected << endl;    // OK：protected继承后仍为protected
        // cout << m_Private << endl;   // 错误：private成员不可访问
    }
};

int main() {
    Son son;
    // cout << son.m_Public << endl;       // 错误：protected继承后变为protected，外部不可访问
    // cout << son.m_Protected << endl;    // 错误：protected成员外部不可访问

    return 0;
}
```

### 3.4 private 继承

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    int m_Public;
protected:
    int m_Protected;
private:
    int m_Private;

public:
    Father() : m_Public(10), m_Protected(20), m_Private(30) {}
};

class Son : private Father {
public:
    void show() {
        cout << m_Public << endl;       // OK：private继承后变为private
        cout << m_Protected << endl;    // OK：private继承后变为private
        // cout << m_Private << endl;   // 错误：private成员不可访问
    }
};

// 如果 Son 的子类还想访问 Father 的成员，已经不行了
// 因为 private 继承会把所有成员变为 private，断绝了继续传递的可能
class GrandSon : public Son {
public:
    void show() {
        // cout << m_Public << endl;     // 错误：在Son中已经是private了
        // cout << m_Protected << endl;  // 错误：在Son中已经是private了
    }
};

int main() {
    Son son;
    // cout << son.m_Public << endl;     // 错误：private成员外部不可访问

    return 0;
}
```

### 3.5 三种继承方式对比

```cpp
/*
    实际开发建议：
    1. 几乎只用 public 继承
    2. protected 和 private 继承极少使用
    3. public 继承体现了 "is-a" 关系（猫 is a 动物）
*/

// public 继承：子类对象可以当作父类对象使用
// protected 继承：子类内部可以用，但不能把子类当父类用
// private 继承：完全断绝与外界的关系
```

## 四、继承中的构造和析构顺序

### 4.1 构造顺序：先父后子

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    Father() {
        cout << "Father 的构造函数调用" << endl;
    }
};

class Son : public Father {
public:
    Son() {
        cout << "Son 的构造函数调用" << endl;
    }
};

int main() {
    Son s;
    return 0;
}
```

输出：
```
Father 的构造函数调用
Son 的构造函数调用
```

**原因：** 子类继承了父类的成员，必须先构造父类来初始化继承来的成员，
然后再构造子类自己新增的成员。

### 4.2 析构顺序：先子后父

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    Father() {
        cout << "Father 的构造函数调用" << endl;
    }
    ~Father() {
        cout << "Father 的析构函数调用" << endl;
    }
};

class Son : public Father {
public:
    Son() {
        cout << "Son 的构造函数调用" << endl;
    }
    ~Son() {
        cout << "Son 的析构函数调用" << endl;
    }
};

int main() {
    Son s;
    return 0;
}
```

输出：
```
Father 的构造函数调用
Son 的构造函数调用
Son 的析构函数调用
Father 的析构函数调用
```

**析构顺序与构造顺序相反。** 就像穿衣服和脱衣服：先穿内衣再穿外套，脱的时候先脱外套再脱内衣。

### 4.3 多层继承的构造析构顺序

```cpp
#include <iostream>
using namespace std;

class A {
public:
    A() { cout << "A 构造" << endl; }
    ~A() { cout << "A 析构" << endl; }
};

class B : public A {
public:
    B() { cout << "B 构造" << endl; }
    ~B() { cout << "B 析构" << endl; }
};

class C : public B {
public:
    C() { cout << "C 构造" << endl; }
    ~C() { cout << "C 析构" << endl; }
};

int main() {
    C c;
    return 0;
}
```

输出：
```
A 构造
B 构造
C 构造
C 析构
B 析构
A 析构
```

构造：从最顶层父类开始，从上到下。
析构：从最底层子类开始，从下到上。

### 4.4 子类构造函数调用父类构造函数

```cpp
#include <iostream>
#include <string>
using namespace std;

class Father {
public:
    string m_Name;
    int m_Age;

    // 父类的有参构造
    Father(string name, int age) : m_Name(name), m_Age(age) {
        cout << "Father 有参构造" << endl;
    }
};

class Son : public Father {
public:
    int m_Score;

    // 子类构造函数通过初始化列表调用父类构造函数
    Son(string name, int age, int score) : Father(name, age), m_Score(score) {
        cout << "Son 有参构造" << endl;
    }

    void show() {
        cout << "姓名:" << m_Name << " 年龄:" << m_Age << " 分数:" << m_Score << endl;
    }
};

int main() {
    Son s("张三", 20, 95);
    s.show();
    // 输出：
    // Father 有参构造
    // Son 有参构造
    // 姓名:张三 年龄:20 分数:95

    return 0;
}
```

## 五、同名成员处理

### 5.1 同名成员变量

当子类和父类有同名成员变量时，子类默认访问的是自己的成员。
要访问父类的同名成员，需要使用作用域分辨符 `父类名::`。

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    int m_A = 100;
};

class Son : public Father {
public:
    int m_A = 200;      // 与父类同名的成员变量

    void show() {
        // 默认访问子类自己的 m_A
        cout << "Son 中的 m_A = " << m_A << endl;             // 200

        // 通过作用域分辨符访问父类的 m_A
        cout << "Father 中的 m_A = " << Father::m_A << endl;  // 100
    }
};

int main() {
    Son s;

    // 外部访问，默认访问子类的
    cout << "子类的 m_A = " << s.m_A << endl;              // 200

    // 通过作用域访问父类的
    cout << "父类的 m_A = " << s.Father::m_A << endl;      // 100

    cout << endl;
    s.show();

    return 0;
}
```

输出：
```
子类的 m_A = 200
父类的 m_A = 100

Son 中的 m_A = 200
Father 中的 m_A = 100
```

### 5.2 同名成员函数

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    void func() {
        cout << "Father 的 func 函数" << endl;
    }

    void func(int a) {
        cout << "Father 的 func(int) 函数，参数 = " << a << endl;
    }
};

class Son : public Father {
public:
    // 子类的同名函数会隐藏父类的所有同名函数（包括重载版本）
    void func() {
        cout << "Son 的 func 函数" << endl;
    }
};

int main() {
    Son s;

    // 调用子类的 func
    s.func();                   // 输出：Son 的 func 函数

    // 调用父类的 func（无参）
    s.Father::func();           // 输出：Father 的 func 函数

    // 调用父类的 func（有参）
    s.Father::func(10);         // 输出：Father 的 func(int) 函数，参数 = 10

    // 注意：s.func(10) 会报错！
    // 因为子类定义了同名 func()，会隐藏父类的所有 func 版本

    return 0;
}
```

**重要概念：** 子类定义了与父类同名的函数，会 **隐藏** 父类中所有同名函数（包括重载版本）。
即使参数不同，也需要通过 `父类名::` 才能访问父类版本。

## 六、同名静态成员处理

### 6.1 同名静态成员变量

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    static int m_A;     // 静态成员变量
};

int Father::m_A = 100;  // 类外初始化

class Son : public Father {
public:
    static int m_A;     // 与父类同名的静态成员变量
};

int Son::m_A = 200;     // 类外初始化

int main() {
    // 方式1：通过对象访问
    Son s;
    cout << "子类 m_A = " << s.m_A << endl;              // 200
    cout << "父类 m_A = " << s.Father::m_A << endl;      // 100

    // 方式2：通过类名访问
    cout << "子类 m_A = " << Son::m_A << endl;            // 200
    cout << "父类 m_A = " << Son::Father::m_A << endl;    // 100
    // Son::Father::m_A 表示：通过Son类访问Father类作用域下的m_A

    return 0;
}
```

### 6.2 同名静态成员函数

```cpp
#include <iostream>
using namespace std;

class Father {
public:
    static void func() {
        cout << "Father 的静态 func" << endl;
    }

    static void func(int a) {
        cout << "Father 的静态 func(int)，参数 = " << a << endl;
    }
};

class Son : public Father {
public:
    // 同名静态函数，隐藏父类的所有同名版本
    static void func() {
        cout << "Son 的静态 func" << endl;
    }
};

int main() {
    // 方式1：通过对象调用
    Son s;
    s.func();                   // Son 的静态 func
    s.Father::func();           // Father 的静态 func
    s.Father::func(10);         // Father 的静态 func(int)，参数 = 10

    // 方式2：通过类名调用
    Son::func();                // Son 的静态 func
    Son::Father::func();        // Father 的静态 func
    Son::Father::func(10);      // Father 的静态 func(int)，参数 = 10

    return 0;
}
```

## 七、继承的实际应用示例

```cpp
#include <iostream>
#include <string>
using namespace std;

// 基类：员工
class Employee {
public:
    string m_Name;
    int m_Id;
    double m_BaseSalary;    // 基本工资

    Employee(string name, int id, double baseSalary)
        : m_Name(name), m_Id(id), m_BaseSalary(baseSalary) {}

    // 计算工资（基类只有基本工资）
    virtual double getSalary() {
        return m_BaseSalary;
    }

    void showInfo() {
        cout << "工号:" << m_Id << " 姓名:" << m_Name
             << " 工资:" << getSalary() << endl;
    }
};

// 派生类：经理（额外有奖金）
class Manager : public Employee {
public:
    double m_Bonus;     // 奖金

    Manager(string name, int id, double baseSalary, double bonus)
        : Employee(name, id, baseSalary), m_Bonus(bonus) {}

    double getSalary() {
        return m_BaseSalary + m_Bonus;
    }
};

// 派生类：技术人员（额外有加班费）
class Technician : public Employee {
public:
    int m_OverTime;         // 加班小时数
    double m_HourlyRate;    // 每小时加班费

    Technician(string name, int id, double baseSalary, int overtime, double rate)
        : Employee(name, id, baseSalary), m_OverTime(overtime), m_HourlyRate(rate) {}

    double getSalary() {
        return m_BaseSalary + m_OverTime * m_HourlyRate;
    }
};

int main() {
    Manager mgr("张经理", 1001, 15000, 5000);
    Technician tech("李工", 2001, 10000, 30, 100);

    mgr.showInfo();     // 工号:1001 姓名:张经理 工资:20000
    tech.showInfo();    // 工号:2001 姓名:李工 工资:13000

    return 0;
}
```

## 八、总结

| 知识点 | 要点 |
|--------|------|
| 继承目的 | 代码复用，减少重复代码 |
| 语法 | `class 子类 : 继承方式 父类` |
| 三种继承方式 | public（最常用）、protected、private |
| 构造顺序 | 先父类后子类 |
| 析构顺序 | 先子类后父类（与构造相反） |
| 同名成员 | 默认访问子类的，用 `父类名::` 访问父类的 |
| 隐藏规则 | 子类同名函数会隐藏父类所有同名重载版本 |
| 静态同名成员 | 也可以用 `类名::父类名::` 方式访问 |
