# 结构体定义与使用

## 一、什么是结构体

在实际编程中，我们经常需要把**不同类型的数据组合在一起**描述一个事物。

比如描述一个学生：有姓名（字符串）、年龄（整数）、成绩（浮点数）。
这些数据类型不同，但它们属于同一个人。

**结构体（struct）** 就是 C++ 中用来创建**自定义数据类型**的工具，
可以把多个不同类型的变量打包成一个整体。

---

## 二、结构体的定义语法

```cpp
struct 结构体名
{
    数据类型 成员1;
    数据类型 成员2;
    数据类型 成员3;
    ...
};  // 注意：这里必须有分号！
```

### 示例：定义一个学生结构体

```cpp
struct Student
{
    string name;   // 姓名
    int age;       // 年龄
    float score;   // 成绩
};
```

定义好之后，`Student` 就是一种新的数据类型，和 `int`、`double` 地位相同。

---

## 三、创建结构体变量的三种方式

### 方式一：先定义结构体，再创建变量（推荐）

```cpp
#include <iostream>
#include <string>
using namespace std;

// 第一步：定义结构体类型
struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    // 第二步：用结构体类型创建变量
    Student s1;

    // 给成员赋值
    s1.name = "张三";
    s1.age = 20;
    s1.score = 95.5;

    cout << "姓名: " << s1.name << endl;
    cout << "年龄: " << s1.age << endl;
    cout << "成绩: " << s1.score << endl;

    return 0;
}
```

### 方式二：定义结构体的同时创建变量

```cpp
struct Student
{
    string name;
    int age;
    float score;
} s1, s2, s3;  // 定义类型的同时创建了三个变量

int main()
{
    s1.name = "张三";
    s1.age = 20;
    s1.score = 95.5;

    s2.name = "李四";
    s2.age = 21;
    s2.score = 88.0;

    return 0;
}
```

### 方式三：用初始化列表创建变量

```cpp
struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    // 创建时直接赋初值（初始化列表）
    Student s1 = {"张三", 20, 95.5};
    Student s2 = {"李四", 21, 88.0};

    cout << s1.name << " " << s1.age << " " << s1.score << endl;
    cout << s2.name << " " << s2.age << " " << s2.score << endl;

    return 0;
}
```

## 四、成员访问运算符 `.`

通过 **点号 `.`** 来访问结构体变量的各个成员。

```cpp
struct Student
{
    string name;
    int age;
    float score;
};

int main()
{
    Student s = {"王五", 19, 92.0};

    // 读取成员
    cout << s.name << endl;   // 王五
    cout << s.age << endl;    // 19
    cout << s.score << endl;  // 92

    // 修改成员
    s.age = 20;
    s.score = 95.0;

    cout << "修改后: " << s.name << " " << s.age << " " << s.score << endl;

    return 0;
}
```

## 五、初始化 vs 赋值

**初始化**和**赋值**是两个不同的概念：

```cpp
struct Point
{
    int x;
    int y;
};

int main()
{
    // 初始化：创建变量的同时给值（只做一次）
    Point p1 = {10, 20};

    // 赋值：创建之后修改值（可以多次做）
    p1.x = 30;
    p1.y = 40;

    // 也可以逐个成员初始化
    Point p2;
    p2.x = 100;   // 这其实也是赋值
    p2.y = 200;

    return 0;
}
```

| 区别 | 初始化 | 赋值 |
|------|--------|------|
| 时机 | 创建变量时 | 创建后任何时候 |
| 次数 | 只有一次 | 可以多次 |
| 语法 | `Student s = {"张三", 20, 95.5};` | `s.age = 21;` |

## 六、typedef 与 struct

### 6.1 C 语言中的 typedef struct

在 C 语言中，定义结构体变量必须写 `struct` 关键字：

```cpp
// C 语言风格（C++ 中也可以用，但不必要）
struct Student
{
    string name;
    int age;
};

// C 中创建变量必须写 struct
struct Student s1;  // 不能省略 struct

// 用 typedef 简化
typedef struct
{
    string name;
    int age;
} Student;           // Student 直接就是类型名了

Student s2;          // 不用写 struct 了
```

### 6.2 C++ 中的区别

```cpp
// C++ 中定义结构体后，可以直接用类型名，不需要 typedef
struct Student
{
    string name;
    int age;
};

// C++ 中两种写法都行
struct Student s1;  // C 风格，可以但没必要
Student s2;         // C++ 风格，推荐

// 总结：C++ 中不需要 typedef struct，直接用 struct 就够了
```

## 七、结构体的大小和内存对齐

### 7.1 为什么 sizeof 不等于成员大小之和？

```cpp
#include <iostream>
using namespace std;

struct S1
{
    char a;    // 1 字节
    int b;     // 4 字节
    char c;    // 1 字节
};

struct S2
{
    int b;     // 4 字节
    char a;    // 1 字节
    char c;    // 1 字节
};

int main()
{
    cout << "sizeof(S1) = " << sizeof(S1) << endl;  // 12（不是 1+4+1=6）
    cout << "sizeof(S2) = " << sizeof(S2) << endl;  // 8（不是 4+1+1=6）

    return 0;
}
```

### 7.2 什么是内存对齐

编译器为了**提高访问效率**，会把成员放在特定的内存地址上：
- 每个成员的起始地址必须是**自身大小的整数倍**
- 整个结构体的大小必须是**最大成员大小的整数倍**

```
S1 的内存布局（假设从地址 0 开始）：
地址 0:  [char a  ] 占 1 字节
地址 1:  [填充    ] 填充 3 字节（让 int 对齐到 4 的倍数）
地址 4:  [int b   ] 占 4 字节
地址 8:  [char c  ] 占 1 字节
地址 9:  [填充    ] 填充 3 字节（让总大小是 4 的倍数）
总共 12 字节

S2 的内存布局：
地址 0:  [int b   ] 占 4 字节
地址 4:  [char a  ] 占 1 字节
地址 5:  [char c  ] 占 1 字节
地址 6:  [填充    ] 填充 2 字节（让总大小是 4 的倍数）
总共 8 字节
```

### 7.3 对齐规则

```cpp
struct Example
{
    char a;     // 1 字节，起始地址必须是 1 的倍数
    double b;   // 8 字节，起始地址必须是 8 的倍数
    int c;      // 4 字节，起始地址必须是 4 的倍数
};

// 大小：1 + 7(填充) + 8 + 4 + 4(填充) = 24
```

### 7.4 减少内存浪费的技巧

```cpp
// 浪费内存：小成员散在中间
struct Wasteful
{
    char a;     // 1
    double b;   // 7填充 + 8
    char c;     // 1
    // + 7填充 = 24
};

// 节省内存：按大小从大到小排列
struct Efficient
{
    double b;   // 8
    char a;     // 1
    char c;     // 1
    // + 6填充 = 16
};
```

**建议：** 定义结构体时，按成员大小**从大到小**排列，可以减少内存浪费。

## 八、综合案例

```cpp
#include <iostream>
#include <string>
using namespace std;

// 定义结构体：图书信息
struct Book
{
    string title;      // 书名
    string author;     // 作者
    double price;      // 价格
    int pages;         // 页数
};

int main()
{
    // 创建一本书并初始化
    Book book1 = {"C++ Primer", "Stanley B. Lippman", 128.0, 1000};

    // 输出信息
    cout << "===== 图书信息 =====" << endl;
    cout << "书名: " << book1.title << endl;
    cout << "作者: " << book1.author << endl;
    cout << "价格: " << book1.price << " 元" << endl;
    cout << "页数: " << book1.pages << " 页" << endl;

    // 修改价格（打折）
    book1.price = book1.price * 0.8;
    cout << "打折后价格: " << book1.price << " 元" << endl;

    // 创建第二本书
    Book book2 = {"深入理解计算机系统", "Randal E. Bryant", 139.0, 1048};

    // 比较两本书的价格
    if (book1.price > book2.price)
    {
        cout << book1.title << " 更贵" << endl;
    }
    else
    {
        cout << book2.title << " 更贵" << endl;
    }

    // 查看结构体大小
    cout << "\nBook 结构体大小: " << sizeof(Book) << " 字节" << endl;

    return 0;
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 结构体作用 | 自定义类型，组合不同数据 |
| 定义语法 | `struct 名称 { 成员列表 };` |
| 创建变量 | 先定义再创建、同时创建、初始化列表 |
| 访问成员 | 用 `.` 运算符 |
| typedef | C 语言用法，C++ 中不需要 |
| 内存对齐 | 编译器自动对齐，大小不是简单相加 |
| 减少浪费 | 按成员大小从大到小排列 |
