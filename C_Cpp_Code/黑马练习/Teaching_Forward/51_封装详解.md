# 封装详解

## 一、封装的意义

封装是面向对象编程的第一大特性，核心思想是：

1. **将属性和行为作为一个整体**，表示生活中的事物
2. **对外隐藏实现细节**，只暴露必要的接口

### 生活中的例子

电视机就是一个"封装"的例子：
- **内部实现**（隐藏）：电路板、芯片、信号处理等
- **外部接口**（暴露）：遥控器上的按钮（开关、换台、调音量）

你不需要知道电视内部怎么工作，只需要会用遥控器就行。

## 二、成员属性私有化（核心设计思想）

### 设计原则

```
成员属性 → 设为 private（私有）
成员函数 → 设为 public（公共接口）
```

通过 public 的 **getter（获取器）** 和 **setter（设置器）** 函数来访问私有属性。

### 为什么这样做？

```cpp
#include <iostream>
#include <string>
using namespace std;

// ========== 不好的写法：属性公开 ==========
class PersonBad {
public:
    string name;
    int age;
};

// ========== 好的写法：属性私有 + 公共接口 ==========
class PersonGood {
private:
    string name;    // 私有属性
    int age;        // 私有属性

public:
    // setter：设置姓名
    void setName(string n) {
        name = n;
    }

    // getter：获取姓名
    string getName() {
        return name;
    }

    // setter：设置年龄（带校验逻辑！）
    void setAge(int a) {
        if (a >= 0 && a <= 150) {
            age = a;          // 合法，设置新值
        } else {
            cout << "年龄不合法！请输入0-150之间的值" << endl;
            age = 0;          // 设置默认值
        }
    }

    // getter：获取年龄
    int getAge() {
        return age;
    }
};

int main() {
    // --- 不好的写法 ---
    PersonBad pb;
    pb.age = -100;     // 没有任何校验！不合理的值也能设置

    // --- 好的写法 ---
    PersonGood pg;
    pg.setAge(-100);   // 会提示：年龄不合法！
    pg.setAge(25);     // 正常设置
    cout << "年龄：" << pg.getAge() << endl;  // 输出：25

    return 0;
}
```

### 好处总结

1. **安全性**：在 setter 中可以加校验逻辑，防止设置非法值
2. **只读属性**：只提供 getter，不提供 setter，就是只读属性
3. **灵活性**：内部实现可以随时修改，不影响外部调用
4. **可维护性**：修改实现时只需要改类内部，不需要改所有调用的地方

## 三、示例：设计一个 Person 类

```cpp
#include <iostream>
#include <string>
using namespace std;

class Person {
private:
    // --- 私有属性 ---
    string m_name;      // 姓名
    int m_age;          // 年龄
    string m_phone;     // 手机号

public:
    // --- 公共接口 ---

    // 设置姓名
    void setName(string name) {
        m_name = name;
    }

    // 获取姓名
    string getName() {
        return m_name;
    }

    // 设置年龄
    void setAge(int age) {
        if (age >= 0 && age <= 150) {
            m_age = age;
        } else {
            cout << "错误：年龄必须在0-150之间！" << endl;
        }
    }

    // 获取年龄
    int getAge() {
        return m_age;
    }

    // 设置手机号（带格式校验）
    void setPhone(string phone) {
        if (phone.length() == 11) {
            m_phone = phone;
        } else {
            cout << "错误：手机号必须是11位！" << endl;
        }
    }

    // 获取手机号（隐藏中间4位，保护隐私）
    string getPhone() {
        // 只显示前3位和后4位，中间用 **** 代替
        return m_phone.substr(0, 3) + "****" + m_phone.substr(7, 4);
    }

    // 显示所有信息
    void showInfo() {
        cout << "===== 个人信息 =====" << endl;
        cout << "姓名：" << m_name << endl;
        cout << "年龄：" << m_age << "岁" << endl;
        cout << "手机：" << getPhone() << endl;  // 调用自己的 getter
        cout << "===================" << endl;
    }
};

int main() {
    Person p;

    // 通过 setter 设置属性
    p.setName("张三");
    p.setAge(25);
    p.setPhone("13812345678");

    // 通过 getter 获取属性
    cout << "姓名：" << p.getName() << endl;

    // 显示完整信息
    p.showInfo();

    // 测试校验功能
    p.setAge(-5);           // 输出：错误：年龄必须在0-150之间！
    p.setPhone("123");      // 输出：错误：手机号必须是11位！

    return 0;
}
```

## 四、示例：设计一个 Cube 类

设计一个立方体类，要求：
- 长、宽、高为私有属性
- 提供设置和获取长宽高的接口
- 提供计算面积的接口
- 提供计算体积的接口
- 提供判断两个立方体是否相等的接口

```cpp
#include <iostream>
using namespace std;

class Cube {
private:
    double m_length;    // 长
    double m_width;     // 宽
    double m_height;    // 高

public:
    // --- setter ---
    void setLength(double l) {
        if (l > 0) m_length = l;
        else cout << "长度必须大于0！" << endl;
    }

    void setWidth(double w) {
        if (w > 0) m_width = w;
        else cout << "宽度必须大于0！" << endl;
    }

    void setHeight(double h) {
        if (h > 0) m_height = h;
        else cout << "高度必须大于0！" << endl;
    }

    // --- getter ---
    double getLength() { return m_length; }
    double getWidth()  { return m_width; }
    double getHeight() { return m_height; }

    // 计算表面积
    double getArea() {
        return 2 * (m_length * m_width + m_width * m_height + m_length * m_height);
    }

    // 计算体积
    double getVolume() {
        return m_length * m_width * m_height;
    }

    // 判断两个立方体是否相同（通过成员函数比较）
    bool isEqual(Cube &c) {
        return (m_length == c.m_length) &&
               (m_width == c.m_width) &&
               (m_height == c.m_height);
    }
};

int main() {
    // 创建第一个立方体
    Cube cube1;
    cube1.setLength(10);
    cube1.setWidth(5);
    cube1.setHeight(3);

    cout << "立方体1：" << endl;
    cout << "  长=" << cube1.getLength() << endl;
    cout << "  宽=" << cube1.getWidth() << endl;
    cout << "  高=" << cube1.getHeight() << endl;
    cout << "  表面积=" << cube1.getArea() << endl;
    cout << "  体积=" << cube1.getVolume() << endl;

    // 创建第二个立方体
    Cube cube2;
    cube2.setLength(10);
    cube2.setWidth(5);
    cube2.setHeight(3);

    cout << "\n立方体2：" << endl;
    cout << "  表面积=" << cube2.getArea() << endl;
    cout << "  体积=" << cube2.getVolume() << endl;

    // 判断两个立方体是否相同
    if (cube1.isEqual(cube2)) {
        cout << "\n两个立方体相同！" << endl;
    } else {
        cout << "\n两个立方体不同！" << endl;
    }

    return 0;
}
```

## 五、示例：设计 Circle 和 Point 类（判断点和圆的关系）

这是一个经典的面向对象设计题，需要设计两个类：

1. **Point 类**：表示一个点（有 x, y 坐标）
2. **Circle 类**：表示一个圆（有圆心和半径）

判断点在圆内、圆上还是圆外。

```cpp
#include <iostream>
#include <cmath>    // 用到 sqrt 和 pow
using namespace std;

// ============ 点类 ============
class Point {
private:
    double m_x;     // x 坐标
    double m_y;     // y 坐标

public:
    // setter
    void setX(double x) { m_x = x; }
    void setY(double y) { m_y = y; }

    // getter
    double getX() { return m_x; }
    double getY() { return m_y; }
};

// ============ 圆类 ============
class Circle {
private:
    double m_radius;    // 半径
    Point m_center;     // 圆心（Point 类型的成员）

public:
    // 设置半径
    void setRadius(double r) {
        if (r > 0) m_radius = r;
        else cout << "半径必须大于0！" << endl;
    }

    // 获取半径
    double getRadius() { return m_radius; }

    // 设置圆心
    void setCenter(Point center) {
        m_center = center;
    }

    // 获取圆心
    Point getCenter() { return m_center; }

    // 判断点和圆的关系
    // 返回值：1=圆内，2=圆上，3=圆外
    int isPointInCircle(Point &p) {
        // 计算点到圆心的距离
        double distance = sqrt(
            pow(p.getX() - m_center.getX(), 2) +
            pow(p.getY() - m_center.getY(), 2)
        );

        // 比较距离和半径
        if (distance < m_radius) {
            return 1;   // 点在圆内
        } else if (distance == m_radius) {
            return 2;   // 点在圆上
        } else {
            return 3;   // 点在圆外
        }
    }
};

int main() {
    // 创建圆
    Circle circle;
    circle.setRadius(5);

    // 设置圆心坐标 (0, 0)
    Point center;
    center.setX(0);
    center.setY(0);
    circle.setCenter(center);

    // 创建要判断的点
    Point p1, p2, p3;
    p1.setX(1);  p1.setY(1);    // 点(1,1) 应该在圆内
    p2.setX(3);  p2.setY(4);    // 点(3,4) 应该在圆上（距离=5）
    p3.setX(10); p3.setY(10);   // 点(10,10) 应该在圆外

    // 判断并输出结果
    int result;

    result = circle.isPointInCircle(p1);
    cout << "点(1,1)：";
    if (result == 1) cout << "在圆内" << endl;
    else if (result == 2) cout << "在圆上" << endl;
    else cout << "在圆外" << endl;

    result = circle.isPointInCircle(p2);
    cout << "点(3,4)：";
    if (result == 1) cout << "在圆内" << endl;
    else if (result == 2) cout << "在圆上" << endl;
    else cout << "在圆外" << endl;

    result = circle.isPointInCircle(p3);
    cout << "点(10,10)：";
    if (result == 1) cout << "在圆内" << endl;
    else if (result == 2) cout << "在圆上" << endl;
    else cout << "在圆外" << endl;

    return 0;
}
```

**输出结果：**
```
点(1,1)：在圆内
点(3,4)：在圆上
点(10,10)：在圆外
```

## 六、封装的实际意义

### 1. 安全性

```cpp
class BankAccount {
private:
    double m_balance;   // 余额（私有，不能随便改）

public:
    // 存款（只能增加，不能直接设置余额）
    void deposit(double amount) {
        if (amount > 0) {
            m_balance += amount;
            cout << "存入" << amount << "元，余额：" << m_balance << "元" << endl;
        }
    }

    // 取款（需要校验余额是否足够）
    bool withdraw(double amount) {
        if (amount > 0 && amount <= m_balance) {
            m_balance -= amount;
            cout << "取出" << amount << "元，余额：" << m_balance << "元" << endl;
            return true;
        } else {
            cout << "取款失败！余额不足或金额不合法" << endl;
            return false;
        }
    }

    // 查询余额（只读）
    double getBalance() {
        return m_balance;
    }
};
```

### 2. 灵活性

```cpp
class Temperature {
private:
    double m_celsius;   // 存储为摄氏度

public:
    // 设置摄氏度
    void setCelsius(double c) {
        m_celsius = c;
    }

    // 设置华氏度（内部转换为摄氏度存储）
    void setFahrenheit(double f) {
        m_celsius = (f - 32) * 5.0 / 9.0;
    }

    // 获取摄氏度
    double getCelsius() {
        return m_celsius;
    }

    // 获取华氏度（从摄氏度转换）
    double getFahrenheit() {
        return m_celsius * 9.0 / 5.0 + 32;
    }
};
```

## 七、常见错误和注意事项

### 错误1：忘记提供 getter/setter

```cpp
class Person {
private:
    int age;
    // 既没有 setter 也没有 getter
    // 那 age 就永远无法被外部访问，等于没用
};

int main() {
    Person p;
    // p.age = 20;         // ❌ private 不能直接访问
    // 没有 setter，也没法设置
}
```

### 错误2：setter 中忘记校验

```cpp
class Person {
private:
    int age;

public:
    void setAge(int a) {
        age = a;    // ❌ 没有校验！可以设置负数
    }

    // 好的写法：
    // void setAge(int a) {
    //     if (a >= 0 && a <= 150) {
    //         age = a;
    //     }
    // }
};
```

### 错误3：属性全部设为 public

```cpp
class Person {
public:
    int age;        // ❌ 不好！任何人都可以随意修改
    string name;
};
```

### 注意事项

1. **属性私有，接口公开**是 C++ 面向对象的基本设计原则
2. setter 中要**加入校验逻辑**，保证数据合法性
3. 只提供 getter 不提供 setter = **只读属性**
4. 封装的好处：**安全、灵活、可维护**
5. 类的内部实现可以随时修改，**不影响外部调用**（只要接口不变）
