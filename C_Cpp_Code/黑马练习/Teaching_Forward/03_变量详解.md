# 03 变量详解

## 一、变量的本质

变量的本质是 **内存中的一块有名字的存储空间**。每个变量都有三个核心属性：

| 属性 | 含义 | 示例 |
|------|------|------|
| **地址** | 变量在内存中的位置（门牌号） | `0x7ffeefbff5ac` |
| **类型** | 决定占用多少空间、能存什么数据 | `int` 占4字节，存整数 |
| **值** | 变量当前存储的数据 | `42` |

```cpp
int age = 18;
// age 是变量名
// int 是类型（决定占4字节）
// 18 是值
// &age 是地址（用 & 取地址符可以查看）
```

```cpp
// 查看变量的地址
int a = 10;
cout << "a 的值: " << a << endl;
cout << "a 的地址: " << &a << endl;
// 输出类似: a 的地址: 0x7ffeefbff5ac
```

## 二、创建变量的语法

```cpp
// 语法：数据类型 变量名;
// 或者：数据类型 变量名 = 初始值;

int age;            // 创建一个整型变量，未初始化（值是随机的）
int score = 100;    // 创建一个整型变量，并初始化为 100
double price = 9.9; // 创建一个浮点型变量
char grade = 'A';   // 创建一个字符型变量
string name = "张三"; // 创建一个字符串变量
```

## 三、变量命名规则（必须遵守）

```cpp
// 规则1：只能由 字母、数字、下划线 组成
int my_age = 18;      // 正确
int _count = 10;      // 正确（可以下划线开头）
int num1 = 100;       // 正确（可以数字结尾）

// 规则2：不能以数字开头
// int 2name = 10;    // 错误！不能以数字开头

// 规则3：不能是 C++ 关键字
// int int = 10;      // 错误！int 是关键字
// int class = 10;    // 错误！class 是关键字
// int return = 10;   // 错误！return 是关键字

// 规则4：区分大小写
int Age = 1;    // Age 和 age 是两个不同的变量
int age = 2;    // 不冲突

// 规则5：不能包含特殊符号（空格、@、#、$ 等）
// int my age = 10;   // 错误！不能有空格
// int my@name = 10;  // 错误！不能有 @
```

## 四、变量命名风格（建议遵守）

```cpp
// 风格1：小驼峰命名法（C++ 最常用）
int studentAge = 18;
int maxScore = 100;
string firstName = "张";

// 风格2：下划线命名法（也常用）
int student_age = 18;
int max_score = 100;
string first_name = "张";

// 风格3：大驼峰命名法（通常用于类名）
class StudentInfo {};    // 类名用大驼峰
struct PointData {};     // 结构体名用大驼峰

// 好的命名：见名知意
int studentCount = 50;     // 一看就知道是学生人数
double averageScore = 85.5; // 一看就知道是平均分

// 不好的命名：看不懂
int a = 50;      // a 是什么？
double x = 85.5; // x 是什么？
```

## 五、初始化 vs 赋值

```cpp
// 初始化：创建变量的同时给值（声明和赋值一步完成）
int age = 18;       // 创建 age 的同时赋值为 18

// 赋值：变量已经存在，修改它的值
age = 20;           // 把 age 的值改为 20

// 多种初始化写法（效果相同）
int a = 10;         // C 风格初始化（最常用）
int b(10);          // C++ 函数式初始化（了解即可）
int c{10};          // C++11 列表初始化（推荐，最安全）
int d = {10};       // 也是列表初始化
```

### 未初始化的危险

```cpp
int x;              // 没有初始化！
cout << x << endl;  // 输出一个随机的垃圾值！程序行为不可预测！

// 正确做法：创建变量时就初始化
int x = 0;          // 安全！
```

## 六、作用域

### 6.1 局部变量

```cpp
int main()
{
    int a = 10;  // a 是 main 函数内的局部变量
    cout << a << endl;  // 正确，a 在作用域内

    // cout << b;  // 错误！b 不在这个作用域内
    return 0;
}

void func()
{
    int b = 20;  // b 是 func 函数内的局部变量
    // cout << a;  // 错误！a 不在这个作用域内
}
```

### 6.2 全局变量

```cpp
#include <iostream>
using namespace std;

int globalVar = 100;  // 全局变量，定义在所有函数外面

int main()
{
    cout << globalVar << endl;  // 正确，全局变量任何地方都能访问
    return 0;
}

void func()
{
    cout << globalVar << endl;  // 正确，也能访问
}
```

### 6.3 同名变量：局部优先

```cpp
int value = 100;  // 全局变量

int main()
{
    int value = 200;  // 局部变量（与全局变量同名）
    cout << value << endl;  // 输出 200（局部优先！）

    // 如果想访问全局变量，用 :: 作用域运算符
    cout << ::value << endl;  // 输出 100

    return 0;
}
```

> 建议：**尽量避免全局变量和局部变量同名**，容易造成混淆。

### 6.4 代码块作用域

```cpp
int main()
{
    int a = 10;

    // 花括号 {} 创建一个新的作用域
    {
        int b = 20;
        cout << a << endl;  // 正确，外层变量在内层可见
        cout << b << endl;  // 正确，b 在自己的作用域内
    }

    cout << a << endl;  // 正确
    // cout << b << endl;  // 错误！b 已经出了作用域，被销毁了
}
```

## 七、生命周期

```cpp
void demo()
{
    int x = 10;     // x 在函数调用时创建
    cout << x << endl;
}                   // x 在函数结束时销毁

// 每次调用 demo()，x 都会重新创建
// demo() 结束后，x 就不存在了
```

```cpp
// 静态局部变量：生命周期延长到程序结束
void counter()
{
    static int count = 0;  // 只在第一次调用时初始化，之后保持值
    count++;
    cout << "调用次数: " << count << endl;
}

int main()
{
    counter();  // 输出: 调用次数: 1
    counter();  // 输出: 调用次数: 2
    counter();  // 输出: 调用次数: 3
    return 0;
}
```

## 八、常见错误

### 8.1 使用未初始化的变量

```cpp
int x;          // 未初始化
cout << x;      // 未定义行为！输出垃圾值
```

### 8.2 变量重复定义

```cpp
int a = 10;
// int a = 20;   // 错误！变量 a 已经定义过了
a = 20;          // 正确：这是赋值，不是重新定义
```

### 8.3 超出作用域使用

```cpp
if (true)
{
    int temp = 100;
}
// cout << temp;  // 错误！temp 已经超出作用域
```

### 8.4 变量名拼写错误

```cpp
int studentAge = 18;
// cout << studnetAge;  // 错误！拼写错了，编译器会报 "未声明的标识符"
cout << studentAge;     // 正确
```

## 九、综合示例

```cpp
#include <iostream>
#include <string>
using namespace std;

// 全局变量（不推荐过多使用）
int totalCount = 0;

int main()
{
    // 声明并初始化各种类型的变量
    string name = "张三";    // 姓名
    int age = 18;             // 年龄
    double score = 95.5;      // 成绩
    char grade = 'A';         // 等级
    bool isPass = true;       // 是否及格

    // 输出所有变量
    cout << "姓名: " << name << endl;
    cout << "年龄: " << age << endl;
    cout << "成绩: " << score << endl;
    cout << "等级: " << grade << endl;
    cout << "及格: " << isPass << endl;

    // 变量可以修改
    age = 19;
    score = 98.0;
    cout << "\n修改后:" << endl;
    cout << "年龄: " << age << endl;
    cout << "成绩: " << score << endl;

    // 查看变量占用的内存大小
    cout << "\n内存占用:" << endl;
    cout << "int:    " << sizeof(int)    << " 字节" << endl;
    cout << "double: " << sizeof(double) << " 字节" << endl;
    cout << "char:   " << sizeof(char)   << " 字节" << endl;
    cout << "bool:   " << sizeof(bool)   << " 字节" << endl;

    return 0;
}
```

## 十、总结

| 要点 | 说明 |
|------|------|
| 本质 | 内存中一块有名字的空间，有地址、类型、值三个属性 |
| 命名规则 | 字母/数字/下划线，不能数字开头，不能是关键字，区分大小写 |
| 初始化 | 创建时就赋值，避免使用未初始化的变量 |
| 作用域 | 局部变量只在 {} 内有效，同名时局部优先 |
| 生命周期 | 局部变量随函数结束销毁，static 变量持续到程序结束 |
