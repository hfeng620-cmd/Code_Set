# 类的定义与访问权限

## 一、面向对象编程（OOP）的概念

面向对象编程是一种编程思想，把数据和操作数据的方法封装在一起，形成一个"对象"。

C++ 面向对象的三大特性：

### 1. 封装
把属性（数据）和行为（函数）包装在一起，对外隐藏实现细节，只暴露必要的接口。

### 2. 继承
子类可以继承父类的属性和行为，实现代码复用。

### 3. 多态
同一个接口，不同的对象有不同的实现方式。

## 二、类的定义语法

```cpp
class 类名 {
    访问权限:
        成员属性;   // 变量
        成员函数;   // 方法
};
```

**示例：定义一个简单的 Person 类**

```cpp
#include <iostream>
#include <string>
using namespace std;

// 定义一个 Person 类
class Person {
public:
    // --- 成员属性（描述人的特征）---
    string name;   // 姓名
    int age;       // 年龄

    // --- 成员函数（描述人的行为）---
    // 自我介绍
    void introduce() {
        // 在类内部可以访问所有成员（public、protected、private）
        cout << "大家好，我叫" << name << "，今年" << age << "岁" << endl;
    }

    // 吃饭
    void eat() {
        cout << name << "正在吃饭" << endl;
    }
};

int main() {
    // 创建对象（实例化）
    Person p1;

    // 给成员属性赋值
    p1.name = "张三";
    p1.age = 20;

    // 调用成员函数
    p1.introduce();  // 输出：大家好，我叫张三，今年20岁
    p1.eat();        // 输出：张三正在吃饭

    return 0;
}
```

## 三、class vs struct 的区别

在 C++ 中，`class` 和 `struct` 都可以定义类，**唯一区别是默认访问权限不同**：

- **class**：默认访问权限是 **private**（私有）
- **struct**：默认访问权限是 **public**（公共）

```cpp
// 用 class 定义
class PersonClass {
    // 不写访问权限，默认是 private
    int age;  // 私有，类外不能直接访问
};

// 用 struct 定义
struct PersonStruct {
    // 不写访问权限，默认是 public
    int age;  // 公共，类外可以直接访问
};

int main() {
    PersonClass pc;
    // pc.age = 20;  // ❌ 错误！class 默认 private，类外不能访问

    PersonStruct ps;
    ps.age = 20;     // ✅ 正确！struct 默认 public，类外可以访问

    return 0;
}
```

**建议：** 在 C++ 中定义类一般用 `class`，因为面向对象强调封装，属性通常设为私有。

## 四、三种访问权限

C++ 有三种访问权限（也叫访问控制符）：

| 访问权限 | 类内 | 类外 | 子类 |
|---------|------|------|------|
| **public**（公共） | 可以访问 | 可以访问 | 可以访问 |
| **protected**（保护） | 可以访问 | 不能访问 | 可以访问 |
| **private**（私有） | 可以访问 | 不能访问 | 不能访问 |

### 4.1 public（公共权限）

```cpp
class Person {
public:
    string name;   // 公共成员属性
    int age;       // 公共成员属性

    // 公共成员函数
    void show() {
        // 类内可以访问 public 成员
        cout << "姓名：" << name << "，年龄：" << age << endl;
    }
};

int main() {
    Person p;

    // 类外可以直接访问 public 成员
    p.name = "李四";    // ✅ 正确
    p.age = 25;         // ✅ 正确
    p.show();           // ✅ 正确

    return 0;
}
```

### 4.2 protected（保护权限）

```cpp
class Person {
protected:
    string cardId;  // 保护成员（比如身份证号）

public:
    string name;

    void show() {
        // 类内可以访问 protected 成员
        cout << "姓名：" << name << "，身份证号：" << cardId << endl;
    }
};

int main() {
    Person p;
    p.name = "王五";       // ✅ 正确，public 可以在类外访问
    // p.cardId = "12345"; // ❌ 错误！protected 不能在类外访问

    return 0;
}
```

### 4.3 private（私有权限）

```cpp
class Person {
private:
    string password;  // 私有成员（比如密码）

public:
    string name;

    void show() {
        // 类内可以访问 private 成员
        cout << "姓名：" << name << endl;
        cout << "密码：" << password << endl;  // ✅ 类内可以访问
    }
};

int main() {
    Person p;
    p.name = "赵六";             // ✅ 正确
    // p.password = "123456";    // ❌ 错误！private 不能在类外访问

    return 0;
}
```

### 4.4 三种权限的完整对比示例

```cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
public:
    string name;        // 公共：类内、类外、子类都能访问

protected:
    string school;      // 保护：类内、子类可以访问，类外不行

private:
    string secret;      // 私有：只有类内可以访问

public:
    // 成员函数可以访问所有权限的成员
    void showAll() {
        cout << "姓名：" << name << endl;      // ✅ 访问 public
        cout << "学校：" << school << endl;    // ✅ 访问 protected
        cout << "秘密：" << secret << endl;    // ✅ 访问 private
    }
};

int main() {
    Student s;
    s.name = "小明";              // ✅ public 可以在类外访问
    // s.school = "清华";         // ❌ protected 不能在类外访问
    // s.secret = "考试答案";     // ❌ private 不能在类外访问

    s.showAll();                  // ✅ 通过公共函数间接访问所有成员

    return 0;
}
```

## 五、成员属性和成员函数

### 5.1 成员属性

成员属性是类中的变量，用于描述对象的特征。

```cpp
class Car {
public:
    string brand;    // 品牌
    string color;    // 颜色
    double price;    // 价格
    int speed;       // 速度
};
```

### 5.2 成员函数

成员函数是类中的函数，用于描述对象的行为。

```cpp
class Car {
public:
    string brand;
    int speed;

    // 成员函数：加速
    void accelerate(int value) {
        speed += value;
        cout << brand << "加速到" << speed << "km/h" << endl;
    }

    // 成员函数：刹车
    void brake() {
        speed = 0;
        cout << brand << "已停车" << endl;
    }
};

int main() {
    Car car;
    car.brand = "宝马";
    car.speed = 0;

    car.accelerate(60);   // 输出：宝马加速到60km/h
    car.accelerate(40);   // 输出：宝马加速到100km/h
    car.brake();          // 输出：宝马已停车

    return 0;
}
```

## 六、对象的创建和使用

```cpp
#include <iostream>
#include <string>
using namespace std;

class Dog {
public:
    string name;
    int age;

    void bark() {
        cout << name << "：汪汪汪！" << endl;
    }
};

int main() {
    // 方式1：先创建对象，再赋值
    Dog dog1;
    dog1.name = "旺财";
    dog1.age = 3;
    dog1.bark();

    // 方式2：创建对象的同时，用花括号初始化（C++11）
    Dog dog2 = {"小白", 2};
    dog2.bark();

    // 方式3：创建多个对象
    Dog dogs[3];  // 创建对象数组
    dogs[0].name = "狗1";
    dogs[1].name = "狗2";
    dogs[2].name = "狗3";

    for (int i = 0; i < 3; i++) {
        dogs[i].bark();
    }

    return 0;
}
```

## 七、对象的大小（sizeof）

**重要概念：对象的大小只计算成员属性，成员函数不占对象空间。**

成员函数是所有对象共享的，存储在代码区（不是对象中）。

```cpp
#include <iostream>
using namespace std;

class Empty {
    // 空类，没有任何成员
};

class Person {
public:
    int age;          // 4 字节
    double height;    // 8 字节
    char gender;      // 1 字节

    // 成员函数不占对象空间
    void show() {
        cout << age << endl;
    }
};

class OnlyFunc {
public:
    // 只有成员函数，没有成员属性
    void func1() {}
    void func2() {}
    void func3() {}
};

int main() {
    Empty e;
    cout << "空类大小：" << sizeof(e) << endl;       // 输出：1（占位，表示对象存在）

    Person p;
    // 注意：由于内存对齐，实际大小可能是 16 或 24 字节
    cout << "Person 大小：" << sizeof(p) << endl;    // 输出：16（4+8+1+对齐填充）

    OnlyFunc of;
    cout << "OnlyFunc 大小：" << sizeof(of) << endl; // 输出：1（函数不占对象空间）

    return 0;
}
```

**内存对齐规则：**
- 编译器会按照最大的成员类型进行对齐
- Person 中最大的成员是 double（8字节），所以按 8 字节对齐
- age(4) + padding(4) + height(8) + gender(1) + padding(7) = 24 字节
- 但有些编译器优化后可能输出 16

## 八、`.` 和 `->` 访问成员

- **`.`**（点运算符）：用于普通对象访问成员
- **`->`**（箭头运算符）：用于对象指针访问成员

```cpp
#include <iostream>
#include <string>
using namespace std;

class Student {
public:
    string name;
    int score;

    void show() {
        cout << "姓名：" << name << "，分数：" << score << endl;
    }
};

int main() {
    // ========== 1. 普通对象，用 . 访问 ==========
    Student s1;
    s1.name = "张三";
    s1.score = 90;
    s1.show();       // 用 . 调用成员函数

    // ========== 2. 对象指针，用 -> 访问 ==========
    Student *p = &s1;        // p 指向 s1
    p->name = "李四";       // 用 -> 访问成员属性
    p->score = 95;          // 用 -> 访问成员属性
    p->show();              // 用 -> 调用成员函数

    // ========== 3. new 出来的对象，用 -> 访问 ==========
    Student *p2 = new Student;  // 在堆区创建对象
    p2->name = "王五";
    p2->score = 88;
    p2->show();

    delete p2;  // 记得释放

    // ========== 4. 也可以用 (*p). 访问（不常用）==========
    Student s2;
    Student *p3 = &s2;
    (*p3).name = "赵六";    // 先解引用，再用 . 访问
    (*p3).show();

    return 0;
}
```

**总结：**

```cpp
Student s;          // 栈上的对象
Student *p = &s;    // 指向对象的指针

s.name;             // 普通对象用 .
p->name;            // 指针用 ->
(*p).name;          // 等价于 p->name（不常用）
```

## 九、完整综合示例

```cpp
#include <iostream>
#include <string>
using namespace std;

// 定义一个 Book 类
class Book {
public:
    // --- 公共成员（类外可以访问）---
    string title;       // 书名
    string author;      // 作者

protected:
    // --- 保护成员（类外不能访问，子类可以访问）---
    string publisher;   // 出版社

private:
    // --- 私有成员（只有类内可以访问）---
    double cost;        // 成本价

public:
    // 构造函数（后面会详细讲）
    Book(string t, string a, string p, double c) {
        title = t;
        author = a;
        publisher = p;
        cost = c;
    }

    // 公共接口：显示书的信息
    void showInfo() {
        // 类内可以访问所有权限的成员
        cout << "==================" << endl;
        cout << "书名：" << title << endl;
        cout << "作者：" << author << endl;
        cout << "出版社：" << publisher << endl;
        cout << "成本价：" << cost << "元" << endl;
        cout << "==================" << endl;
    }

    // 公共接口：设置成本价（带校验）
    void setCost(double c) {
        if (c > 0) {
            cost = c;      // 合法，设置新值
        } else {
            cout << "成本价不能为负数！" << endl;
        }
    }

    // 公共接口：获取成本价
    double getCost() {
        return cost;
    }
};

int main() {
    // 创建 Book 对象
    Book book("C++入门", "张三", "清华大学出版社", 35.5);

    // 访问 public 成员
    cout << "书名：" << book.title << endl;    // ✅
    cout << "作者：" << book.author << endl;   // ✅

    // 访问 protected 成员
    // cout << book.publisher << endl;         // ❌ 错误！类外不能访问

    // 访问 private 成员
    // cout << book.cost << endl;              // ❌ 错误！类外不能访问

    // 通过公共接口访问私有成员
    book.setCost(40.0);                        // ✅ 通过 setter 修改
    cout << "成本价：" << book.getCost() << endl; // ✅ 通过 getter 获取

    // 显示完整信息
    book.showInfo();

    return 0;
}
```

## 十、常见错误和注意事项

### 错误1：忘记写访问权限

```cpp
class Person {
    int age;  // 没写访问权限
    // class 默认是 private，所以 age 是私有的
};

int main() {
    Person p;
    // p.age = 20;  // ❌ 错误！age 是 private
}
```

### 错误2：在类外直接访问私有成员

```cpp
class Person {
private:
    int age;
};

int main() {
    Person p;
    // p.age = 20;  // ❌ 错误！private 成员不能在类外访问
}
```

### 错误3：混淆 `.` 和 `->`

```cpp
class Person {
public:
    int age;
};

int main() {
    Person p;
    Person *ptr = &p;

    // p.age = 20;     // ✅ 普通对象用 .
    // ptr.age = 20;   // ❌ 指针不能用 .
    ptr->age = 20;      // ✅ 指针用 ->
}
```

### 注意事项

1. **class 和 struct 的唯一区别**就是默认访问权限，其他完全一样
2. **成员函数**可以直接访问类内所有权限的成员
3. **类外**只能访问 public 成员
4. **对象的大小**只包含成员属性，不包含成员函数
5. **空类的大小**为 1 字节（占位，表示对象存在）
6. 一般将**成员属性设为 private**，通过 public 的函数来访问（封装思想）
