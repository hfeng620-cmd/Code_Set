# 14 - 赋值运算符详解

## 一、什么是赋值运算符？

赋值运算符用于把一个值**存放到变量中**。最基础的就是 `=` 号。
注意：在 C++ 中 `=` 是赋值（把右边的值放到左边），`==` 才是比较是否相等！

## 二、基本赋值运算符 =

```cpp
#include <iostream>
using namespace std;

int main() {
    // 基本赋值
    int a = 10;         // 定义变量a，赋初始值10
    cout << a << endl;  // 输出 10

    // 修改变量的值（重新赋值）
    a = 20;
    cout << a << endl;  // 输出 20

    // 用表达式的值赋值
    a = 3 + 5;
    cout << a << endl;  // 输出 8

    // 用另一个变量的值赋值
    int b = 100;
    a = b;
    cout << a << endl;  // 输出 100

    return 0;
}
```

### 赋值的方向：从右到左

```cpp
int x;
x = 50;     // 把 50 赋给 x，正确！
// 50 = x;  // 错误！不能把值赋给一个常量（50不是左值）

int a, b;
a = b = 100;  // 右结合性：等价于 a = (b = 100)
// 先执行 b = 100（b变成100），再把100赋给a
cout << "a=" << a << ", b=" << b << endl;  // a=100, b=100
```

### 赋值时的类型转换

```cpp
#include <iostream>
using namespace std;

int main() {
    // 大类型赋值给小类型：可能丢失精度
    int a = 3.14;    // double转int，小数部分丢弃，a = 3
    cout << "int a = 3.14 -> a = " << a << endl;  // 3

    // 小类型赋值给大类型：自动提升，不丢失精度
    double b = 10;   // int转double，b = 10.0
    cout << "double b = 10 -> b = " << b << endl;  // 10

    // char 和 int 之间的赋值
    char c = 65;     // ASCII 65 是 'A'
    cout << "char c = 65 -> c = " << c << endl;  // A

    int d = 'A';     // 'A' 的 ASCII 码是 65
    cout << "int d = 'A' -> d = " << d << endl;  // 65

    // 布尔类型的赋值
    bool flag1 = 0;    // false
    bool flag2 = 1;    // true
    bool flag3 = -1;   // true（非0就是true）
    bool flag4 = 3.14; // true（非0就是true）
    cout << flag1 << " " << flag2 << " " << flag3 << " " << flag4 << endl;
    // 输出：0 1 1 1

    return 0;
}
```

## 三、复合赋值运算符

复合赋值运算符是"运算+赋值"的简写形式，让代码更简洁。

| 复合赋值 | 等价写法 | 含义 |
|----------|----------|------|
| `a += b` | `a = a + b` | 加后赋值 |
| `a -= b` | `a = a - b` | 减后赋值 |
| `a *= b` | `a = a * b` | 乘后赋值 |
| `a /= b` | `a = a / b` | 除后赋值 |
| `a %= b` | `a = a % b` | 取模后赋值 |

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;

    // += 加赋值
    a += 3;      // 等价于 a = a + 3;  a变成13
    cout << "a += 3  -> " << a << endl;   // 13

    // -= 减赋值
    a -= 5;      // 等价于 a = a - 5;  a变成8
    cout << "a -= 5  -> " << a << endl;   // 8

    // *= 乘赋值
    a *= 2;      // 等价于 a = a * 2;  a变成16
    cout << "a *= 2  -> " << a << endl;   // 16

    // /= 除赋值
    a /= 4;      // 等价于 a = a / 4;  a变成4
    cout << "a /= 4  -> " << a << endl;   // 4

    // %= 取模赋值
    a %= 3;      // 等价于 a = a % 3;  a变成1
    cout << "a %= 3  -> " << a << endl;   // 1

    return 0;
}
```

### 复合赋值的实际应用场景

```cpp
#include <iostream>
using namespace std;

int main() {
    // 场景1：累加（统计总分）
    int totalScore = 0;
    totalScore += 85;    // 加上第1门成绩
    totalScore += 92;    // 加上第2门成绩
    totalScore += 78;    // 加上第3门成绩
    cout << "总分: " << totalScore << endl;  // 255

    // 场景2：累减（倒计时）
    int countdown = 10;
    countdown -= 3;
    cout << "倒计时: " << countdown << endl;  // 7

    // 场景3：翻倍增长
    int money = 100;
    money *= 2;   // 翻一倍
    cout << "翻倍后: " << money << endl;     // 200
    money *= 2;   // 再翻一倍
    cout << "再翻倍: " << money << endl;     // 400

    // 场景4：取余循环
    int counter = 0;
    for (int i = 0; i < 10; i++) {
        counter += 1;
        counter %= 3;   // 让counter在0,1,2之间循环
        cout << counter << " ";
    }
    // 输出: 1 2 0 1 2 0 1 2 0 1
    cout << endl;

    // 场景5：字符串拼接（string的+=）
    string s = "Hello";
    s += " World";
    cout << s << endl;  // Hello World

    return 0;
}
```

## 四、赋值的右结合性

赋值运算符是**右结合**的，意味着连续赋值时从**右边开始**计算。

```cpp
#include <iostream>
using namespace std;

int main() {
    // 连续赋值：从右往左依次赋值
    int a, b, c;
    a = b = c = 100;
    // 等价于：a = (b = (c = 100))
    // 第1步：c = 100，c变成100
    // 第2步：b = c，b变成100
    // 第3步：a = b，a变成100
    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;  // 100, 100, 100

    // 更复杂的例子
    int x, y, z;
    x = (y = (z = 5) + 3) + 2;
    // 第1步：z = 5，表达式值为5
    // 第2步：y = 5 + 3 = 8，表达式值为8
    // 第3步：x = 8 + 2 = 10
    cout << "x=" << x << ", y=" << y << ", z=" << z << endl;
    // 输出：x=10, y=8, z=5

    return 0;
}
```

## 五、赋值 vs 初始化（重要区别！）

### 5.1 什么是初始化？

初始化是在**创建变量的同时**给它一个值。

### 5.2 什么是赋值？

赋值是**变量已经存在**之后，**修改**它的值。

```cpp
#include <iostream>
using namespace std;

int main() {
    // =============================================
    // 初始化：声明变量的同时给它一个初始值
    // =============================================
    int a = 10;          // 初始化（声明+赋初值）
    int b(20);           // 直接初始化（C++风格）
    int c{30};           // 列表初始化（C++11）
    int d = {40};        // 拷贝列表初始化（C++11）

    cout << "a=" << a << ", b=" << b << ", c=" << c << ", d=" << d << endl;

    // =============================================
    // 赋值：对已经存在的变量修改其值
    // =============================================
    a = 100;  // 赋值（变量a已经存在，修改它的值）
    b = 200;  // 赋值
    cout << "a=" << a << ", b=" << b << endl;

    // =============================================
    // 初始化和赋值的主要区别
    // =============================================

    // 区别1：const 变量必须初始化，不能赋值
    const int CONST_A = 100;  // 初始化：正确
    // CONST_A = 200;         // 赋值：错误！const变量不能修改

    // 区别2：引用必须初始化，不能先声明再赋值
    int value = 42;
    int& ref = value;    // 初始化引用：正确
    // int& ref2;         // 错误！引用必须初始化
    // ref2 = value;      // 错误！

    // 区别3：C++11 的列表初始化可以防止窄化转换
    int x = 3.14;     // 允许，小数部分被截断（编译器可能警告）
    // int y{3.14};    // C++11列表初始化，不允许窄化转换，编译报错！

    return 0;
}
```

## 六、位运算复合赋值（了解即可）

除了算术类的复合赋值，还有位运算类的，这里先了解，后面位运算会详细讲。

| 复合赋值 | 等价写法 |
|----------|----------|
| `a &= b` | `a = a & b` |
| `a \|= b` | `a = a \| b` |
| `a ^= b` | `a = a ^ b` |
| `a <<= n` | `a = a << n` |
| `a >>= n` | `a = a >> n` |

```cpp
int flags = 0b1010;
flags &= 0b1100;   // 按位与，flags变为 0b1000
flags <<= 2;       // 左移2位，flags变为 0b10000000
```

## 七、常见的赋值错误

### 7.1 在 if 中把 == 写成 =

```cpp
int a = 5;

// 错误写法（但编译器通常不报错，只是警告）
if (a = 10) {          // 这是赋值，不是比较！a被赋值为10，条件永远为真
    cout << "条件为真" << endl;  // 这行总是会执行
}
cout << a << endl;     // 输出 10（a的值被改了！）

// 正确写法
if (a == 10) {         // 这是比较，判断a是否等于10
    cout << "a等于10" << endl;
}
```

> **防坑技巧**：把常量写在左边，这样如果误写成 `=` 编译器会报错。
> ```cpp
> if (10 == a) { ... }   // 正确
> // if (10 = a) { ... } // 编译错误！不能给常量赋值
> ```

### 7.2 赋值表达式本身有值

```cpp
int a;
// 赋值表达式 "a = 5" 的值就是 5（即所赋的那个值）
cout << (a = 5) << endl;  // 输出 5

// 这个特性可以用于链式赋值
int x, y, z;
x = y = z = 42;
```

### 7.3 溢出问题

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // 当赋值超出变量类型的范围时，会发生溢出
    short s = 32767;   // short 最大值
    s += 1;            // 溢出！变为 -32768
    cout << "short最大值 + 1 = " << s << endl;  // -32768

    int i = INT_MAX;   // int 最大值（约21亿）
    i += 1;            // 溢出！变为 INT_MIN
    cout << "int最大值 + 1 = " << i << endl;    // -2147483648

    return 0;
}
```

## 八、综合练习

### 练习1：成绩累加器

```cpp
#include <iostream>
using namespace std;

int main() {
    double totalScore = 0;
    double score;

    cout << "请输入5门课的成绩:" << endl;
    for (int i = 1; i <= 5; i++) {
        cout << "第" << i << "门: ";
        cin >> score;
        totalScore += score;   // 累加每门成绩
    }

    double average = totalScore / 5;
    cout << "总分: " << totalScore << endl;
    cout << "平均分: " << average << endl;

    return 0;
}
```

### 练习2：计算1到100的累加和与阶乘

```cpp
#include <iostream>
using namespace std;

int main() {
    // 用 += 计算累加和
    int sum = 0;
    for (int i = 1; i <= 100; i++) {
        sum += i;  // 等价于 sum = sum + i
    }
    cout << "1到100的和: " << sum << endl;   // 5050

    // 用 *= 计算阶乘
    long long factorial = 1;
    for (int i = 1; i <= 20; i++) {
        factorial *= i;  // 等价于 factorial = factorial * i
    }
    cout << "20的阶乘: " << factorial << endl;  // 2432902008176640000

    // 用 /= 逐位去掉数字，判断位数
    int num = 12345;
    int digits = 0;
    int temp = num;
    while (temp > 0) {
        digits++;
        temp /= 10;  // 每次去掉一位
    }
    cout << num << " 是 " << digits << " 位数" << endl;  // 5 位数

    return 0;
}
```

## 九、总结

1. `=` 是赋值运算符，作用是把右边的值存入左边的变量
2. 复合赋值 `+=` `-=` `*=` `/=` `%=` 是"运算+赋值"的简写
3. 赋值运算符是右结合的：`a = b = c = 0` 从右往左赋值
4. 初始化是创建变量时赋值，赋值是修改已有变量的值
5. `const` 变量和引用只能初始化，不能赋值
6. C++11 列表初始化 `{}` 可以防止窄化转换
7. 注意区分 `=`（赋值）和 `==`（比较），尤其在 `if` 语句中
8. 注意类型转换和溢出问题
