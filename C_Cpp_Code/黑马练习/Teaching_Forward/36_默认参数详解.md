# 默认参数详解

## 一、什么是默认参数

> **默认参数（Default Arguments）是指在函数声明或定义时，给参数指定一个默认值。**
> 当调用函数时如果没有传入该参数，就使用默认值；如果传入了实参，则使用传入的值。

默认参数可以让函数调用更加灵活，减少函数重载的数量。

```cpp
#include <iostream>
using namespace std;

// 声明函数时指定默认参数
// b 的默认值是 10，c 的默认值是 20
int add(int a, int b = 10, int c = 20) {
    return a + b + c;
}

int main() {
    cout << add(1, 2, 3) << endl;    // 全部传参：1+2+3 = 6
    cout << add(1, 2) << endl;       // 只传a和b：1+2+20 = 23（c用默认值20）
    cout << add(1) << endl;          // 只传a：1+10+20 = 31
    // cout << add() << endl;        // 错误！a没有默认值，必须传
    return 0;
}
```

**运行结果：**
```
6
23
31
```

## 二、默认参数的语法

### 2.1 基本语法

```cpp
// 语法格式：
// 返回值类型 函数名(参数1, 参数2 = 默认值, 参数3 = 默认值, ...)
//                              ↑ 有默认值          ↑ 有默认值

// 示例1：单个默认参数
void printMessage(string msg = "Hello") {
    cout << msg << endl;
}

// 示例2：带默认值的计算函数
double calculateArea(double length, double width = 1.0) {
    return length * width;
}

// 示例3：幂运算
int power(int base, int exponent = 2) {
    int result = 1;
    for (int i = 0; i < exponent; i++) {
        result *= base;
    }
    return result;
}
```

### 2.2 调用方式

```cpp
#include <iostream>
using namespace std;

// 带默认参数的函数
void greet(string name, string greeting = "你好", string suffix = "！") {
    cout << greeting << "，" << name << suffix << endl;
}

int main() {
    // 方式1：全部使用默认参数（只传必要的参数）
    greet("小明");                // 输出：你好，小明！

    // 方式2：覆盖部分默认参数
    greet("小明", "早上好");      // 输出：早上好，小明！

    // 方式3：覆盖所有默认参数
    greet("小明", "晚上好", "~"); // 输出：晚上好，小明~

    return 0;
}
```

**运行结果：**
```
你好，小明！
早上好，小明！
晚上好，小明~
```

## 三、默认参数的重要规则

### 3.1 规则一：某位置有默认值，后面所有参数都必须有默认值

**这是最重要的规则！从某个参数开始有默认值，它右边的所有参数都必须有默认值。**

```cpp
// ====== 正确写法 ======
void func1(int a, int b = 10, int c = 20);           // OK
void func2(int a, int b, int c = 30);                 // OK
void func3(int a = 1, int b = 2, int c = 3);          // OK

// ====== 错误写法 ======
// void func4(int a = 1, int b, int c);               // 错误！a有默认值但b和c没有
// void func5(int a, int b = 10, int c);              // 错误！b有默认值但c没有
// void func6(int a = 1, int b, int c = 3);           // 错误！中间的b没有默认值

// 为什么有这个规则？
// 因为函数调用时参数是从左到右匹配的
// 如果允许"跳过"中间的参数，编译器不知道你传的值对应哪个参数
```

**图解说明：**
```
正确的参数默认值排列：

  a     b     c
  无    无    30      -> func(a, b, c) 或 func(a, b) 都可以
  无    10    20      -> func(a, b, c) 或 func(a, b) 或 func(a) 都可以
  1     2     3       -> 四种调用方式都可以

错误的参数默认值排列：

  a     b     c
  1     无    3       -> 编译器无法确定传入的值对应哪个参数！
```

### 3.2 规则二：声明和定义中只能有一个写默认值

**函数声明和函数定义不能同时指定默认参数值，只能在其中一个地方写。推荐在声明处写。**

```cpp
// ====== 推荐做法：在声明处写默认值 ======

// 文件：myheader.h（声明）
void showInfo(string name, int age = 18);   // 声明时指定默认值

// 文件：mycode.cpp（定义）
void showInfo(string name, int age) {       // 定义时不写默认值
    cout << "姓名：" << name << "，年龄：" << age << endl;
}

// ====== 错误做法：声明和定义都写默认值 ======
// void showInfo(string name, int age = 18);   // 声明写了一次
// void showInfo(string name, int age = 18) {  // 定义又写了一次 -> 编译错误！
//     cout << name << ", " << age << endl;
// }

// ====== 另一种合法做法：只在定义处写（没有单独声明时） ======
// 如果函数没有提前声明（直接定义），可以在定义处写默认值
```

### 3.3 完整示例：声明与定义分离

```cpp
#include <iostream>
#include <string>
using namespace std;

// ====== 函数声明（带默认参数）======
void printStudent(string name, int age = 18, string school = "黑马");

int main() {
    printStudent("张三");                    // 用所有默认值
    printStudent("李四", 20);                // 覆盖age
    printStudent("王五", 22, "清华大学");     // 覆盖所有默认值
    return 0;
}

// ====== 函数定义（不写默认参数）======
void printStudent(string name, int age, string school) {
    cout << "姓名：" << name << endl;
    cout << "年龄：" << age << endl;
    cout << "学校：" << school << endl;
    cout << "-------------------" << endl;
}
```

**运行结果：**
```
姓名：张三
年龄：18
学校：黑马
-------------------
姓名：李四
年龄：20
学校：黑马
-------------------
姓名：王五
年龄：22
学校：清华大学
-------------------
```

## 四、默认参数与函数重载的冲突

**默认参数和函数重载一起使用时，可能会产生二义性（ambiguity），导致编译错误。**

```cpp
#include <iostream>
using namespace std;

// ====== 冲突示例 ======

// 重载版本1：接受一个参数
void func(int a) {
    cout << "一个参数：" << a << endl;
}

// 重载版本2：接受两个参数，第二个有默认值
void func(int a, int b = 10) {
    cout << "两个参数：" << a << ", " << b << endl;
}

int main() {
    // func(5);      // 错误！编译器不知道调用哪个版本
                     // 可以匹配 func(int a)
                     // 也可以匹配 func(int a, int b = 10)
                     // 产生二义性！
    func(5, 10);     // OK！明确调用第二个版本
    return 0;
}
```

**如何避免冲突：**
```cpp
#include <iostream>
using namespace std;

// 方法：不要让有默认参数的函数与重载产生歧义

// 方案1：只用默认参数，不用重载
void print(int a, int b = 0) {
    cout << a << ", " << b << endl;
}

// 方案2：只用重载，不用默认参数
// void print(int a) { cout << a << endl; }
// void print(int a, int b) { cout << a << ", " << b << endl; }

int main() {
    print(5);      // OK：调用 print(5, 0)
    print(5, 10);  // OK：调用 print(5, 10)
    return 0;
}
```

## 五、常见错误汇总

| 错误类型 | 示例 | 说明 |
|---------|------|------|
| 默认值位置不对 | `void f(int a=1, int b, int c)` | 有默认值的参数后面必须全有默认值 |
| 声明和定义都写 | 声明 `int f(int a=10)` + 定义 `int f(int a=10){}` | 只能在一处写默认值 |
| 与重载冲突 | `void f(int a)` + `void f(int a, int b=10)` | 调用 `f(5)` 时二义性 |

## 六、默认参数的实际应用

```cpp
#include <iostream>
#include <string>
using namespace std;

// 实际应用1：设置配置默认值
void connectToDatabase(
    string host = "localhost",
    int port = 3306,
    string username = "root",
    string password = ""
) {
    cout << "连接数据库：" << endl;
    cout << "  主机：" << host << endl;
    cout << "  端口：" << port << endl;
    cout << "  用户：" << username << endl;
    cout << endl;
}

// 实际应用2：绘制矩形（正方形是特殊情况）
void drawRect(int width, int height = 0) {
    if (height == 0) height = width;
    cout << "绘制矩形：" << width << " x " << height << endl;
}

// 实际应用3：日志输出级别
void log(string message, string level = "INFO", bool showTime = true) {
    if (showTime) cout << "[2024-01-01 12:00:00] ";
    cout << "[" << level << "] " << message << endl;
}

int main() {
    connectToDatabase();
    connectToDatabase("192.168.1.100", 5432);

    cout << "---" << endl;

    drawRect(10);       // 画10x10的正方形
    drawRect(10, 5);    // 画10x5的矩形

    cout << "---" << endl;

    log("程序启动");
    log("发生错误", "ERROR");
    log("调试信息", "DEBUG", false);

    return 0;
}
```

## 七、默认参数的值可以是哪些类型

```cpp
#include <iostream>
using namespace std;

// 默认参数可以是常量
void f1(int a = 100) { cout << a << endl; }

// 默认参数可以是全局变量
int globalVal = 50;
void f2(int a = globalVal) { cout << a << endl; }

// 默认参数可以是表达式（参数a先有值，然后b = a*2）
void f3(int a, int b = 0) { cout << a + b << endl; }

// 默认参数不能是局部变量
// void f4() {
//     int local = 10;
//     void inner(int a = local);  // 错误！
// }

int main() {
    f1();          // 100
    f2();          // 50
    f3(5);         // 5（a=5, b=0）
    f3(5, 10);     // 15（a=5, b=10）
    return 0;
}
```

## 八、默认参数 vs 函数重载的选择

```
什么时候用默认参数？什么时候用函数重载？

场景1：功能相同，只是参数个数不同 -> 用默认参数
  一个函数搞定，代码更简洁

场景2：功能完全不同，参数类型不同 -> 用函数重载
  需要不同的函数体

场景3：参数类型不同，但逻辑相同 -> 用函数模板
  参见"函数模板"一节
```

```cpp
#include <iostream>
using namespace std;

// 适合用默认参数：功能相同，参数数量不同
void printLine(string msg, int count = 1) {
    for (int i = 0; i < count; i++) {
        cout << msg << endl;
    }
}

// 适合用函数重载：参数类型不同，处理逻辑也不同
void process(int x) {
    cout << "处理整数：" << x * 2 << endl;
}
void process(string s) {
    cout << "处理字符串：" << s.length() << " 个字符" << endl;
}

int main() {
    printLine("Hello");       // 打印1次
    printLine("Hello", 3);   // 打印3次

    cout << "---" << endl;

    process(42);              // 调用整数版本
    process("Hello");         // 调用字符串版本

    return 0;
}
```

## 九、注意事项总结

| 序号 | 规则 | 说明 |
|------|------|------|
| 1 | 从右往左连续设置 | 某参数有默认值，其右边所有参数都必须有默认值 |
| 2 | 声明或定义只写一处 | 推荐在函数声明处写默认参数 |
| 3 | 注意与重载冲突 | 有默认参数的函数可能与重载版本产生二义性 |
| 4 | 默认值可以是常量或表达式 | 但不能是局部变量 |
| 5 | 默认参数是编译期确定的 | 编译器在调用处自动填入默认值 |
| 6 | 覆盖时从左往右 | 要覆盖中间的参数，左边的也必须传 |

## 十、练习题

1. 写一个函数 `circleArea(double r, double pi = 3.14159)` 计算圆的面积。
2. 写一个函数 `repeat(string s, int times = 1, string sep = " ")` 将字符串重复输出。
3. 思考：为什么默认参数必须从右往左连续设置？试着从函数调用的角度解释。

```cpp
// 练习1 参考答案：
#include <iostream>
using namespace std;

double circleArea(double r, double pi = 3.14159) {
    return pi * r * r;
}

int main() {
    cout << "面积1：" << circleArea(5) << endl;
    cout << "面积2：" << circleArea(5, 3.14) << endl;
    return 0;
}

// 练习2 参考答案：
#include <iostream>
#include <string>
using namespace std;

void repeat(string s, int times = 1, string sep = " ") {
    for (int i = 0; i < times; i++) {
        if (i > 0) cout << sep;
        cout << s;
    }
    cout << endl;
}

int main() {
    repeat("hello");           // 输出：hello
    repeat("hello", 3);       // 输出：hello hello hello
    repeat("hello", 3, "-");  // 输出：hello-hello-hello
    return 0;
}
```
