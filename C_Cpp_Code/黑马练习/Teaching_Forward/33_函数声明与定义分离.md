# 33 - 函数声明与定义分离

## 一、函数声明（函数原型）的概念

### 1.1 什么是函数声明

函数声明（也叫函数原型）是告诉编译器函数的基本信息：函数名、返回值类型、参数类型。它不包含函数体。

```cpp
// 函数声明的格式：
// 返回值类型 函数名(参数类型列表);

// 示例：
int add(int a, int b);        // 声明一个返回int，接收两个int参数的函数
void printHello();             // 声明一个无参无返回值的函数
double calculateArea(double);  // 声明时可以省略参数名
```

### 1.2 函数声明 vs 函数定义

```cpp
// 函数声明（原型）：只有函数头，没有函数体，以分号结尾
int add(int a, int b);

// 函数定义：有函数头和函数体
int add(int a, int b) {
    return a + b;
}
```

## 二、为什么需要函数声明

### 2.1 C++编译器的规则

C++编译器从上到下编译代码。如果函数调用在函数定义之前，编译器不认识这个函数，会报错。

```cpp
#include <iostream>
using namespace std;

int main() {
    int result = add(3, 5);  // 错误！编译器还没看到add的定义
    cout << result << endl;
    return 0;
}

// 函数定义在main之后
int add(int a, int b) {
    return a + b;
}
```

### 2.2 解决方案：提前声明

```cpp
#include <iostream>
using namespace std;

// 方法1：在main之前添加函数声明
int add(int a, int b);  // 函数声明（原型）

int main() {
    int result = add(3, 5);  // 正确！编译器知道add是什么了
    cout << result << endl;
    return 0;
}

// 函数定义放在main之后
int add(int a, int b) {
    return a + b;
}
```

### 2.3 另一个例子

```cpp
#include <iostream>
using namespace std;

// 函数声明
void greet(string name);
void farewell(string name);
void printLine();

int main() {
    greet("小明");      // 调用在定义之前，但有声明，所以可以
    printLine();
    farewell("小明");   // 同样可以
    return 0;
}

// 函数定义
void greet(string name) {
    cout << "你好，" << name << "！欢迎！" << endl;
}

void farewell(string name) {
    cout << "再见，" << name << "！" << endl;
}

void printLine() {
    cout << "--------------------" << endl;
}
```

## 三、函数声明的写法

### 3.1 基本语法

```cpp
// 格式：返回值类型 函数名(参数类型列表);

// 示例1：完整写法（带参数名）
int add(int a, int b);

// 示例2：省略参数名（只写类型）
int add(int, int);

// 示例3：无参函数
void printHello();

// 示例4：多参数
double calculate(double x, double y, double z);
```

### 3.2 参数名可以省略

```cpp
#include <iostream>
using namespace std;

// 完整写法
int add(int a, int b);

// 省略参数名的写法（效果完全一样）
int subtract(int, int);

// 两种写法都可以，选择你喜欢的风格
// 建议：在头文件中省略参数名，在源文件中写完整

int main() {
    cout << add(10, 5) << endl;       // 15
    cout << subtract(10, 5) << endl;  // 5
    return 0;
}

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}
```

### 3.3 声明中的参数名可以和定义不同

```cpp
#include <iostream>
using namespace std;

// 声明中的参数名
int multiply(int x, int y);

// 定义中的参数名可以不同（不影响）
int multiply(int a, int b) {
    return a * b;
}

int main() {
    cout << multiply(3, 4) << endl;  // 输出12
    return 0;
}
```

## 四、函数声明可以写多次，定义只能写一次

### 4.1 多次声明是允许的

```cpp
#include <iostream>
using namespace std;

// 同一个函数可以声明多次（只要内容一致）
int add(int a, int b);
int add(int, int);        // 可以重复声明
int add(int x, int y);    // 参数名不同也没关系

int main() {
    cout << add(3, 5) << endl;  // 输出8
    return 0;
}

// 定义只能写一次！
int add(int a, int b) {
    return a + b;
}

// 错误！重复定义
// int add(int a, int b) {
//     return a + b;
// }
```

### 4.2 声明不一致会报错

```cpp
#include <iostream>
using namespace std;

// 声明
int add(int a, int b);

// 错误！声明和定义不一致
// double add(int a, int b) {  // 返回值类型不同
//     return a + b;
// }

// 正确：声明和定义一致
int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(3, 5) << endl;
    return 0;
}
```

## 五、函数定义放在 main 之后的情况

### 5.1 实际项目中的常见做法

在实际项目中，通常把函数声明放在前面，函数定义放在后面，或者放在不同的文件中。

```cpp
#include <iostream>
using namespace std;

// ============ 函数声明区域 ============
// 把所有函数声明放在一起，便于管理

// 数学运算函数
int add(int a, int b);
int subtract(int a, int b);
int multiply(int a, int b);
double divide(int a, int b);

// 工具函数
void printResult(string operation, double result);
void printLine();

// ============ 主函数 ============
int main() {
    int a = 10, b = 5;

    printLine();
    printResult("加法", add(a, b));
    printResult("减法", subtract(a, b));
    printResult("乘法", multiply(a, b));
    printResult("除法", divide(a, b));
    printLine();

    return 0;
}

// ============ 函数定义区域 ============
// 把所有函数定义放在一起

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

double divide(int a, int b) {
    if (b == 0) {
        cout << "错误：除数不能为0！" << endl;
        return 0;
    }
    return (double)a / b;
}

void printResult(string operation, double result) {
    cout << operation << "的结果是: " << result << endl;
}

void printLine() {
    cout << "================================" << endl;
}
```

## 六、函数声明放在头文件中

### 6.1 为什么要放在头文件中

在实际项目中，函数声明通常放在头文件（.h）中，函数定义放在源文件（.cpp）中。这样做的好处：
1. 其他文件只需要 `#include` 头文件就能使用这些函数
2. 实现细节被隐藏
3. 代码组织更清晰

### 6.2 头文件示例

```cpp
// math_utils.h - 数学工具函数的头文件
#ifndef MATH_UTILS_H
#define MATH_UTILS_H

// 函数声明
int add(int a, int b);
int subtract(int a, int b);
int multiply(int a, int b);
double divide(int a, int b);

#endif  // MATH_UTILS_H
```

### 6.3 源文件示例

```cpp
// math_utils.cpp - 数学工具函数的实现
#include "math_utils.h"
#include <iostream>
using namespace std;

// 函数定义
int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

double divide(int a, int b) {
    if (b == 0) {
        cout << "错误：除数不能为0！" << endl;
        return 0;
    }
    return (double)a / b;
}
```

### 6.4 主程序示例

```cpp
// main.cpp - 主程序
#include <iostream>
#include "math_utils.h"  // 包含头文件，就能使用声明的函数
using namespace std;

int main() {
    cout << "3 + 5 = " << add(3, 5) << endl;
    cout << "10 - 4 = " << subtract(10, 4) << endl;
    cout << "6 * 7 = " << multiply(6, 7) << endl;
    cout << "15 / 4 = " << divide(15, 4) << endl;
    return 0;
}
```

## 七、常见错误

### 7.1 声明和定义不一致

```cpp
#include <iostream>
using namespace std;

// 声明
int add(int a, int b);

// 错误1：返回值类型不一致
// double add(int a, int b) {  // 编译错误
//     return a + b;
// }

// 错误2：参数类型不一致
// int add(double a, double b) {  // 编译错误
//     return a + b;
// }

// 错误3：参数个数不一致
// int add(int a, int b, int c) {  // 编译错误
//     return a + b + c;
// }

// 正确：声明和定义完全一致
int add(int a, int b) {
    return a + b;
}

int main() {
    cout << add(3, 5) << endl;
    return 0;
}
```

### 7.2 忘记声明

```cpp
#include <iostream>
using namespace std;

// 如果没有声明，函数定义必须在调用之前

int main() {
    // 错误！add还没定义
    // cout << add(3, 5) << endl;

    return 0;
}

int add(int a, int b) {
    return a + b;
}
```

### 7.3 声明写了函数体

```cpp
#include <iostream>
using namespace std;

// 错误！声明不应该有函数体
// int add(int a, int b) {
//     return a + b;
// };  // 这是定义，不是声明

// 正确：声明只有函数头和分号
int add(int a, int b);

int main() {
    cout << add(3, 5) << endl;
    return 0;
}

int add(int a, int b) {
    return a + b;
}
```

### 7.4 头文件中写函数定义

```cpp
// 错误的做法！
// my_functions.h
#ifndef MY_FUNCTIONS_H
#define MY_FUNCTIONS_H

// 错误：头文件中不应该写函数定义（除非是inline函数或模板）
// int add(int a, int b) {
//     return a + b;
// }

// 正确：头文件中只写声明
int add(int a, int b);

#endif
```

### 8.3 math_utils.cpp

```cpp
// math_utils.cpp
#include "math_utils.h"
#include <iostream>
using namespace std;

int add(int a, int b) {
    return a + b;
}

int subtract(int a, int b) {
    return a - b;
}

int multiply(int a, int b) {
    return a * b;
}

double divide(int a, int b) {
    if (b == 0) {
        cout << "错误：除数不能为0！" << endl;
        return 0;
    }
    return (double)a / b;
}

int power(int base, int exponent) {
    int result = 1;
    for (int i = 0; i < exponent; i++) {
        result *= base;
    }
    return result;
}

int factorial(int n) {
    if (n <= 1) return 1;
    int result = 1;
    for (int i = 2; i <= n; i++) {
        result *= i;
    }
    return result;
}
```

### 8.5 string_utils.cpp

```cpp
// string_utils.cpp
#include "string_utils.h"
#include <algorithm>
using namespace std;

int getStringLength(string str) {
    return str.length();
}

string toUpperCase(string str) {
    string result = str;
    for (int i = 0; i < result.length(); i++) {
        if (result[i] >= 'a' && result[i] <= 'z') {
            result[i] = result[i] - 32;
        }
    }
    return result;
}

string toLowerCase(string str) {
    string result = str;
    for (int i = 0; i < result.length(); i++) {
        if (result[i] >= 'A' && result[i] <= 'Z') {
            result[i] = result[i] + 32;
        }
    }
    return result;
}

string reverseString(string str) {
    string result = str;
    reverse(result.begin(), result.end());
    return result;
}

bool isPalindrome(string str) {
    string reversed = reverseString(str);
    return str == reversed;
}
```

### 8.6 main.cpp

```cpp

#include <iostream>
#include "math_utils.h"
#include "string_utils.h"
using namespace std;

int main() {
    // 使用数学工具
    cout << "===== 数学工具测试 =====" << endl;
    cout << "3 + 5 = " << add(3, 5) << endl;
    cout << "2^10 = " << power(2, 10) << endl;
    cout << "5! = " << factorial(5) << endl;

    // 使用字符串工具
    cout << "\n===== 字符串工具测试 =====" << endl;
    string str = "Hello World";
    cout << "原始字符串: " << str << endl;
    cout << "大写: " << toUpperCase(str) << endl;
    cout << "小写: " << toLowerCase(str) << endl;
    cout << "反转: " << reverseString(str) << endl;

    cout << "\n\"abcba\"是回文吗？" << (isPalindrome("abcba") ? "是" : "否") << endl;
    cout << "\"hello\"是回文吗？" << (isPalindrome("hello") ? "是" : "否") << endl;

    return 0;
}
```

## 九、注意事项总结

1. **声明以分号结尾**：函数声明必须以分号结束，这是和函数定义的区别
2. **声明可以省略参数名**：只写参数类型即可
3. **声明可以写多次**：只要内容一致
4. **定义只能写一次**：重复定义会编译错误
5. **声明和定义要一致**：返回值类型、参数类型、参数个数都要一致
6. **头文件用声明**：头文件中通常只放声明，不放定义
7. **使用头文件保护**：用 `#ifndef` 或 `#pragma once` 防止重复包含
8. **建议**：养成先声明后定义的习惯，代码组织更清晰
