# 冒泡排序详解

## 一、什么是冒泡排序？

冒泡排序（Bubble Sort）是最基础、最经典的排序算法之一。

**名字的由来：** 就像水里的气泡一样，大的气泡会慢慢浮到水面。在冒泡排序中，大的元素会像气泡一样，通过一轮轮的比较和交换，慢慢"浮"到数组的末尾。

**核心思想：** 相邻的两个元素进行比较，如果前面的比后面的大，就交换它们的位置。

---

## 二、冒泡排序的原理（图解）

我们用一个具体的数组 `{4, 2, 8, 0, 5}` 来一步步演示。

### 第1轮（把最大的数"冒泡"到最后）

```
初始数组: [4, 2, 8, 0, 5]

第1步: 比较 arr[0]和arr[1] → 4和2 → 4>2，交换 → [2, 4, 8, 0, 5]
第2步: 比较 arr[1]和arr[2] → 4和8 → 4<8，不换 → [2, 4, 8, 0, 5]
第3步: 比较 arr[2]和arr[3] → 8和0 → 8>0，交换 → [2, 4, 0, 8, 5]
第4步: 比较 arr[3]和arr[4] → 8和5 → 8>5，交换 → [2, 4, 0, 5, 8]
                                                                ↑
                                                          最大的8已经到了最后！

第1轮结束: [2, 4, 0, 5, 8]  （8已经就位）
```

### 第2轮（把第2大的数倒数第二位）

```
开始: [2, 4, 0, 5, 8]  （8已经确定，不用再动）

第1步: 比较 arr[0]和arr[1] → 2和4 → 2<4，不换 → [2, 4, 0, 5, 8]
第2步: 比较 arr[1]和arr[2] → 4和0 → 4>0，交换 → [2, 0, 4, 5, 8]
第3步: 比较 arr[2]和arr[3] → 4和5 → 4<5，不换 → [2, 0, 4, 5, 8]
                                                          ↑
                                                    5已经就位！

第2轮结束: [2, 0, 4, 5, 8]
```

### 第3轮

```
开始: [2, 0, 4, 5, 8]

第1步: 比较 arr[0]和arr[1] → 2和0 → 2>0，交换 → [0, 2, 4, 5, 8]
第2步: 比较 arr[1]和arr[2] → 2和4 → 2<4，不换 → [0, 2, 4, 5, 8]
                                                      ↑
                                                4已经就位！

第3轮结束: [0, 2, 4, 5, 8]
```

### 第4轮

```
开始: [0, 2, 4, 5, 8]

第1步: 比较 arr[0]和arr[1] → 0和2 → 0<2，不换 → [0, 2, 4, 5, 8]

第4轮结束: [0, 2, 4, 5, 8]  （排序完成！）
```

### 规律总结

- **5个元素**需要 **4轮** 排序 → 排序轮数 = 元素个数 - 1
- 第1轮比较 **4次**（5-1），第2轮比较 **3次**（5-2），第3轮比较 **2次**（5-3），第4轮比较 **1次**（5-4）
- 第 i 轮的比较次数 = 元素个数 - 1 - i

## 三、冒泡排序的完整代码实现

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {4, 2, 8, 0, 5};
    int len = sizeof(arr) / sizeof(arr[0]);

    // 排序前输出
    cout << "排序前: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    // ========== 冒泡排序核心代码 ==========

    // 外层循环：控制排序的轮数
    // n个元素需要 n-1 轮
    for (int i = 0; i < len - 1; i++) {

        // 内层循环：每轮中进行相邻元素的比较和交换
        // 第 i 轮比较 (len - 1 - i) 次
        // 为什么要减 i？因为每轮结束后，末尾都有一个元素已经排好序了
        for (int j = 0; j < len - 1 - i; j++) {

            // 如果前面的元素比后面的大，就交换
            if (arr[j] > arr[j + 1]) {
                // 交换 arr[j] 和 arr[j+1]
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }

        // 打印每轮排序后的结果（方便学习理解）
        cout << "第" << i + 1 << "轮排序后: ";
        for (int k = 0; k < len; k++) {
            cout << arr[k] << " ";
        }
        cout << endl;
    }

    // 排序后输出
    cout << "排序后: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

**运行结果：**
```
排序前: 4 2 8 0 5
第1轮排序后: 2 4 0 5 8
第2轮排序后: 2 0 4 5 8
第3轮排序后: 0 2 4 5 8
第4轮排序后: 0 2 4 5 8
排序后: 0 2 4 5 8
```

## 四、详细代码解析

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {4, 2, 8, 0, 5};
    int len = sizeof(arr) / sizeof(arr[0]);  // len = 5

    // ==================== 外层循环详解 ====================
    // i 表示当前是第几轮（从0开始计数）
    // 总共需要 len-1 = 4 轮
    // i = 0: 第1轮
    // i = 1: 第2轮
    // i = 2: 第3轮
    // i = 3: 第4轮
    for (int i = 0; i < len - 1; i++) {

        // ==================== 内层循环详解 ====================
        // j 表示当前比较到哪个位置
        // 每轮比较的次数 = len - 1 - i
        //
        // 第1轮(i=0): j从0到3，比较4次 → j < 5-1-0 = 4
        // 第2轮(i=1): j从0到2，比较3次 → j < 5-1-1 = 3
        // 第3轮(i=2): j从0到1，比较2次 → j < 5-1-2 = 2
        // 第4轮(i=3): j从0到0，比较1次 → j < 5-1-3 = 1
        for (int j = 0; j < len - 1 - i; j++) {

            // 比较相邻的两个元素
            // arr[j] 是当前元素，arr[j+1] 是下一个元素
            if (arr[j] > arr[j + 1]) {
                // 交换（三步交换法）
                int temp = arr[j];      // 1. 保存当前值到临时变量
                arr[j] = arr[j + 1];    // 2. 把后面的值放到前面
                arr[j + 1] = temp;      // 3. 把临时变量（原来的前面值）放到后面
            }
        }
    }

    return 0;
}
```

## 五、优化版本：提前结束

如果在某一轮中没有发生任何交换，说明数组已经有序了，可以提前结束排序。

```cpp
#include <iostream>
using namespace std;

int main() {
    // 用一个接近有序的数组来演示优化效果
    int arr[] = {1, 2, 3, 5, 4};
    int len = sizeof(arr) / sizeof(arr[0]);

    cout << "排序前: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    int swapCount = 0;  // 统计总交换次数

    // ========== 优化版冒泡排序 ==========
    for (int i = 0; i < len - 1; i++) {

        bool swapped = false;  // 标记本轮是否发生了交换

        for (int j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;

                swapped = true;  // 标记发生了交换
                swapCount++;
            }
        }

        cout << "第" << i + 1 << "轮结束, ";
        if (swapped) {
            cout << "有交换发生" << endl;
        } else {
            cout << "没有交换，数组已有序，提前结束！" << endl;
            break;  // 提前退出循环
        }
    }

    cout << "总交换次数: " << swapCount << endl;

    cout << "排序后: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

**运行结果：**
```
排序前: 1 2 3 5 4
第1轮结束, 有交换发生
第2轮结束, 没有交换，数组已有序，提前结束！
总交换次数: 1
排序后: 1 2 3 4 5
```

**优化效果：** 原本需要4轮，但第2轮发现没有交换就提前结束了，节省了时间。

## 六、封装成函数

```cpp
#include <iostream>
using namespace std;

// 冒泡排序函数
// 参数说明：
//   arr[] - 要排序的数组
//   len   - 数组长度
void bubbleSort(int arr[], int len) {
    for (int i = 0; i < len - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < len - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) {
            break;  // 优化：没有交换就提前结束
        }
    }
}

// 打印数组函数
void printArray(int arr[], int len) {
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[] = {64, 34, 25, 12, 22, 11, 90};
    int len = sizeof(arr) / sizeof(arr[0]);

    cout << "排序前: ";
    printArray(arr, len);

    bubbleSort(arr, len);  // 调用冒泡排序

    cout << "排序后: ";
    printArray(arr, len);

    return 0;
}
```

## 七、从大到小排序

如果想从大到小排序，只需要把比较条件中的 `>` 改成 `<`。

```cpp
#include <iostream>
using namespace std;

// 从大到小排序
void bubbleSortDesc(int arr[], int len) {
    for (int i = 0; i < len - 1; i++) {
        bool swapped = false;
        for (int j = 0; j < len - 1 - i; j++) {
            // 只需要把 > 改成 < 即可实现从大到小
            if (arr[j] < arr[j + 1]) {  // 注意这里是 <
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapped = true;
            }
        }
        if (!swapped) {
            break;
        }
    }
}

void printArray(int arr[], int len) {
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

int main() {
    int arr[] = {4, 2, 8, 0, 5};
    int len = sizeof(arr) / sizeof(arr[0]);

    cout << "排序前: ";
    printArray(arr, len);

    bubbleSortDesc(arr, len);

    cout << "从大到小排序后: ";
    printArray(arr, len);
    // 输出: 8 5 4 2 0

    return 0;
}
```

## 八、复杂度分析

### 时间复杂度

```
对于 n 个元素：
- 最好情况：数组已经有序，只需1轮（优化版），O(n)
- 最坏情况：数组完全逆序，需要 n-1 轮，每轮比较次数递减
  总比较次数 = (n-1) + (n-2) + ... + 1 = n*(n-1)/2
  所以时间复杂度 = O(n²)
- 平均情况：O(n²)
```

```cpp
#include <iostream>
using namespace std;

int main() {
    // 用计数器来观察实际的比较次数和交换次数
    int arr[] = {5, 4, 3, 2, 1};  // 完全逆序，最坏情况
    int len = sizeof(arr) / sizeof(arr[0]);

    int compareCount = 0;  // 比较次数
    int swapCount = 0;     // 交换次数

    for (int i = 0; i < len - 1; i++) {
        for (int j = 0; j < len - 1 - i; j++) {
            compareCount++;  // 每次进入内层循环就比较一次
            if (arr[j] > arr[j + 1]) {
                int temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
                swapCount++;
            }
        }
    }

    cout << "元素个数: " << len << endl;
    cout << "比较次数: " << compareCount << endl;  // 10 = 5*4/2
    cout << "交换次数: " << swapCount << endl;     // 10（完全逆序时交换最多）

    // n*(n-1)/2 = 5*4/2 = 10

    return 0;
}
```

### 空间复杂度

```
冒泡排序只需要一个临时变量 temp 来交换元素，
不需要额外的数组空间，
所以空间复杂度 = O(1)
```

### 复杂度总结

| 指标 | 复杂度 | 说明 |
|------|--------|------|
| 最好时间 | O(n) | 数组已有序（优化版） |
| 最坏时间 | O(n²) | 数组完全逆序 |
| 平均时间 | O(n²) | 一般情况 |
| 空间 | O(1) | 只用了一个临时变量 |
| 稳定性 | 稳定 | 相等元素不会交换位置 |

## 九、常见错误

### 错误1：外层循环条件写错

```cpp
// 错误写法1：i < len（多循环了一轮）
for (int i = 0; i < len; i++) {      // 错！应该是 len-1

// 错误写法2：i <= len - 1 和 i < len 等价，也多了一轮
for (int i = 0; i <= len - 1; i++) {  // 错！

// 正确写法：
for (int i = 0; i < len - 1; i++) {   // 对！n个元素只需要n-1轮
```

### 错误2：内层循环条件写错

```cpp
// 错误写法1：j < len - 1（没有减去 i）
for (int j = 0; j < len - 1; j++) {      // 错！每轮都比较同样的次数

// 错误写法2：j < len（完全没减）
for (int j = 0; j < len; j++) {           // 错！会越界访问 arr[j+1]

// 正确写法：
for (int j = 0; j < len - 1 - i; j++) {  // 对！
```

### 错误3：交换变量时出错

```cpp
// 错误写法：没有用临时变量，导致数据丢失
arr[j] = arr[j + 1];       // arr[j] 的原值被覆盖了！
arr[j + 1] = arr[j];       // 这里拿到的已经是 arr[j+1] 的值了，等于没交换

// 正确写法：用临时变量
int temp = arr[j];          // 先保存 arr[j] 的值
arr[j] = arr[j + 1];       // 把 arr[j+1] 的值放到 arr[j]
arr[j + 1] = temp;         // 把原来的 arr[j] 值放到 arr[j+1]
```

### 错误4：数组越界

```cpp
// 内层循环中访问 arr[j+1]，如果 j 最大到 len-1，
// 那 arr[j+1] = arr[len] 就越界了！
// 所以 j 必须小于 len-1-i（最大到 len-2-i，此时 j+1 = len-1-i）
```

## 十、完整注释版代码（适合复制学习）

```cpp
#include <iostream>
using namespace std;

int main() {
    // ==================== 定义数组 ====================
    int arr[] = {4, 2, 8, 0, 5};
    int len = sizeof(arr) / sizeof(arr[0]);  // 计算数组长度 = 5

    // ==================== 排序前输出 ====================
    cout << "排序前: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    // ==================== 冒泡排序 ====================
    // 外层循环：控制排序轮数
    // n个元素需要 n-1 轮（i从0到n-2，共n-1次）
    for (int i = 0; i < len - 1; i++) {

        // 标记本轮是否有交换发生（用于优化）
        bool swapped = false;

        // 内层循环：每轮进行相邻元素的比较和交换
        // 第i轮比较 len-1-i 次
        // 因为每轮结束后，末尾都会多一个已排好序的元素
        for (int j = 0; j < len - 1 - i; j++) {

            // 比较相邻两个元素
            if (arr[j] > arr[j + 1]) {
                // 如果前面的比后面的大，就交换

                // 交换三步曲：
                int temp = arr[j];      // 第1步：把 arr[j] 存到临时变量
                arr[j] = arr[j + 1];    // 第2步：把 arr[j+1] 的值放到 arr[j]
                arr[j + 1] = temp;      // 第3步：把临时变量的值放到 arr[j+1]

                swapped = true;  // 标记发生了交换
            }
        }

        // 打印每轮结果（方便学习调试）
        cout << "第" << i + 1 << "轮: ";
        for (int k = 0; k < len; k++) {
            cout << arr[k] << " ";
        }
        cout << endl;

        // 优化：如果本轮没有交换，说明已经有序，提前结束
        if (!swapped) {
            cout << "没有交换发生，提前结束排序！" << endl;
            break;
        }
    }

    // ==================== 排序后输出 ====================
    cout << "排序后: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 十一、本节要点总结

1. **核心思想**：相邻元素比较，大的往后移
2. **外层循环**：控制轮数，n个元素需要 n-1 轮
3. **内层循环**：控制每轮比较次数，第i轮比较 n-1-i 次
4. **交换方法**：必须用临时变量 temp，三步完成交换
5. **优化方法**：如果某轮没有交换，说明已有序，提前结束
6. **时间复杂度**：O(n²)（最坏和平均），O(n)（最好，优化版）
7. **空间复杂度**：O(1)
8. **稳定性**：稳定排序
9. **适用场景**：数据量小、对性能要求不高的场合
