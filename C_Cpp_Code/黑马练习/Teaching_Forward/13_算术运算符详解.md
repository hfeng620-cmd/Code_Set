# 13 - 算术运算符详解

## 一、什么是算术运算符？

算术运算符就是我们小学学过的加减乘除，只不过在 C++ 中用特定的符号来表示。
它们用于对数据进行数学计算，是最基本、最常用的运算符。

## 二、五种基本算术运算符

| 运算符 | 含义 | 示例 | 结果 |
|--------|------|------|------|
| `+`    | 加法 | `3 + 2` | `5` |
| `-`    | 减法 | `3 - 2` | `1` |
| `*`    | 乘法 | `3 * 2` | `6` |
| `/`    | 除法 | `7 / 2` | `3`（整数除法）|
| `%`    | 取模（求余数）| `7 % 2` | `1` |

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 3;

    cout << "加法: " << a + b << endl;   // 输出 13
    cout << "减法: " << a - b << endl;   // 输出 7
    cout << "乘法: " << a * b << endl;   // 输出 30
    cout << "除法: " << a / b << endl;   // 输出 3（不是3.333！）
    cout << "取模: " << a % b << endl;   // 输出 1（10除以3余1）

    return 0;
}
```

## 三、整数除法 vs 浮点除法（重点！）

这是初学者最容易踩的坑！

### 3.1 整数除法——结果只保留整数部分

当除号 `/` 的两边**都是整数**时，执行的是**整数除法**，
结果会**直接丢弃小数部分**（不是四舍五入，是直接砍掉！）。

```cpp
int a = 7, b = 2;
int result = a / b;
cout << result << endl;   // 输出 3，不是 3.5！小数部分 0.5 被丢弃了

cout << 10 / 3 << endl;   // 输出 3（不是 3.333）
cout << 1  / 3 << endl;   // 输出 0（不是 0.333）
cout << 5  / 7 << endl;   // 输出 0（5比7小，连1都不到）
```

### 3.2 浮点除法——保留小数

只要除号 `/` 的**任意一边**是浮点数（`float` 或 `double`），
就会执行**浮点除法**，结果会保留小数部分。

```cpp
// 方法1：直接用浮点数
double result1 = 7.0 / 2.0;
cout << result1 << endl;   // 输出 3.5

// 方法2：强制类型转换（把整数转成浮点数再除）
int a = 7, b = 2;
double result2 = (double)a / (double)b;
cout << result2 << endl;   // 输出 3.5

// 方法3：至少一边是浮点数即可
double result3 = 7.0 / 2;     // 左边是浮点，输出 3.5
double result4 = 7 / 2.0;     // 右边是浮点，输出 3.5
double result5 = (double)7 / 2; // 强转左边，输出 3.5
```

### 3.3 经典面试题：计算两个整数的平均值

```cpp
int a = 60, b = 70;

// 错误写法：整数除法，结果可能丢精度
int avg1 = (a + b) / 2;       // (60+70)/2 = 130/2 = 65，碰巧整除没问题

int c = 61, d = 70;
int avg2 = (c + d) / 2;       // (61+70)/2 = 131/2 = 65（实际65.5，精度丢失！）

// 正确写法：使用浮点除法
double avg3 = (double)(c + d) / 2;   // 65.5
cout << avg3 << endl;
```

## 四、除以0的问题（致命错误！）

### 4.1 整数除以0——程序崩溃

```cpp
int a = 10, b = 0;
// int result = a / b;   // 致命错误！程序会直接崩溃（运行时报错）
// 任何整数都不能除以0，这是未定义行为
```

### 4.2 浮点除以0——不会崩溃，但结果特殊

```cpp
double a = 10.0, b = 0.0;
cout << a / b << endl;     // 输出 inf（无穷大）

double c = 0.0, d = 0.0;
cout << c / d << endl;     // 输出 -nan（非数字，Not a Number）
```

> 提示：在实际项目中，做除法之前一定要检查除数是否为0！

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 100, b = 0;

    if (b != 0) {
        cout << "结果: " << a / b << endl;
    } else {
        cout << "错误：除数不能为0！" << endl;
    }

    return 0;
}
```

## 五、取模运算符 % 详解

### 5.1 基本规则

取模运算就是求**除法的余数**。注意：`%` 只能用于**整数**，不能用于浮点数！

```cpp
cout << 10 % 3 << endl;    // 10 ÷ 3 = 3 余 1，输出 1
cout << 10 % 5 << endl;    // 10 ÷ 5 = 2 余 0，输出 0（整除）
cout << 7  % 2 << endl;    // 7  ÷ 2 = 3 余 1，输出 1
cout << 1  % 10 << endl;   // 1  ÷ 10 = 0 余 1，输出 1

// double a = 3.14;
// cout << a % 2 << endl;  // 编译错误！%不能用于浮点数
```

### 5.2 负数取模的规则

取模结果的符号**与被除数（左边的数）保持一致**。

```cpp
cout <<  10 %  3 << endl;   //  1（符号跟被除数 10，为正）
cout << -10 %  3 << endl;   // -1（符号跟被除数-10，为负）
cout <<  10 % -3 << endl;   //  1（符号跟被除数 10，为正）
cout << -10 % -3 << endl;   // -1（符号跟被除数-10，为负）

// 记忆口诀：余数符号看左边（被除数）
```

### 5.3 应用一：判断奇偶数

一个数对 2 取模，结果为 0 就是偶数，结果不为 0 就是奇数。

```cpp
#include <iostream>
using namespace std;

int main() {
    int num;
    cout << "请输入一个整数: ";
    cin >> num;

    if (num % 2 == 0) {
        cout << num << " 是偶数" << endl;
    } else {
        cout << num << " 是奇数" << endl;
    }

    // 示例输出：
    // 输入 4  -> "4 是偶数"
    // 输入 7  -> "7 是奇数"
    // 输入 -3 -> "-3 是奇数"（-3 % 2 = -1，不等于0，所以是奇数）

    return 0;
}
```

### 5.4 应用二：循环计数（周期性问题）

取模可以让数值在 0 到 N-1 之间循环，非常适用于周期性场景。

```cpp
#include <iostream>
using namespace std;

int main() {
    // 模拟星期循环：0=周一, 1=周二, ..., 6=周日
    string weekName[] = {"周一", "周二", "周三", "周四", "周五", "周六", "周日"};

    for (int day = 0; day < 14; day++) {
        int weekday = day % 7;   // 结果始终在 0~6 之间循环
        cout << "第" << day << "天是" << weekName[weekday] << endl;
    }
    // 第0天是周一, 第1天是周二, ... 第6天是周日, 第7天又是周一...

    return 0;
}
```

### 5.5 应用三：提取整数的各位数字

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 4567;

    // 提取个位：对10取模
    int ge = num % 10;            // 7
    cout << "个位: " << ge << endl;

    // 提取十位：先去掉个位（除以10），再对10取模
    int shi = (num / 10) % 10;    // 6
    cout << "十位: " << shi << endl;

    // 提取百位
    int bai = (num / 100) % 10;   // 5
    cout << "百位: " << bai << endl;

    // 提取千位
    int qian = (num / 1000) % 10; // 4
    cout << "千位: " << qian << endl;

    // 练习：输入任意三位数，输出其各位数字之和
    int n;
    cout << "请输入一个三位数: ";
    cin >> n;

    int g = n % 10;              // 个位
    int s = (n / 10) % 10;      // 十位
    int b = (n / 100) % 10;     // 百位
    cout << "各位数字之和: " << g + s + b << endl;

    return 0;
}
```

### 5.6 应用四：判断能否整除

```cpp
#include <iostream>
using namespace std;

int main() {
    // 判断一个数能否被另一个数整除
    int year = 2024;

    // 闰年判断：能被4整除但不能被100整除，或者能被400整除
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0)) {
        cout << year << " 是闰年" << endl;
    } else {
        cout << year << " 不是闰年" << endl;
    }
    // 输出：2024 是闰年

    // 求所有三位数中的水仙花数
    // 水仙花数：各位数字的立方和等于该数本身
    // 例如：153 = 1^3 + 5^3 + 3^3
    cout << "三位数中的水仙花数：" << endl;
    for (int i = 100; i <= 999; i++) {
        int ge  = i % 10;
        int shi = i / 10 % 10;
        int bai = i / 100;
        if (ge*ge*ge + shi*shi*shi + bai*bai*bai == i) {
            cout << i << endl;
        }
    }
    // 输出：153  370  371  407

    return 0;
}
```

## 六、算术运算中的类型转换

当不同类型的值混合运算时，小类型会自动提升为大类型。

```cpp
#include <iostream>
using namespace std;

int main() {
    // 规则：小类型自动转为大类型
    // char < short < int < long < long long < float < double

    // int 和 double 运算，int 转为 double
    int a = 10;
    double b = 3.0;
    auto result = a + b;  // result 是 double 类型
    cout << "10 + 3.0 = " << result << endl;  // 13

    // char 参与运算时会转为 int
    char ch = 'A';  // ASCII码 65
    int x = ch + 1;
    cout << "'A' + 1 = " << x << endl;        // 66
    cout << "转为字符: " << (char)x << endl;   // B

    // 除法中的类型陷阱（再强调一次！）
    cout << "整数除法: 5 / 2 = " << 5 / 2 << endl;         // 2
    cout << "浮点除法: 5.0 / 2 = " << 5.0 / 2 << endl;    // 2.5
    cout << "强制转换: (double)5 / 2 = " << (double)5 / 2 << endl; // 2.5

    return 0;
}
```

## 七、综合练习

### 练习1：时间转换

把秒数转换为 X小时X分X秒 的格式。

```cpp
#include <iostream>
using namespace std;

int main() {
    int totalSeconds;
    cout << "请输入总秒数: ";
    cin >> totalSeconds;

    int hours   = totalSeconds / 3600;         // 1小时=3600秒
    int minutes = (totalSeconds % 3600) / 60;  // 去掉小时后，剩下的秒数除以60
    int seconds = totalSeconds % 60;           // 去掉分钟后，剩下的秒数

    cout << totalSeconds << "秒 = "
         << hours << "小时"
         << minutes << "分"
         << seconds << "秒" << endl;

    // 示例：输入 3661
    // 3661 / 3600 = 1（小时）
    // 3661 % 3600 = 61，61 / 60 = 1（分）
    // 3661 % 60 = 1（秒）
    // 输出：3661秒 = 1小时1分1秒

    return 0;
}
```

### 练习2：温度转换（华氏度转摄氏度）

```cpp
#include <iostream>
using namespace std;

int main() {
    // 公式：C = (F - 32) * 5 / 9
    double fahrenheit = 98.6;
    double celsius = (fahrenheit - 32) * 5.0 / 9.0;
    // 注意：这里必须用 5.0 / 9.0，如果写 5/9 结果是 0！
    cout << fahrenheit << "F = " << celsius << "C" << endl;
    // 输出：98.6F = 37C

    return 0;
}
```

### 练习3：交换两个整数的值

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;
    cout << "交换前: a=" << a << ", b=" << b << endl;

    // 方法1：使用临时变量（推荐，最清晰）
    int temp = a;
    a = b;
    b = temp;
    cout << "交换后: a=" << a << ", b=" << b << endl;

    // 方法2：使用算术运算（不需要临时变量）
    a = 10; b = 20;  // 恢复原值
    a = a + b;   // a = 30
    b = a - b;   // b = 30 - 20 = 10（原来的a）
    a = a - b;   // a = 30 - 10 = 20（原来的b）
    cout << "再次交换: a=" << a << ", b=" << b << endl;

    return 0;
}
```

## 八、常见错误总结

| 错误 | 说明 |
|------|------|
| `7 / 2` 期望得到 3.5 | 整数除法只保留整数，应写 `7.0 / 2` |
| `3.14 % 2` | `%` 不能用于浮点数，编译报错 |
| `10 / 0` | 整数除以0导致程序崩溃 |
| 忘记取模结果的符号 | C++中 `-7 % 3` 的结果是 `-1`（余数符号跟被除数一致）|

## 九、总结

1. `+` `-` `*` `/` `%` 是 C++ 的五种算术运算符
2. 整数 `/` 整数 = 整数除法（丢弃小数部分，向0截断）
3. 只要 `/` 两边有一个浮点数，就是浮点除法（保留小数）
4. `%` 取模只能用于整数，结果符号与被除数相同
5. 取模常用于：判断奇偶、循环计数、提取位数、判断整除
6. 除法前一定要检查除数是否为0
7. 想要精确结果时，别忘了把整数转成浮点数再除
