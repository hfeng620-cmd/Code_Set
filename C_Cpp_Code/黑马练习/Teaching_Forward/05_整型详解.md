# C++ 整型（Integer）详解

## 一、什么是整型？

整型就是用来存储**整数**（没有小数点的数）的数据类型。
比如：`0, 1, -1, 42, 100, -999` 这些都是整数。

---

## 二、整型家族

C++ 提供了 4 种整型，它们的区别在于**占用的内存大小不同**，能存储的**数值范围也不同**。

| 类型 | 关键字 | 占用字节 | 大致范围（有符号） |
|------|--------|----------|---------------------|
| 短整型 | `short` | 2 字节 | -32768 ~ 32767 |
| 整型 | `int` | 4 字节 | 约 -21亿 ~ 21亿 |
| 长整型 | `long` | 4 字节 | 约 -21亿 ~ 21亿 |
| 长长整型 | `long long` | 8 字节 | 约 -9.2×10¹⁸ ~ 9.2×10¹⁸ |

```cpp
#include <iostream>
using namespace std;

int main() {
    // ============ 整型家族演示 ============

    // 1. short 短整型（2字节）
    short s = 100;
    cout << "short s = " << s << endl;
    cout << "short 占用字节数: " << sizeof(short) << endl;  // 输出 2

    // 2. int 整型（4字节，最常用）
    int a = 100000;
    cout << "int a = " << a << endl;
    cout << "int 占用字节数: " << sizeof(int) << endl;  // 输出 4

    // 3. long 长整型（4字节，和 int 一样大）
    long l = 100000L;  // L 后缀表示 long 类型（可选）
    cout << "long l = " << l << endl;
    cout << "long 占用字节数: " << sizeof(long) << endl;  // 输出 4

    // 4. long long 长长整型（8字节）
    long long ll = 1000000000000LL;  // LL 后缀表示 long long
    cout << "long long ll = " << ll << endl;
    cout << "long long 占用字节数: " << sizeof(long long) << endl;  // 输出 8

    return 0;
}
```

**输出：**
```
short s = 100
short 占用字节数: 2
int a = 100000
int 占用字节数: 4
long l = 100000
long 占用字节数: 4
long long ll = 1000000000000
long long 占用字节数: 8
```

## 三、有符号（signed）与无符号（unsigned）

### 3.1 概念解释

- **signed（有符号）**：可以存储正数、负数和零。这是**默认**类型。
- **unsigned（无符号）**：只能存储正数和零（没有负数）。

```cpp
#include <iostream>
using namespace std;

int main() {
    // ============ signed vs unsigned ============

    // signed 是默认的，以下两种写法等价：
    int a = -10;           // 默认就是 signed
    signed int b = -10;    // 显式写 signed，效果一样

    // unsigned 只能存非负数
    unsigned int c = 10;   // 正确
    // unsigned int d = -10; // 错误！不能存负数（编译器会给警告）

    cout << "a = " << a << endl;  // -10
    cout << "b = " << b << endl;  // -10
    cout << "c = " << c << endl;  // 10

    return 0;
}
```

### 3.2 为什么 unsigned 能存更大的正数？

以 1 字节（8 位）为例：

```
signed char（有符号）:
  最高位是符号位（0=正，1=负）
  范围：-128 ~ 127（共256个数）

unsigned char（无符号）:
  所有位都用来表示数值
  范围：0 ~ 255（共256个数）
```

**原理：**
- 1 字节 = 8 位，能表示 2⁸ = 256 个不同的值
- signed：一半给负数，一半给正数 → -128 ~ 127
- unsigned：全部给非负数 → 0 ~ 255

```cpp
#include <iostream>
using namespace std;

int main() {
    signed char sc = 127;      // signed char 最大值
    unsigned char uc = 255;    // unsigned char 最大值

    cout << "signed char 最大值: " << (int)sc << endl;    // 127
    cout << "unsigned char 最大值: " << (int)uc << endl;  // 255

    // 注意：char 和 signed char 不一定一样！
    // char 是否有符号取决于编译器实现
    cout << "char 占用字节数: " << sizeof(char) << endl;           // 1
    cout << "signed char 占用字节数: " << sizeof(signed char) << endl; // 1
    cout << "unsigned char 占用字节数: " << sizeof(unsigned char) << endl; // 1

    return 0;
}
```

## 四、使用 climits 查看取值范围

C++ 提供了 `<climits>`（C 语言中是 `<limits.h>`）头文件，里面定义了各种整型的最大值和最小值。

```cpp
#include <iostream>
#include <climits>  // 包含整型的范围信息
using namespace std;

int main() {
    cout << "============ 各种整型的取值范围 ============" << endl;

    // short 的范围
    cout << "short 最小值: " << SHRT_MIN << endl;   // -32768
    cout << "short 最大值: " << SHRT_MAX << endl;   // 32767

    // int 的范围
    cout << "int 最小值: " << INT_MIN << endl;      // -2147483648
    cout << "int 最大值: " << INT_MAX << endl;      // 2147483647

    // long 的范围
    cout << "long 最小值: " << LONG_MIN << endl;    // -2147483648
    cout << "long 最大值: " << LONG_MAX << endl;    // 2147483647

    // long long 的范围
    cout << "long long 最小值: " << LLONG_MIN << endl; // -9223372036854775808
    cout << "long long 最大值: " << LLONG_MAX << endl; // 9223372036854775807

    // unsigned 的范围
    cout << "unsigned int 最大值: " << UINT_MAX << endl;    // 4294967295
    cout << "unsigned long 最大值: " << ULONG_MAX << endl;  // 4294967295
    cout << "unsigned long long 最大值: " << ULLONG_MAX << endl;

    return 0;
}
```

**常用常量表：**

| 常量 | 含义 | 值 |
|------|------|-----|
| `SHRT_MIN` | short 最小值 | -32768 |
| `SHRT_MAX` | short 最大值 | 32767 |
| `USHRT_MAX` | unsigned short 最大值 | 65535 |
| `INT_MIN` | int 最小值 | -2147483648 |
| `INT_MAX` | int 最大值 | 2147483647 |
| `UINT_MAX` | unsigned int 最大值 | 4294967295 |
| `LLONG_MIN` | long long 最小值 | -9223372036854775808 |
| `LLONG_MAX` | long long 最大值 | 9223372036854775807 |

## 五、整型溢出问题

### 5.1 什么是溢出？

当一个变量存储的值**超过了它能表示的最大范围**，就会发生**溢出**。
溢出后数据会"绕回来"，产生不可预期的结果。

### 5.2 正溢出（超过最大值）

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // ============ 正溢出示例 ============

    int max_val = INT_MAX;  // 2147483647
    cout << "INT_MAX = " << max_val << endl;

    // 超过最大值 1，会发生正溢出
    int overflow = max_val + 1;
    cout << "INT_MAX + 1 = " << overflow << endl;
    // 输出：-2147483648（变成了最小值！这就是"绕回来了"）

    // unsigned 的正溢出
    unsigned int umax = UINT_MAX;  // 4294967295
    cout << "UINT_MAX = " << umax << endl;
    cout << "UINT_MAX + 1 = " << umax + 1 << endl;  // 输出 0（绕回到0）

    return 0;
}
```

**输出：**
```
INT_MAX = 2147483647
INT_MAX + 1 = -2147483648
UINT_MAX = 4294967295
UINT_MAX + 1 = 0
```

### 5.3 负溢出（低于最小值）

```cpp
#include <iostream>
#include <climits>
using namespace std;

int main() {
    // ============ 负溢出示例 ============

    int min_val = INT_MIN;  // -2147483648
    cout << "INT_MIN = " << min_val << endl;

    // 低于最小值 1，会发生负溢出
    int underflow = min_val - 1;
    cout << "INT_MIN - 1 = " << underflow << endl;
    // 输出：2147483647（变成了最大值！绕回来了）

    return 0;
}
```

**输出：**
```
INT_MIN = -2147483648
INT_MIN - 1 = 2147483647
```

### 5.4 溢出的危害

溢出会导致程序产生**错误的计算结果**，而且**编译器不会报错**！

```cpp
#include <iostream>
using namespace std;

int main() {
    // 实际开发中容易遇到的溢出场景

    // 场景：两个大数相乘
    int a = 100000;
    int b = 100000;
    int result = a * b;  // 100000 * 100000 = 100亿，超过 int 范围！

    cout << "100000 * 100000 = " << result << endl;
    // 输出错误结果！（溢出了）

    // 正确做法：使用更大的类型
    long long correct_result = (long long)a * b;
    cout << "正确结果: " << correct_result << endl;  // 10000000000

    return 0;
}
```

## 六、整型字面量的表示方式

C++ 支持用不同的进制来表示整数。

### 6.1 四种进制

```cpp
#include <iostream>
using namespace std;

int main() {
    // ============ 四种进制表示 ============

    // 1. 十进制（最常见的，没有前缀）
    int dec = 42;
    cout << "十进制 42: " << dec << endl;  // 42

    // 2. 八进制（以 0 开头）
    int oct = 052;  // 0开头表示八进制
    cout << "八进制 052: " << oct << endl;  // 42（八进制52 = 5×8+2 = 42）

    // 3. 十六进制（以 0x 或 0X 开头）
    int hex = 0x2A;  // 0x开头表示十六进制
    cout << "十六进制 0x2A: " << hex << endl;  // 42（十六进制2A = 2×16+10 = 42）

    // 4. 二进制（以 0b 或 0B 开头，C++14起支持）
    int bin = 0b101010;  // 0b开头表示二进制
    cout << "二进制 0b101010: " << bin << endl;  // 42

    return 0;
}
```

**输出：**
```
十进制 42: 42
八进制 052: 42
十六进制 0x2A: 42
二进制 0b101010: 42
```

### 6.2 用不同格式输出

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 255;

    // 用不同进制输出同一个数
    cout << "十进制: " << dec << num << endl;   // 255
    cout << "八进制: " << oct << num << endl;   // 377
    cout << "十六进制: " << hex << num << endl; // ff

    // 恢复为十进制输出
    cout << dec;  // 注意：设置后会影响后续所有输出，直到再次修改

    // 显示进制前缀
    cout << showbase;  // 显示进制前缀
    cout << "八进制: " << oct << num << endl;   // 0377
    cout << "十六进制: " << hex << num << endl; // 0xff
    cout << "十进制: " << dec << num << endl;   // 255
    cout << noshowbase;  // 关闭进制前缀显示

    return 0;
}
```

### 6.3 字面量后缀

```cpp
#include <iostream>
using namespace std;

int main() {
    // ============ 类型后缀 ============

    auto a = 42;          // int
    auto b = 42L;         // long（L 后缀）
    auto c = 42LL;        // long long（LL 后缀）
    auto d = 42U;         // unsigned int（U 后缀）
    auto e = 42UL;        // unsigned long
    auto f = 42ULL;       // unsigned long long

    cout << "42   -> sizeof: " << sizeof(a) << endl;  // 4
    cout << "42L  -> sizeof: " << sizeof(b) << endl;  // 4
    cout << "42LL -> sizeof: " << sizeof(c) << endl;  // 8
    cout << "42U  -> sizeof: " << sizeof(d) << endl;  // 4

    return 0;
}
```

## 七、整型的选择建议

### 7.1 什么时候用什么类型？

```
1. 一般整数运算 → 用 int（最常用，效率高）
2. 可能超过 21 亿 → 用 long long
3. 需要节省内存（大量数据）→ 用 short
4. 确定不会是负数 → 用 unsigned int
5. 涉及字节操作 → 用 unsigned char
```

### 7.2 实际建议

```cpp
#include <iostream>
using namespace std;

int main() {
    // ====== 推荐做法 ======

    // 1. 日常使用：直接用 int
    int age = 20;
    int score = 95;

    // 2. 大数计算：用 long long
    long long population = 1400000000LL;  // 中国人口
    long long distance = 149597870700LL;  // 地日距离（米）

    // 3. 循环计数器：用 int（除非确实需要更大范围）
    for (int i = 0; i < 100; i++) {
        // 用 int 就够了
    }

    // 4. 不要无脑用 long long
    // 虽然 long long 范围最大，但它占用 8 字节
    // 如果只需要存人的年龄，用 int 就够了

    // 5. 不推荐使用 short（除非确实需要节省内存）
    // short 的范围太小（-32768~32767），容易溢出

    cout << "age = " << age << endl;
    cout << "score = " << score << endl;
    cout << "population = " << population << endl;

    return 0;
}
```

## 八、常见错误和注意事项

### 错误1：混淆 0 开头的数字

```cpp
int a = 010;   // 这是八进制的10，等于十进制的8！不是10！
int b = 10;    // 这才是十进制的10
cout << a << endl;  // 输出 8
cout << b << endl;  // 输出 10
```

### 错误2：整型除法丢失小数

```cpp
int a = 5;
int b = 2;
int result = a / b;   // 整型除法，结果是 2，不是 2.5！
cout << result << endl;  // 输出 2

// 如果想要小数结果，需要类型转换
double correct = (double)a / b;
cout << correct << endl;  // 输出 2.5
```

### 错误3：忘记 unsigned 的减法陷阱

```cpp
unsigned int a = 5;
unsigned int b = 10;
unsigned int result = a - b;  // 5 - 10 会变成一个巨大的正数！
cout << result << endl;  // 输出 4294967291（不是 -5！）

// 原因：unsigned 不能存负数，结果绕回去了
```

### 错误4：大数乘法溢出

```cpp
int a = 100000;
int b = 100000;
int result = a * b;  // 溢出！结果是错的
cout << result << endl;  // 错误输出

// 正确做法
long long correct = 1LL * a * b;  // 先把一个转成 long long
cout << correct << endl;  // 10000000000
```

## 九、总结

| 项目 | 要点 |
|------|------|
| 最常用 | `int`（4字节，范围约±21亿） |
| 大数 | `long long`（8字节） |
| signed vs unsigned | 默认 signed，unsigned 不能存负数但正数范围翻倍 |
| 查范围 | 用 `<climits>` 头文件 |
| 查字节数 | `sizeof(类型)` |
| 溢出 | 超过范围会"绕回来"，编译器不报错，要小心！ |
| 进制 | 十进制直接写、八进制 0 开头、十六进制 0x 开头、二进制 0b 开头 |
| 建议 | 日常用 `int`，大数用 `long long`，不要无脑用最大类型 |
