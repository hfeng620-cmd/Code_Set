# 31 - 输入输出详解

## 一、cout 输出流

`cout` 是 C++ 标准输出流对象，配合 `<<`（插入运算符）使用。

### 1.1 基本输出

```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World!" << endl;

    int a = 10;
    double b = 3.14;
    char c = 'A';

    cout << "整数：" << a << endl;
    cout << "浮点数：" << b << endl;
    cout << "字符：" << c << endl;

    return 0;
}
```

### 1.2 链式输出（连续使用 `<<`）

`<<` 运算符返回 `cout` 的引用，所以可以连续链式调用：

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20, c = 30;

    // 链式输出：一行代码输出多个内容
    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;

    // 等价于
    cout << "a=";
    cout << a;
    cout << ", b=";
    cout << b;
    cout << ", c=";
    cout << c;
    cout << endl;

    return 0;
}
```

## 二、endl vs "\n"（换行方式）

### 2.1 两种换行方式

```cpp
#include <iostream>
using namespace std;

int main() {
    // 方式1：endl（end line）
    cout << "第一行" << endl;
    cout << "第二行" << endl;

    // 方式2："\n"（转义字符）
    cout << "第三行\n";
    cout << "第四行\n";

    // 方式3：'\n'（字符）
    cout << "第五行" << '\n';

    return 0;
}
```

### 2.2 endl vs "\n" 的区别

| 特性 | `endl` | `"\n"` |
|------|--------|--------|
| 功能 | 换行 + **刷新缓冲区** | 只换行 |
| 性能 | 较慢（频繁刷新） | 较快 |
| 适用场景 | 需要立即看到输出时 | 大量输出时 |

```cpp
#include <iostream>
using namespace std;

int main() {
    // 性能测试：endl 比 \n 慢很多
    // 因为 endl 每次都刷新缓冲区

    // 快速输出（推荐用 \n）
    for (int i = 0; i < 10000; i++) {
        cout << i << "\n";   // 更快
    }

    // 较慢输出
    for (int i = 0; i < 10000; i++) {
        cout << i << endl;   // 每次刷新，慢
    }

    return 0;
}
```

**建议**：日常使用 `"\n"` 即可，调试时用 `endl` 确保立即输出。

## 三、格式化输出

使用 `<iomanip>` 头文件中的格式化函数。

### 3.1 setw：设置输出宽度

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int a = 42;

    // setw 设置最小宽度，默认右对齐
    cout << "[" << setw(10) << a << "]" << endl;
    // 输出：[        42]
    // 总宽度10，42占2位，左边补8个空格

    cout << "[" << setw(3) << a << "]" << endl;
    // 输出：[ 42]
    // setw 只是最小宽度，数据超过不会截断

    // setw 只影响紧跟着的一个输出
    cout << setw(5) << 1 << setw(5) << 2 << setw(5) << 3 << endl;
    // 输出：    1    2    3

    return 0;
}
```

### 3.2 setfill：设置填充字符

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    // 默认填充空格
    cout << "[" << setw(10) << 42 << "]" << endl;
    // 输出：[        42]

    // 设置填充字符为 '*'
    cout << setfill('*');
    cout << "[" << setw(10) << 42 << "]" << endl;
    // 输出：[********42]

    // 设置填充字符为 '0'（常用于补零）
    cout << setfill('0');
    cout << setw(5) << 42 << endl;
    // 输出：00042

    // 恢复填充空格
    cout << setfill(' ');

    return 0;
}
```

### 3.3 left / right：设置对齐方式

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    // 默认右对齐
    cout << setw(10) << "Hello" << "|" << endl;
    // 输出：     Hello|

    // 设置左对齐
    cout << left;
    cout << setw(10) << "Hello" << "|" << endl;
    // 输出：Hello     |

    // 恢复右对齐
    cout << right;
    cout << setw(10) << "Hello" << "|" << endl;
    // 输出：     Hello|

    return 0;
}
```

### 3.4 setprecision 和 fixed：控制小数位数

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    double pi = 3.14159265358979;

    // 默认精度（总共显示几位有效数字）
    cout << setprecision(4) << pi << endl;   // 3.142（4位有效数字）

    // fixed：固定小数位数模式
    cout << fixed << setprecision(2) << pi << endl;   // 3.14
    cout << fixed << setprecision(4) << pi << endl;   // 3.1416
    cout << fixed << setprecision(6) << pi << endl;   // 3.141593

    // 取消 fixed
    cout << defaultfloat;
    cout << setprecision(6) << pi << endl;   // 3.14159（有效数字模式）

    return 0;
}
```

### 3.5 boolalpha：输出 true/false

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    bool flag = true;

    // 默认输出 1/0
    cout << flag << endl;       // 1

    // 设置输出 true/false
    cout << boolalpha;
    cout << flag << endl;       // true
    cout << false << endl;      // false

    // 取消 boolalpha
    cout << noboolalpha;
    cout << flag << endl;       // 1

    return 0;
}
```

### 3.6 hex / oct / dec：进制输出

```cpp
#include <iostream>
#include <iomanip>
using namespace std;

int main() {
    int num = 255;

    // 十进制（默认）
    cout << dec << num << endl;     // 255

    // 十六进制
    cout << hex << num << endl;     // ff

    // 八进制
    cout << oct << num << endl;     // 377

    // 带前缀显示
    cout << showbase;
    cout << dec << num << endl;     // 255
    cout << hex << num << endl;     // 0xff
    cout << oct << num << endl;     // 0377

    // 大写十六进制
    cout << uppercase << hex << num << endl;   // 0XFF

    // 恢复默认
    cout << nouppercase << noshowbase << dec;

    return 0;
}
```

## 四、cin 输入流

`cin` 是 C++ 标准输入流对象，配合 `>>`（提取运算符）使用。

### 4.1 基本输入

```cpp
#include <iostream>
using namespace std;

int main() {
    int age;
    double height;
    string name;

    cout << "请输入年龄：";
    cin >> age;

    cout << "请输入身高：";
    cin >> height;

    cout << "请输入姓名：";
    cin >> name;

    cout << "姓名：" << name << "，年龄：" << age << "，身高：" << height << endl;

    return 0;
}
```

### 4.2 链式输入

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b, c;

    // 一次输入三个数（用空格或回车分隔）
    cout << "输入三个整数：";
    cin >> a >> b >> c;

    cout << "a=" << a << ", b=" << b << ", c=" << c << endl;

    return 0;
}
```

## 五、cin 缓冲区问题

### 5.1 缓冲区残留换行符

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int num;
    string line;

    cout << "输入一个数字：";
    cin >> num;

    // 问题：cin >> num 只读取数字，换行符留在缓冲区
    // getline 读到换行符就停了，得到空字符串！
    cout << "输入一行文字：";
    getline(cin, line);   // 这里会读到空字符串！

    cout << "你输入的是：[" << line << "]" << endl;   // []

    return 0;
}
```

### 5.2 解决方案：cin.ignore

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int num;
    string line;

    cout << "输入一个数字：";
    cin >> num;

    // 忽略缓冲区中的换行符
    cin.ignore(numeric_limits<streamsize>::max(), '\n');

    cout << "输入一行文字：";
    getline(cin, line);   // 现在能正确读取了

    cout << "你输入的是：[" << line << "]" << endl;

    return 0;
}
```

### 5.3 cin.ignore 的参数

```cpp
// ignore(数量, 终止字符)
cin.ignore(100, '\n');   // 最多忽略100个字符，遇到换行符停止

// 忽略所有直到换行符
cin.ignore(numeric_limits<streamsize>::max(), '\n');
// 需要 #include <limits>
```

### 5.4 cin.clear：清除错误标志

```cpp
#include <iostream>
#include <limits>
using namespace std;

int main() {
    int num;
    cout << "输入一个整数：";
    cin >> num;

    // 如果用户输入了非数字（如 "abc"），cin 会进入错误状态
    if (cin.fail()) {
        cout << "输入错误！" << endl;

        // 清除错误标志
        cin.clear();

        // 清空缓冲区中的错误输入
        cin.ignore(numeric_limits<streamsize>::max(), '\n');

        cout << "请重新输入：";
        cin >> num;
    }

    cout << "你输入的是：" << num << endl;

    return 0;
}
```

## 六、getline 读取一行

### 6.1 基本用法

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string line;

    cout << "输入一行文字：";
    getline(cin, line);   // 读取整行，包括空格

    cout << "你输入的是：" << line << endl;

    return 0;
}
```

### 6.2 cin >> vs getline 的区别

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string s1, s2;

    // cin >> 遇到空格就停
    cout << "cin >> 输入：";
    cin >> s1;
    // 输入 "Hello World"
    cout << "读到的：" << s1 << endl;   // Hello

    // getline 读整行
    cin.ignore();   // 清除换行符
    cout << "getline 输入：";
    getline(cin, s2);
    // 输入 "Hello World"
    cout << "读到的：" << s2 << endl;   // Hello World

    return 0;
}
```

### 6.3 自定义分隔符

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string line;

    cout << "输入（遇到 # 停止）：";
    getline(cin, line, '#');   // 用 '#' 作为分隔符

    cout << "读到的：" << line << endl;

    return 0;
}
```

## 七、cin.get 与 getline

### 7.1 cin.get()：读取单个字符

```cpp
#include <iostream>
using namespace std;

int main() {
    char ch;

    cout << "输入一个字符：";
    ch = cin.get();   // 读取一个字符（包括空格和换行）

    cout << "ASCII码：" << (int)ch << endl;

    return 0;
}
```

### 7.2 cin.get() 读取字符数组

```cpp
#include <iostream>
using namespace std;

int main() {
    char buffer[100];

    cout << "输入一行：";
    cin.get(buffer, 100);   // 最多读99个字符

    cout << "读到的：" << buffer << endl;

    return 0;
}
```

### 7.3 cin.get() vs getline() 的区别

两者都能读取含空格的字符串，但处理分隔符的方式不同：

```cpp
#include <iostream>
using namespace std;

int main() {
    char buf1[10], buf2[10];

    // cin.get(buf, n)：遇到换行符停止，但不提取换行符（留在缓冲区）
    cin.get(buf1, 10);
    // 输入 "Hello" 后回车
    // buf1 = "Hello"，换行符仍在缓冲区

    // cin.getline(buf, n)：遇到换行符停止，并提取（丢弃）换行符
    cin.getline(buf2, 10);
    // buf2 能正确读取下一行

    return 0;
}
```

## 八、常见错误汇总

### 错误1：混合使用 cin >> 和 getline

```cpp
int num;
string line;

cin >> num;
// 错误：换行符残留，getline 读到空串
getline(cin, line);

// 正确：先忽略换行符
cin >> num;
cin.ignore();
getline(cin, line);
```

### 错误2：输入类型不匹配

```cpp
int num;
cin >> num;
// 如果输入 "abc"，cin 进入错误状态
// 后续所有 cin 操作都会失败！

// 正确做法：检查并处理错误
if (cin.fail()) {
    cin.clear();
    cin.ignore(1000, '\n');
    cout << "请输入有效数字！" << endl;
}
```

### 错误3：缓冲区溢出（C风格字符串）

```cpp
char name[5];   // 只能存4个字符 + '\0'
cin >> name;    // 输入超过4个字符就会溢出！

// 安全做法：使用 string
string name2;
cin >> name2;   // string 自动管理内存，不会溢出
```

## 十、总结

| 知识点 | 要点 |
|--------|------|
| `cout <<` | 输出，支持链式调用 |
| `endl` vs `"\n"` | endl 刷新缓冲区，\n 更快 |
| `cin >>` | 输入，遇空格停止 |
| `getline()` | 读整行，含空格 |
| 缓冲区问题 | cin >> 后用 getline 需先 ignore |
| 错误处理 | `cin.clear()` + `cin.ignore()` |
| 格式化 | `<iomanip>` 中的 setw、setfill、setprecision 等 |
