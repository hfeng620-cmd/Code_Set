# 静态成员详解

## 一、静态成员变量

### 1.1 什么是静态成员变量

在类中用 `static` 关键字修饰的成员变量叫做**静态成员变量**。

特点：
1. **所有对象共享**：不管创建多少个对象，大家共用同一份数据
2. **不占对象空间**：`sizeof` 对象时不包含静态成员变量

### 1.2 基本用法

```cpp
#include <iostream>
using namespace std;

class Person
{
public:
    string name;
    int age;
    static int count;  // 静态成员变量：所有对象共享
};

// 类外初始化（必须做，否则链接错误）
int Person::count = 0;

int main()
{
    Person p1;
    p1.name = "张三";
    p1.age = 20;

    Person p2;
    p2.name = "李四";
    p2.age = 21;

    cout << "p1.count = " << p1.count << endl;  // 0
    cout << "p2.count = " << p2.count << endl;  // 0

    p1.count = 100;
    cout << "修改后 p2.count = " << p2.count << endl;  // 100

    return 0;
}
```

### 1.3 类内声明，类外初始化

```cpp
class Person
{
public:
    static int count;  // 类内：声明
};

int Person::count = 0;  // 类外：初始化
```

### 1.4 两种访问方式

```cpp
// 方式一：通过对象访问
Person p;
p.count = 10;

// 方式二：通过类名访问（推荐）
Person::count = 20;
```

## 二、静态成员变量的特点

### 2.1 共享性

```cpp
#include <iostream>
using namespace std;

class Student
{
public:
    string name;
    static int totalCount;
};
int Student::totalCount = 0;

int main()
{
    Student s1;
    Student::totalCount++;  // 1

    Student s2;
    Student::totalCount++;  // 2

    Student s3;
    Student::totalCount++;  // 3

    // 所有对象看到的都是同一个值
    cout << "s1: " << s1.totalCount << endl;  // 3
    cout << "s2: " << s2.totalCount << endl;  // 3
    cout << "s3: " << s3.totalCount << endl;  // 3

    return 0;
}
```

### 2.2 不占对象空间

```cpp
class Person
{
public:
    int age;            // 4 字节
    static int count;   // 不占对象空间
};
int Person::count = 0;

int main()
{
    Person p;
    cout << "sizeof(Person) = " << sizeof(Person) << endl;  // 4
    // count 不算在内

    cout << "p.age 地址: " << &p.age << endl;
    cout << "count 地址: " << &Person::count << endl;
    // 地址差距大，说明在不同区域

    return 0;
}
```

### 2.3 存放在全局区

```cpp
// 普通成员变量：存放在对象中（栈区）
// 静态成员变量：存放在全局区（独立于对象）
// 这就是为什么 sizeof 不包含静态成员
```

## 三、静态成员函数

### 3.1 特点

1. **所有对象共享**：可以通过类名直接调用
2. **只能访问静态成员**：不能访问普通成员变量
3. **没有 `this` 指针**

### 3.2 基本用法

```cpp
#include <iostream>
using namespace std;

class Person
{
public:
    string name;
    static int count;

    // 静态成员函数
    static void showCount()
    {
        cout << "人数: " << count << endl;
        // cout << name;  // 错误！不能访问普通成员
    }

    // 普通成员函数
    void showName()
    {
        cout << "姓名: " << name << endl;
        cout << "人数: " << count << endl;  // 可以访问静态成员
    }
};

int Person::count = 0;

int main()
{
    Person p;
    p.name = "张三";
    Person::count = 10;

    Person::showCount();  // 人数: 10（类名调用）
    p.showCount();        // 人数: 10（对象调用）
    p.showName();         // 姓名: 张三

    return 0;
}
```

### 3.3 为什么不能访问普通成员

```cpp
// 普通成员属于某个具体的对象
// 静态函数不知道你在说哪个对象的 name
// 所以不能访问

// 类比：
// 普通成员函数 = 某个人的专属技能
// 静态成员函数 = 全局工具
```

## 四、应用场景

### 4.1 统计对象数量

```cpp
#include <iostream>
using namespace std;

class Student
{
public:
    string name;
    int id;
    static int totalCount;

    Student(string n)
    {
        name = n;
        totalCount++;
        id = totalCount;
        cout << "创建 " << name << "，编号 " << id << endl;
    }

    ~Student()
    {
        totalCount--;
        cout << "销毁 " << name << "，剩余 " << totalCount << endl;
    }

    static int getTotal() { return totalCount; }
};

int Student::totalCount = 0;

int main()
{
    Student s1("张三");
    Student s2("李四");
    Student s3("王五");
    cout << "当前: " << Student::getTotal() << endl;  // 3

    {
        Student s4("赵六");
        cout << "当前: " << Student::getTotal() << endl;  // 4
    }
    cout << "当前: " << Student::getTotal() << endl;  // 3

    return 0;
}
```

### 4.2 单例模式

```cpp
#include <iostream>
using namespace std;

class Singleton
{
private:
    Singleton() {}
    Singleton(const Singleton &) = delete;
    static Singleton *instance;

public:
    static Singleton* getInstance()
    {
        if (instance == nullptr)
            instance = new Singleton();
        return instance;
    }

    void show() { cout << "单例对象" << endl; }

    static void destroy()
    {
        if (instance != nullptr)
        {
            delete instance;
            instance = nullptr;
        }
    }
};

Singleton* Singleton::instance = nullptr;

int main()
{
    Singleton *s1 = Singleton::getInstance();
    Singleton *s2 = Singleton::getInstance();

    s1->show();
    cout << "s1: " << s1 << endl;
    cout << "s2: " << s2 << endl;  // 地址相同

    Singleton::destroy();
    return 0;
}
```

## 五、sizeof 与静态成员

```cpp
class Test
{
public:
    int a;           // 4 字节
    double b;        // 8 字节
    static int c;    // 不占对象空间
    static double d; // 不占对象空间
};

// sizeof(Test) = 16（只有 a 和 b）
// a(4) + 填充(4) + b(8) = 16
```

## 六、对比表

| 对比项 | 普通成员变量 | 静态成员变量 |
|--------|------------|------------|
| 存储位置 | 对象中 | 全局区 |
| 是否共享 | 各一份 | 共享一份 |
| sizeof | 包含 | 不包含 |
| 初始化 | 构造函数 | 类内声明类外初始化 |
| 访问方式 | 对象.成员 | 类名::成员 |

| 对比项 | 普通成员函数 | 静态成员函数 |
|--------|------------|------------|
| this 指针 | 有 | 没有 |
| 访问普通成员 | 能 | 不能 |
| 访问静态成员 | 能 | 能 |
| 调用方式 | 对象.函数() | 类名::函数() |

---

## 总结

| 要点 | 说明 |
|------|------|
| 静态成员变量 | 所有对象共享，存放在全局区 |
| 类内声明类外初始化 | `int Person::count = 0;` |
| 访问方式 | 推荐 `类名::成员` |
| 静态成员函数 | 没有 this，只能访问静态成员 |
| sizeof | 不包含静态成员变量 |
| 应用场景 | 统计对象数量、单例模式、共享数据 |
