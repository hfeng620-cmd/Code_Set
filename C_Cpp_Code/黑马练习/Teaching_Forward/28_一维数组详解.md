# 一维数组详解

## 一、什么是数组？

数组是一种**数据结构**，用来存储**一组相同类型**的数据。

打个比方：
- 变量就像一个"盒子"，只能装一个值
- 数组就像一排"连在一起的盒子"，可以装多个值

数组的特点：
1. **类型相同**：数组中所有元素必须是同一种类型（全部是int，或全部是float等）
2. **连续存储**：数组在内存中是连续存放的，一个挨着一个
3. **固定大小**：数组一旦定义，大小就确定了，不能改变

```
内存示意图：
+----+----+----+----+----+
| 10 | 20 | 30 | 40 | 50 |   <-- int arr[5] = {10,20,30,40,50};
+----+----+----+----+----+
 arr[0] arr[1] arr[2] arr[3] arr[4]
```

## 二、一维数组的三种定义方式

### 方式1：指定大小，完全初始化（最规范）

```cpp
#include <iostream>
using namespace std;

int main() {
    // 定义一个大小为5的int数组，并给每个元素赋初值
    // 语法：数据类型 数组名[元素个数] = {值1, 值2, ...};
    int arr[5] = {10, 20, 30, 40, 50};

    // 输出每个元素（下标从0开始！）
    cout << "arr[0] = " << arr[0] << endl;  // 10
    cout << "arr[1] = " << arr[1] << endl;  // 20
    cout << "arr[2] = " << arr[2] << endl;  // 30
    cout << "arr[3] = " << arr[3] << endl;  // 40
    cout << "arr[4] = " << arr[4] << endl;  // 50

    return 0;
}
```

**注意：** `arr[5]` 表示有5个元素，下标是 0、1、2、3、4，**没有 arr[5]**！

---

### 方式2：不指定大小，由编译器自动推断

```cpp
#include <iostream>
using namespace std;

int main() {
    // 不写 [] 中的数字，编译器会根据初始化列表自动确定大小
    // 初始化列表有5个元素，所以数组大小就是5
    int arr[] = {10, 20, 30, 40, 50};

    // 此时 arr 的大小是5
    cout << "arr[0] = " << arr[0] << endl;  // 10
    cout << "arr[4] = " << arr[4] << endl;  // 50

    // 用 sizeof 计算元素个数（后面会详细讲）
    int len = sizeof(arr) / sizeof(arr[0]);
    cout << "数组元素个数: " << len << endl;  // 输出 5

    return 0;
}
```

**什么时候用这种方式？** 当你已经知道所有初始值，不想手动数个数时很方便。

---

### 方式3：部分初始化（剩余元素自动补0）

```cpp
#include <iostream>
using namespace std;

int main() {
    // 数组大小为5，但只给了3个初始值
    // 剩下的元素自动初始化为 0
    int arr[5] = {10, 20, 30};

    cout << "arr[0] = " << arr[0] << endl;  // 10
    cout << "arr[1] = " << arr[1] << endl;  // 20
    cout << "arr[2] = " << arr[2] << endl;  // 30
    cout << "arr[3] = " << arr[3] << endl;  // 0（自动补0）
    cout << "arr[4] = " << arr[4] << endl;  // 0（自动补0）

    // 特殊情况：如果想把数组全部初始化为0，可以这样写
    int brr[100] = {0};  // 所有100个元素都是0
    cout << "brr[50] = " << brr[50] << endl;  // 0

    return 0;
}
```

**重要规则：** 如果你提供了初始值列表，即使只提供了部分，剩余的数值类型元素会自动补 0。

---

### 三种方式的对比总结

| 定义方式 | 大小 | 初始值 | 适用场景 |
|---------|------|--------|---------|
| `int arr[5] = {1,2,3,4,5};` | 明确指定5 | 完全赋值 | 最规范，推荐初学者使用 |
| `int arr[] = {1,2,3,4,5};` | 自动推断为5 | 完全赋值 | 已知所有初始值时 |
| `int arr[5] = {1,2};` | 明确指定5 | 部分赋值，其余为0 | 只知道部分初始值时 |

## 三、数组的访问（下标从0开始！）

**这是初学者最容易犯的错误：数组下标是从 0 开始的，不是从 1 开始！**

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // 正确的访问方式
    cout << "第1个元素: " << arr[0] << endl;  // 10（下标0 = 第1个）
    cout << "第2个元素: " << arr[1] << endl;  // 20（下标1 = 第2个）
    cout << "第3个元素: " << arr[2] << endl;  // 30（下标2 = 第3个）
    cout << "第4个元素: " << arr[3] << endl;  // 40（下标3 = 第4个）
    cout << "第5个元素: " << arr[4] << endl;  // 50（下标4 = 第5个）

    // 通过下标修改元素的值
    arr[0] = 100;  // 把第一个元素改成100
    cout << "修改后 arr[0] = " << arr[0] << endl;  // 100

    return 0;
}
```

**记忆方法：** 如果数组有 n 个元素，有效下标范围是 `0` 到 `n-1`。

## 四、数组的遍历

### 方法1：普通 for 循环遍历（最常用）

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // 用 for 循环遍历数组
    // i 从 0 开始，到 4 结束（即 arr 的有效下标范围）
    cout << "遍历数组: ";
    for (int i = 0; i < 5; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    // 输出: 10 20 30 40 50

    // 更好的写法：用 sizeof 计算元素个数，避免硬编码
    int len = sizeof(arr) / sizeof(arr[0]);
    cout << "更好的遍历: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

### 方法2：范围 for 循环遍历（C++11 新特性）

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // 范围 for（也叫 for-each）
    // 语法：for (数据类型 变量名 : 数组名)
    // 意思是：依次取出数组中的每个元素，赋值给变量
    cout << "范围for遍历: ";
    for (int val : arr) {
        cout << val << " ";
    }
    cout << endl;
    // 输出: 10 20 30 40 50

    // 如果要修改元素，需要用引用 &
    cout << "每个元素翻倍: ";
    for (int &val : arr) {
        val = val * 2;  // 修改原数组中的值
    }
    for (int val : arr) {
        cout << val << " ";
    }
    cout << endl;
    // 输出: 20 40 60 80 100

    return 0;
}
```

### 方法3：while 循环遍历

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    int i = 0;
    int len = sizeof(arr) / sizeof(arr[0]);
    while (i < len) {
        cout << arr[i] << " ";
        i++;
    }
    cout << endl;

    return 0;
}
```

## 五、数组名的用途

数组名不仅仅是一个名字，它还包含了很多有用的信息。

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // ========== 用途1：计算整个数组占用的字节数 ==========
    cout << "整个数组的字节数: " << sizeof(arr) << endl;
    // 输出: 20（5个int，每个int占4字节，5*4=20）

    // ========== 用途2：计算单个元素占用的字节数 ==========
    cout << "一个元素的字节数: " << sizeof(arr[0]) << endl;
    // 输出: 4（一个int占4字节）

    // ========== 用途3：计算数组的元素个数 ==========
    // 元素个数 = 整个数组的字节数 / 单个元素的字节数
    int len = sizeof(arr) / sizeof(arr[0]);
    cout << "数组元素个数: " << len << endl;
    // 输出: 5

    // ========== 用途4：获取数组的首地址 ==========
    // 数组名就是数组首元素的地址
    cout << "数组名 arr 的值:   " << arr << endl;
    cout << "首元素地址 &arr[0]: " << &arr[0] << endl;
    // 这两个值是相同的！

    // 注意：整个数组的地址 &arr 和首元素地址在数值上相同
    // 但含义不同（类型不同，后面指针章节会详细讲）
    cout << "整个数组地址 &arr:  " << &arr << endl;

    return 0;
}
```

**总结表格：**

| 用法 | 含义 | 示例 |
|------|------|------|
| `sizeof(arr)` | 整个数组的字节数 | 20（5个int） |
| `sizeof(arr[0])` | 一个元素的字节数 | 4（一个int） |
| `sizeof(arr)/sizeof(arr[0])` | 元素个数 | 5 |
| `arr` | 首元素地址 | 等于 `&arr[0]` |

## 六、数组的内存特点

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // 打印每个元素的地址
    cout << "各元素的内存地址:" << endl;
    for (int i = 0; i < 5; i++) {
        cout << "arr[" << i << "] 的地址: " << &arr[i]
             << "  值: " << arr[i] << endl;
    }

    // 观察输出的地址，你会发现：
    // 每个元素的地址相差 4（因为一个int占4字节）
    // 例如：假设 arr[0] 地址是 0x6ffe40
    //       那么 arr[1] 地址是 0x6ffe44
    //            arr[2] 地址是 0x6ffe48
    //       ...依次递增4

    return 0;
}
```

**内存特点总结：**
1. **连续存储**：数组元素在内存中是一个挨着一个存放的
2. **地址递增**：每个元素的地址比前一个元素大 `sizeof(元素类型)` 个字节
3. **随机访问**：可以通过下标直接定位到任意元素（时间复杂度 O(1)）

## 七、数组作为函数参数

**重要：当数组作为函数参数传递时，传递的是数组的地址（首地址），而不是整个数组的拷贝！**

```cpp
#include <iostream>
using namespace std;

// ========== 错误示范 ==========
// 在函数内用 sizeof 无法获取数组的真实大小！
// 因为传进来的是地址（指针），sizeof(指针) = 4或8字节（取决于系统）
void printArrayWrong(int arr[]) {
    // 下面这行代码不能得到正确的元素个数！
    // sizeof(arr) 得到的是指针的大小（32位系统是4，64位系统是8）
    // sizeof(arr[0]) 得到的是4（int的大小）
    // 所以结果是 4/4=1 或 8/4=2，都不对！
    int len = sizeof(arr) / sizeof(arr[0]);
    cout << "错误方式得到的长度: " << len << endl;  // 输出 1 或 2，不是 5！
}

// ========== 正确做法 ==========
// 方法：额外传递一个参数表示数组长度
void printArrayRight(int arr[], int len) {
    cout << "遍历数组: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

// 数组作为参数时，函数内可以修改原数组的值（因为传的是地址）
void doubleArray(int arr[], int len) {
    for (int i = 0; i < len; i++) {
        arr[i] = arr[i] * 2;  // 直接修改原数组
    }
}

int main() {
    int arr[5] = {10, 20, 30, 40, 50};
    int len = sizeof(arr) / sizeof(arr[0]);

    // 错误方式
    printArrayWrong(arr);  // 输出不正确的长度

    // 正确方式
    printArrayRight(arr, len);  // 输出: 10 20 30 40 50

    // 修改数组
    doubleArray(arr, len);
    cout << "翻倍后的数组: ";
    printArrayRight(arr, len);  // 输出: 20 40 60 80 100

    return 0;
}
```

## 八、常见错误与注意事项

### 错误1：数组越界访问

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // 错误！arr[5] 不存在，下标最大是 4
    // 这是未定义行为，可能读到垃圾值，也可能导致程序崩溃
    // cout << arr[5] << endl;  // 不要这样写！

    // 正确的范围：arr[0] 到 arr[4]
    for (int i = 0; i < 5; i++) {  // i < 5，不是 i <= 5
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

### 错误2：数组未初始化就使用

```cpp
#include <iostream>
using namespace std;

int main() {
    // 未初始化的数组，元素值是随机的（垃圾值）
    int arr[5];
    cout << "未初始化的数组:" << endl;
    for (int i = 0; i < 5; i++) {
        cout << "arr[" << i << "] = " << arr[i] << endl;
        // 输出的是随机值，每次运行可能不同
    }

    // 好习惯：定义数组时就初始化
    int brr[5] = {0};  // 全部初始化为0
    cout << "初始化为0:" << endl;
    for (int i = 0; i < 5; i++) {
        cout << "brr[" << i << "] = " << brr[i] << endl;  // 全是0
    }

    return 0;
}
```

### 错误3：数组之间不能直接赋值

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr1[5] = {1, 2, 3, 4, 5};
    int arr2[5];

    // 错误！数组不能直接用 = 赋值给另一个数组
    // arr2 = arr1;  // 编译错误！

    // 正确方法：逐个元素赋值（用循环）
    for (int i = 0; i < 5; i++) {
        arr2[i] = arr1[i];
    }

    cout << "arr2: ";
    for (int i = 0; i < 5; i++) {
        cout << arr2[i] << " ";
    }
    cout << endl;
    // 输出: 1 2 3 4 5

    return 0;
}
```

### 错误4：数组大小必须是常量

```cpp
#include <iostream>
using namespace std;

int main() {
    // 正确：用常量定义数组大小
    const int SIZE = 5;
    int arr1[SIZE] = {1, 2, 3, 4, 5};  // OK

    // 正确：直接用数字
    int arr2[5] = {1, 2, 3, 4, 5};  // OK

    // 注意：用变量定义数组大小在标准C++中是不允许的
    // （叫做 VLA - Variable Length Array，是C99的特性，C++标准不支持）
    // 但有些编译器（如GCC）扩展支持，不推荐使用
    int n = 5;
    // int arr3[n] = {1, 2, 3, 4, 5};  // 不推荐！非标准C++

    // 如果确实需要动态大小，后面会学到用 new 或 vector
    return 0;
}
```

### 错误5：混淆数组大小和最大下标

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[5] = {10, 20, 30, 40, 50};

    // arr[5] 表示有5个元素
    // 最大下标是 4，不是 5！
    // 下标范围: 0, 1, 2, 3, 4

    // 常见错误写法：
    // for (int i = 1; i <= 5; i++)  // 错！下标从1开始且越界

    // 正确写法：
    for (int i = 0; i < 5; i++) {  // 从0开始，到4结束
        cout << arr[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 九、综合实战练习

### 练习1：求数组中的最大值

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {35, 12, 67, 4, 89, 23, 56};
    int len = sizeof(arr) / sizeof(arr[0]);

    // 假设第一个元素是最大值
    int maxVal = arr[0];

    // 逐个比较，如果发现更大的就更新 maxVal
    for (int i = 1; i < len; i++) {
        if (arr[i] > maxVal) {
            maxVal = arr[i];
        }
    }

    cout << "最大值是: " << maxVal << endl;  // 89

    return 0;
}
```

### 练习2：求数组元素的平均值

```cpp
#include <iostream>
using namespace std;

int main() {
    int scores[] = {85, 92, 78, 96, 88};
    int len = sizeof(scores) / sizeof(scores[0]);

    int sum = 0;
    for (int i = 0; i < len; i++) {
        sum += scores[i];  // 累加每个元素
    }

    double avg = (double)sum / len;  // 注意要转成double再除
    cout << "总分: " << sum << endl;
    cout << "平均分: " << avg << endl;
    // 总分: 439, 平均分: 87.8

    return 0;
}
```

### 练习3：数组元素逆序

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {1, 2, 3, 4, 5};
    int len = sizeof(arr) / sizeof(arr[0]);

    cout << "逆序前: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;

    // 逆序的方法：首尾交换
    // 第一个和最后一个交换，第二个和倒数第二个交换...
    int left = 0;           // 左指针，从头开始
    int right = len - 1;    // 右指针，从尾开始

    while (left < right) {
        // 交换 arr[left] 和 arr[right]
        int temp = arr[left];
        arr[left] = arr[right];
        arr[right] = temp;

        left++;     // 左指针右移
        right--;    // 右指针左移
    }

    cout << "逆序后: ";
    for (int i = 0; i < len; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
    // 输出: 5 4 3 2 1

    return 0;
}
```

### 练习4：统计数组中某个值出现的次数

```cpp
#include <iostream>
using namespace std;

int main() {
    int arr[] = {3, 7, 3, 9, 3, 1, 7, 3};
    int len = sizeof(arr) / sizeof(arr[0]);

    int target = 3;  // 要查找的值
    int count = 0;   // 计数器

    for (int i = 0; i < len; i++) {
        if (arr[i] == target) {
            count++;
        }
    }

    cout << target << " 出现了 " << count << " 次" << endl;
    // 输出: 3 出现了 4 次

    return 0;
}
```

## 十、本节要点总结

1. **数组定义**：`数据类型 数组名[大小] = {初始值};`
2. **下标从0开始**：`arr[0]` 到 `arr[n-1]`
3. **部分初始化**：未赋值的元素自动为0
4. **数组名的用途**：`sizeof(arr)` 算总字节，`sizeof(arr)/sizeof(arr[0])` 算个数
5. **数组传参**：传的是地址，函数内无法用 sizeof 获取长度，需要额外传长度参数
6. **不能直接赋值**：数组之间不能用 `=` 赋值，需要用循环逐个复制
7. **注意越界**：访问不存在的下标是未定义行为
8. **连续存储**：数组元素在内存中是连续的，地址依次递增
