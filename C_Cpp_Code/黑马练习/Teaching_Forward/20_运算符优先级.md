# 20 - 运算符优先级详解

## 一、什么是运算符优先级？

运算符优先级决定了在一个表达式中，**哪个运算符先执行**。
就像数学中"先乘除后加减"一样，C++ 中也有明确的优先级规则。

```cpp
int result = 2 + 3 * 4;
// 先算 3 * 4 = 12，再算 2 + 12 = 14
// 而不是先算 2 + 3 = 5，再算 5 * 4 = 20
cout << result << endl;  // 14
```

## 二、完整的优先级表（从高到低）

| 优先级 | 运算符 | 含义 | 结合性 |
|--------|--------|------|--------|
| 1 | `()` `[]` `->` `.` | 括号、下标、成员访问 | 左到右 |
| 2 | `!` `~` `++` `--` `+`(正) `-`(负) `*`(解引用) `&`(取地址) `(类型)` `sizeof` | 单目运算符 | **右到左** |
| 3 | `*` `/` `%` | 乘、除、取模 | 左到右 |
| 4 | `+` `-` | 加、减 | 左到右 |
| 5 | `<<` `>>` | 左移、右移 | 左到右 |
| 6 | `<` `<=` `>` `>=` | 小于、小于等于、大于、大于等于 | 左到右 |
| 7 | `==` `!=` | 等于、不等于 | 左到右 |
| 8 | `&` | 按位与 | 左到右 |
| 9 | `^` | 按位异或 | 左到右 |
| 10 | `\|` | 按位或 | 左到右 |
| 11 | `&&` | 逻辑与 | 左到右 |
| 12 | `\|\|` | 逻辑或 | 左到右 |
| 13 | `?:` | 条件（三目）运算符 | **右到左** |
| 14 | `=` `+=` `-=` `*=` `/=` `%=` `&=` `\|=` `^=` `<<=` `>>=` | 赋值运算符 | **右到左** |
| 15 | `,` | 逗号运算符 | 左到右 |

## 三、分组记忆法

把优先级分成**七大组**，从高到低记忆：

```
第1层：单目运算符（! ~ ++ -- sizeof 等）     优先级最高
  |
第2层：算术运算符（* / % -> + -）
  |
第3层：移位运算符（<< >>）
  |
第4层：关系运算符（< <= > >= -> == !=）
  |
第5层：位运算符（& -> ^ -> |）
  |
第6层：逻辑运算符（&& -> ||）
  |
第7层：赋值运算符（= += -= 等）              优先级最低（逗号更低）
```

> **记忆口诀**：单算移关位逻赋
> （单目 > 算术 > 移位 > 关系 > 位 > 逻辑 > 赋值）

## 四、各组详细说明

### 4.1 第1层：单目运算符（最高优先级）

```cpp
int a = 5;

// ! 优先级最高
cout << !a << endl;        // 0（!5 = !true = false）

// ++ -- 也是单目
int b = 5;
cout << -b << endl;        // -5（取负也是单目）

// sizeof 也是单目
cout << sizeof(int) << endl;  // 4

// 组合使用时，单目先算
int x = 5;
cout << !x + 3 << endl;   // 等价于 (!x) + 3 = 0 + 3 = 3
cout << -x * 2 << endl;   // 等价于 (-x) * 2 = -5 * 2 = -10
```

### 4.2 第2层：算术运算符（先乘除后加减）

```cpp
// * / % 优先级高于 + -
cout << 2 + 3 * 4 << endl;      // 14（先算3*4=12，再算2+12=14）
cout << 10 - 6 / 2 << endl;     // 7（先算6/2=3，再算10-3=7）
cout << 2 + 10 % 3 << endl;     // 4（先算10%3=1，再算2+1=3）

// * / % 同级，从左到右
cout << 12 / 3 * 2 << endl;     // 8（先算12/3=4，再算4*2=8）
cout << 12 * 3 / 2 << endl;     // 18（先算12*3=36，再算36/2=18）

// + - 同级，从左到右
cout << 10 - 3 + 2 << endl;     // 9（先算10-3=7，再算7+2=9）
```

### 4.3 第3层：移位运算符

```cpp
// 移位优先级低于算术
cout << 1 + 2 << 3 << endl;
// 等价于 (1 + 2) << 3 = 3 << 3 = 24
// 不是 1 + (2 << 3) = 1 + 16 = 17

// 移位优先级低于算术但高于关系
cout << (4 << 1 > 5) << endl;
// 等价于 ((4 << 1) > 5) -> (8 > 5) -> true(1)
```

### 4.4 第4层：关系运算符（< <= > >= 高于 == !=）

```cpp
// < > <= >= 优先级高于 == !=
cout << (5 > 3 == 1) << endl;
// 等价于 ((5 > 3) == 1) -> (true == 1) -> true(1)

// == != 优先级低于 < > <= >=
int a = 1, b = 2, c = 2;
// a < b == c
// 等价于 (a < b) == c -> true == 2 -> 1
```

### 4.5 第5层：位运算符（& 高于 ^ 高于 |）

```cpp
// & 优先级高于 ^
cout << (0b1100 ^ 0b1010 & 0b1000) << endl;
// 等价于 0b1100 ^ (0b1010 & 0b1000)
//      = 0b1100 ^ 0b1000
//      = 0b0100 = 4

// | 优先级最低
cout << (0b1100 | 0b1010 & 0b1000) << endl;
// 等价于 0b1100 | (0b1010 & 0b1000)
//      = 0b1100 | 0b1000
//      = 0b1100 = 12

// 位运算优先级容易搞混，强烈建议加括号
```

### 4.6 第6层：逻辑运算符（&& 高于 ||）

```cpp
// && 优先级高于 ||
cout << (true || false && false) << endl;
// 等价于 true || (false && false) -> true || false -> true(1)

// 不是 (true || false) && false -> true && false -> false(0)

// 实际应用中，加括号更安全
int score = 85;
bool result = (score >= 60) && (score <= 100);
```

### 4.7 第7层：赋值运算符（最低优先级之一）

```cpp
// 赋值优先级很低
int a;
a = 5 + 3 * 2;
// 等价于 a = (5 + (3 * 2)) -> a = 11

// 三目运算符优先级高于赋值
int x = 5, y = 10;
int max = x > y ? x : y;
// 等价于 max = (x > y ? x : y)
```

## 五、结合性说明

当**同一优先级**的运算符连续出现时，按结合性决定计算顺序。

### 5.1 左结合（从左到右）

大多数运算符都是左结合的。

```cpp
// 算术运算符：左结合
cout << 10 - 3 - 2 << endl;
// 等价于 (10 - 3) - 2 = 7 - 2 = 5
// 不是 10 - (3 - 2) = 10 - 1 = 9

// 关系运算符：左结合
cout << (3 < 5 > 2) << endl;
// 等价于 ((3 < 5) > 2) -> (true > 2) -> (1 > 2) -> false(0)
```

### 5.2 右结合（从右到左）

赋值运算符和单目运算符是右结合的。

```cpp
// 赋值运算符：右结合
int a, b, c;
a = b = c = 10;
// 等价于 a = (b = (c = 10))
// 从右往左依次赋值

// 三目运算符：右结合（嵌套时）
int x = 5;
string result = x > 10 ? "大" : x > 5 ? "中" : "小";
// 等价于 x > 10 ? "大" : (x > 5 ? "中" : "小")
// 结果是 "小"

// 单目运算符：右结合
int y = 5;
cout << -++y << endl;
// 等价于 -(++y) -> -(6) -> -6
```

## 六、常见陷阱

### 陷阱1：& vs ==（最容易犯的错！）

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 3;

    // 想判断 a 和 b 的最后一位是否都是 1
    // 错误写法：
    if (a & 1 == b & 1) {
        // 实际上是：a & (1 == b) & 1
        // 因为 == 优先级高于 &
    }

    // 正确写法：
    if ((a & 1) == (b & 1)) {
        cout << "最后一位相同" << endl;
    }

    return 0;
}
```

### 陷阱2：<< vs +

```cpp
int a = 1 << 2 + 3;   // 你以为是 (1<<2)+3=7？
                        // 实际是 1<<(2+3)=32！因为 + 优先级高于 <<

int b = (1 << 2) + 3; // 如果你想得到 7，必须加括号
```

### 陷阱3：|| vs &&

```cpp
// 你想表达：a 或者 (b 并且 c)
bool result = a || b && c;    // 这行是对的（&& 优先级高于 ||）

// 你想表达：(a 或者 b) 并且 c
bool result2 = a || b && c;   // 这行是错的！
// 应该写成：
bool result3 = (a || b) && c;
```

### 陷阱4：赋值中的优先级

```cpp
int a = 5;
a *= 2 + 3;   // 等价于 a *= (2+3) -> a = a * 5 -> 25
               // 不是 (a*2) + 3 -> 13
```

### 陷阱5：在 if 中用 = 代替 ==

```cpp
int a = 5;

// 错误！这是赋值，不是比较
if (a = 10) { }   // a 被赋值为10，条件永远为真

// 正确写法
if (a == 10) { }   // 这是比较

// 防御性写法：常量写左边
if (10 == a) { }   // 如果误写成 10 = a 会编译报错
```

## 七、综合示例

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 3, c = 2, d = 1;

    // 示例1：混合运算
    int r1 = a + b * c - d;
    // 等价于 a + (b * c) - d = 5 + 6 - 1 = 10
    cout << "r1 = " << r1 << endl;   // 10

    // 示例2：关系和逻辑
    bool r2 = a > b && c < d || a == 5;
    // 等价于 ((a > b) && (c < d)) || (a == 5)
    //      = (true && false) || true
    //      = false || true
    //      = true
    cout << "r2 = " << r2 << endl;   // 1

    // 示例3：位运算和逻辑运算混合
    int x = 0b1010, y = 0b1100;
    int r3 = x & y | x ^ y;
    // 等价于 (x & y) | (x ^ y)
    //      = 0b1000 | 0b0110
    //      = 0b1110 = 14
    cout << "r3 = " << r3 << endl;   // 14

    // 示例4：赋值和三目
    int p = 10, q = 20;
    int r4 = p > q ? p : q;
    // 等价于 r4 = (p > q ? p : q) -> r4 = 20
    cout << "r4 = " << r4 << endl;   // 20

    // 示例5：完整分析复杂表达式
    int e = 2, f = 3, g = 4, h = 5;
    int r5 = e + f > g && h - e < f || g != h;

    // 第一步：算术运算
    // (e+f) > g && (h-e) < f || g != h
    // 5 > 4 && 3 < 3 || 4 != 5

    // 第二步：关系运算
    // true && false || true

    // 第三步：逻辑与（&& 优先级高于 ||）
    // false || true

    // 第四步：逻辑或
    // true -> 1

    cout << "r5 = " << r5 << endl;   // 1

    return 0;
}
```

## 八、练习

### 练习1：判断以下表达式的值（不运行代码）

```cpp
// 题目1：2 + 3 * 4 = ?
// 答案：2 + 12 = 14

// 题目2：(2 + 3) * 4 = ?
// 答案：5 * 4 = 20

// 题目3：10 > 5 && 3 < 1 = ?
// 答案：true && false = false(0)

// 题目4：true || false && false = ?
// 答案：true || (false && false) = true(1)

// 题目5：5 & 3 + 1 = ?
// 答案：5 & (3 + 1) = 5 & 4 = 4

// 题目6：!!5 = ?
// 答案：!(!5) = !(false) = true(1)
```

### 练习2：给表达式加上括号使其等价

```cpp
// 原始：a + b * c > d && e == f
// 加括号：((a + (b * c)) > d) && (e == f)

// 原始：a | b & c ^ d
// 加括号：a | ((b & c) ^ d)

// 原始：x = a > b ? a : b + 1
// 加括号：x = (a > b ? a : (b + 1))
```

## 九、给初学者的建议

### 建议1：不确定就加括号

```cpp
// 不清晰
if (a & b == c)

// 清晰明了
if ((a & b) == c)
```

### 建议2：复杂表达式拆开写

```cpp
// 不推荐（一行太长，难以理解）
if (a + b > c && d - a < b || c != d && e > 0) {}

// 推荐（拆开写，可读性强）
bool cond1 = (a + b) > c;
bool cond2 = (d - a) < b;
bool cond3 = (c != d);
bool cond4 = (e > 0);

if ((cond1 && cond2) || (cond3 && cond4)) {
    // ...
}
```

### 建议3：记住关键规则

1. **单目 > 双目**（`!` `~` `++` `--` 优先级最高）
2. **算术 > 移位 > 关系 > 位 > 逻辑 > 赋值 > 逗号**
3. **`&&` > `||`**
4. **`==` `!=` 优先级低于 `<` `<=` `>` `>=`**
5. **位运算中 `&` > `^` > `|`**
6. 不确定时，**加括号**永远不会错

## 十、总结

| 分组 | 运算符 | 优先级 |
|------|--------|--------|
| 最高 | `()` `[]` `->` `.` | 1 |
| 单目 | `!` `~` `++` `--` `+` `-` `*` `&` `sizeof` | 2 |
| 算术 | `*` `/` `%` -> `+` `-` | 3-4 |
| 移位 | `<<` `>>` | 5 |
| 关系 | `<` `<=` `>` `>=` -> `==` `!=` | 6-7 |
| 位运算 | `&` -> `^` -> `\|` | 8-10 |
| 逻辑 | `&&` -> `\|\|` | 11-12 |
| 三目 | `?:` | 13 |
| 赋值 | `=` `+=` `-=` 等 | 14 |
| 最低 | `,` | 15 |

**最重要的一条建议**：写代码时，如果不确定优先级，**加括号**！
括号不仅可以保证正确性，还能大大提高代码可读性。
