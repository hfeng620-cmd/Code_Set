# C++ 学习路线总览

> 本文档为 C++ 初学者提供完整的学习路线图，帮助你系统地掌握 C++ 编程。
> ✅ = 已学  📌 = 待学

---

## 第一阶段：C++ 基础入门

本阶段的目标是学会最基本的语法，能够写出简单的程序。

| 状态 | 知识点 | 简要说明 |
|------|--------|----------|
| ✅ | HelloWorld | 学会写出第一个程序，了解程序基本结构 |
| ✅ | 注释 | 单行注释 `//` 和多行注释 `/* */`，用于给代码添加说明 |
| ✅ | 变量 | 在内存中开辟空间存储数据，有名字、类型和值 |
| ✅ | 常量 | 不能被修改的值，用 `#define` 或 `const` 定义 |
| ✅ | 整型 int | 存储整数，如 `int a = 10;` |
| ✅ | 浮点型 float/double | 存储小数，float 精度低占4字节，double 精度高占8字节 |
| ✅ | 字符型 char | 存储单个字符，如 `char c = 'A';`，占1字节 |
| ✅ | 字符串型 string | 存储一串字符，如 `string s = "hello";`，需要 `#include <string>` |
| ✅ | 布尔型 bool | 只有 `true`(1) 和 `false`(0) 两个值 |
| ✅ | sizeof 关键字 | 查询变量或数据类型占用的内存大小，如 `sizeof(int)` |
| ✅ | 转义字符 | 特殊字符表示法，如 `\n` 换行、`\t` 制表符、`\\` 反斜杠 |
| ✅ | cin 输入流 | 从键盘读取用户输入，如 `cin >> a;` |
| ✅ | cout 输出流 | 向屏幕输出数据，如 `cout << a << endl;` |
| 📌 | 数据类型转换 | 隐式转换和显式转换，如 `(int)3.14` |

---

## 第二阶段：运算符与表达式

本阶段学习各种运算符，让程序能够进行计算和判断。

| 状态 | 知识点 | 简要说明 |
|------|--------|----------|
| ✅ | 算术运算符 | `+` `-` `*` `/` `%` 加减乘除取模 |
| ✅ | 赋值运算符 | `=` `+=` `-=` `*=` `/=` `%=` |
| ✅ | 比较运算符 | `==` `!=` `<` `>` `<=` `>=` 结果为 bool |
| ✅ | 逻辑运算符 | `&&` 与、`\|\|` 或、`!` 非 |
| ✅ | 递增递减运算符 | `++` `--`，分前置和后置，区别在于返回值 |
| ✅ | 三目运算符 | `条件 ? 值1 : 值2`，条件为真取值1，否则取值2 |
| 📌 | 位运算符 | `&` `\|` `^` `~` `<<` `>>` 对二进制位操作 |
| 📌 | 运算符优先级 | 各种运算符的执行顺序，不确定时加括号 |

---

## 第三阶段：程序流程控制

本阶段学习控制程序执行顺序，让程序能够做出判断和重复执行。

| 状态 | 知识点 | 简要说明 |
|------|--------|----------|
| ✅ | if 选择结构 | `if`、`else if`、`else` 根据条件执行不同代码 |
| ✅ | switch 选择结构 | 多分支选择，适合值匹配的场景 |
| ✅ | while 循环 | 先判断条件，再执行循环体 |
| ✅ | break | 跳出当前循环或 switch 语句 |
| ✅ | srand / rand | 随机数生成，`srand` 设置种子，`rand` 生成随机数 |
| ✅ | ctime | 获取系统时间，常用于随机数种子 `srand((unsigned int)time(NULL))` |
| 📌 | do...while 循环 | 先执行一次循环体，再判断条件，至少执行一次 |
| 📌 | for 循环 | 最常用的循环，适合已知循环次数的场景 |
| 📌 | continue | 跳过本次循环剩余代码，直接进入下一次循环 |
| 📌 | 嵌套循环 | 循环内部再套循环，常用于二维数据处理 |
| 📌 | goto 语句 | 无条件跳转（不推荐使用，了解即可） |

---

## 第四阶段：复合数据类型与函数

本阶段学习更复杂的数据组织方式和代码复用。

| 状态 | 知识点 | 简要说明 |
|------|--------|----------|
| 📌 | 一维数组 | 存储一组相同类型的数据，如 `int arr[5] = {1,2,3,4,5};` |
| 📌 | 二维数组 | 表格状数据，如 `int arr[3][4];` |
| 📌 | 字符数组与C风格字符串 | 以 `\0` 结尾的字符数组 |
| 📌 | 函数定义与调用 | 封装可复用的代码块 |
| 📌 | 函数参数与返回值 | 值传递、地址传递、引用传递 |
| 📌 | 函数声明（原型） | 在调用前声明函数的签名 |
| 📌 | 函数的分文件编写 | 头文件 `.h` 声明，源文件 `.cpp` 实现 |
| 📌 | 指针基础 | 存储内存地址的变量，如 `int *p = &a;` |
| 📌 | 指针与数组 | 数组名就是首元素地址 |
| 📌 | 指针与函数 | 通过指针在函数中修改外部变量 |
| 📌 | 结构体 struct | 自定义数据类型，将多个变量组合在一起 |
| 📌 | 引用 | 变量的别名，如 `int &b = a;` |
| 📌 | 动态内存分配 | `new` 和 `delete` 管理堆内存 |

---

## 第五阶段：面向对象编程（OOP）

本阶段进入 C++ 最核心的部分——面向对象。

| 状态 | 知识点 | 简要说明 |
|------|--------|----------|
| 📌 | 类与对象 | 类是模板，对象是实例 |
| 📌 | 访问权限 | `public`、`private`、`protected` |
| 📌 | 构造函数与析构函数 | 对象创建和销毁时自动调用 |
| 📌 | 拷贝构造函数 | 用已有对象创建新对象 |
| 📌 | 运算符重载 | 让运算符支持自定义类型 |
| 📌 | 继承 | 子类复用父类的成员 |
| 📌 | 多态 | 同一接口不同实现，虚函数 |
| 📌 | 文件操作 | 读写文件，fstream |
| 📌 | 模板 | 函数模板和类模板，泛型编程 |
| 📌 | STL 标准模板库 | vector、map、string、算法等 |

## 学习顺序建议（按周）

> 以下按每天学习 1-2 小时计算，可根据自己的时间调整。

### 第 1 周：环境搭建与基础语法
- 第 1 天：安装开发环境（VS Code / Visual Studio），写第一个 HelloWorld
- 第 2 天：学习注释、变量、常量
- 第 3 天：学习 int、float、double、char、string、bool 六种数据类型
- 第 4 天：学习 sizeof、转义字符
- 第 5 天：学习算术运算符、赋值运算符
- 第 6 天：学习比较运算符、逻辑运算符
- 第 7 天：复习 + 练习（写一个简单计算器）

### 第 2 周：流程控制
- 第 1 天：学习 if / else 条件判断
- 第 2 天：学习 switch 多分支选择
- 第 3 天：学习 while 循环
- 第 4 天：学习 cin 输入、cout 输出的更多用法
- 第 5 天：学习 break、随机数（srand/rand/ctime）
- 第 6 天：学习三目运算符、递增递减
- 第 7 天：练习（猜数字游戏）

### 第 3 周：进阶流程控制
- 第 1 天：学习 for 循环
- 第 2 天：学习 do...while 循环、continue
- 第 3 天：学习嵌套循环
- 第 4 天：练习九九乘法表
- 第 5 天：练习打印图形（三角形、菱形）
- 第 6 天：综合复习前两周内容
- 第 7 天：综合练习

### 第 4 周：数组与函数入门
- 第 1 天：学习一维数组
- 第 2 天：学习二维数组
- 第 3 天：学习函数的定义和调用
- 第 4 天：学习函数的值传递和返回值
- 第 5 天：学习函数声明、分文件编写
- 第 6 天：练习（冒泡排序）
- 第 7 天：练习（学生成绩管理）

### 第 5-6 周：指针与结构体
- 第 5 周：指针基础（定义、使用、空指针、野指针、const 修饰指针）
- 第 6 周：结构体（定义、使用、结构体数组、结构体指针）
- 练习：通讯录管理系统

### 第 7-8 周：面向对象基础
- 第 7 周：类和对象、封装、访问权限、构造函数、析构函数
- 第 8 周：this 指针、静态成员、内存管理
- 练习：设计一个"学生类"

### 第 9-10 周：面向对象进阶
- 第 9 周：继承、多态
- 第 10 周：文件操作、模板基础
- 练习：职工管理系统

### 第 11-12 周：STL 与现代 C++
- 第 11 周：string、vector、deque、stack、queue
- 第 12 周：set、map、算法、auto、范围 for、lambda
- 练习：演讲比赛管理系统

## 推荐练习项目

按难度递增排列，建议每个阶段至少完成一个项目。

### 入门级项目
1. **个人信息展示** —— 练习变量、输出
2. **简易计算器** —— 练习输入输出、运算符、条件判断
3. **猜数字游戏** —— 练习循环、条件、随机数
4. **九九乘法表** —— 练习嵌套循环

### 初级项目
5. **学生成绩管理** —— 练习数组、循环、函数
6. **通讯录系统** —— 练习结构体、数组、函数
7. **猜拳游戏** —— 综合练习
8. **井字棋游戏** —— 练习二维数组、函数、游戏逻辑

### 中级项目
9. **职工管理系统** —— 练习面向对象（类、继承、多态）
10. **演讲比赛管理系统** —— 综合运用 STL
11. **机房预约系统** —— 练习文件读写、面向对象

### 进阶项目
12. **俄罗斯方块** —— 练习二维数组、游戏循环
13. **图书管理系统** —— 练习文件操作、面向对象、STL
14. **简易聊天室** —— 练习网络编程基础

## 学习建议

1. **不要只看不写** —— 每学一个知识点，至少写 3 个练习程序
2. **不要死记硬背** —— 理解原理比记住语法更重要
3. **善用调试** —— 学会用 IDE 的调试功能，逐行执行观察变量变化
4. **多犯错多总结** —— 遇到错误不要怕，错误是最好的老师
5. **做笔记** —— 把自己容易犯的错误记录下来，经常回顾
6. **循序渐进** —— 不要跳过基础直接学高级内容
7. **参考官方文档** —— cppreference.com 是最权威的 C++ 参考
