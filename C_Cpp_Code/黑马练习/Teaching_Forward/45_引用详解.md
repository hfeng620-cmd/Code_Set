# 引用详解

## 一、什么是引用

引用就是给一个已经存在的变量起一个**别名**。
引用和原变量**共享同一块内存**，操作引用就等于操作原变量。

可以这样理解：一个人有本名，也有小名，但始终是同一个人。

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 10;   // 原变量
    int &b = a;   // b 是 a 的引用（别名）

    cout << "a = " << a << endl;  // 10
    cout << "b = " << b << endl;  // 10

    // 通过引用修改值，原变量也会变
    b = 20;
    cout << "修改 b 之后:" << endl;
    cout << "a = " << a << endl;  // 20
    cout << "b = " << b << endl;  // 20

    // 它们的地址完全相同
    cout << "a 的地址: " << &a << endl;
    cout << "b 的地址: " << &b << endl;
    // 输出的地址一模一样，说明它们是同一块内存

    return 0;
}
```

---

## 二、引用的基本语法

语法格式：`类型 &引用名 = 原变量;`

```cpp
int a = 100;
int &ref = a;    // 正确：ref 是 a 的引用

double d = 3.14;
double &rd = d;  // 正确：rd 是 d 的引用
```

**注意：** `&` 在这里不是取地址运算符，而是引用声明符。

---

## 三、引用必须初始化

引用在创建时**必须指定它引用谁**，不能先创建一个空引用。

```cpp
int a = 10;
int &ref = a;     // 正确：创建时就初始化了

// int &ref2;     // 错误！引用必须初始化，不能为空
// int &ref2 = 10; // 错误！不能直接引用一个字面量（常量引用除外，后面讲）
```

---

## 四、引用初始化后不能更改指向

一旦引用绑定到某个变量，就**永远绑定**到这个变量，不能再改为引用其他变量。

```cpp
int a = 10;
int b = 20;

int &ref = a;   // ref 绑定到 a
cout << "ref = " << ref << endl;  // 10

ref = b;         // 这不是让 ref 改为引用 b！
                 // 这是把 b 的值赋给 ref（也就是赋给 a）
cout << "a = " << a << endl;     // 20
cout << "ref = " << ref << endl; // 20
cout << "b = " << b << endl;     // 20（b 没变）

// 验证：ref 的地址仍然是 a 的地址
cout << "&ref = " << &ref << endl;  // 和 &a 一样
cout << "&b   = " << &b << endl;    // 和 &a 不一样
```

## 五、引用的本质（底层原理）

在 C++ 内部，**引用的本质是指针常量**。

编译器会自动进行转换：

```cpp
// 你写的代码：
int a = 10;
int &ref = a;
ref = 20;

// 编译器内部实际处理的（类似）：
int a = 10;
int * const ref = &a;   // 指针常量：指针指向不能变
*ref = 20;               // 通过指针解引用修改值
```

这就解释了：
- 为什么引用必须初始化 --> 指针常量必须初始化
- 为什么引用不能更改指向 --> 指针常量的指向不能改

## 六、引用做函数参数（重点）

引用做函数参数，可以让函数内部直接修改外部变量，比指针更简洁。

### 6.1 值传递 vs 引用传递

```cpp
#include <iostream>
using namespace std;

// 值传递：不会改变外部变量
void swapByValue(int a, int b)
{
    int temp = a;
    a = b;
    b = temp;
    // 只交换了形参，实参不受影响
}

// 引用传递：会改变外部变量
void swapByRef(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
    // a 和 b 是外部变量的引用，直接修改外部变量
}

int main()
{
    int x = 10, y = 20;

    swapByValue(x, y);
    cout << "值传递后: x=" << x << ", y=" << y << endl;
    // x=10, y=20  没变

    swapByRef(x, y);
    cout << "引用传递后: x=" << x << ", y=" << y << endl;
    // x=20, y=10  交换成功

    return 0;
}
```

### 6.2 引用传递 vs 指针传递

```cpp
// 指针传递版本（C 风格）
void swapByPointer(int *a, int *b)
{
    int temp = *a;
    *a = *b;
    *b = temp;
}

// 引用传递版本（C++ 风格，推荐）
void swapByRef(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
}

int main()
{
    int x = 10, y = 20;

    // 指针方式：需要取地址，容易写错
    swapByPointer(&x, &y);

    // 引用方式：直接传变量，语法简洁
    swapByRef(x, y);

    return 0;
}
```

## 七、引用做函数返回值

函数可以返回引用，但有重要注意事项。

### 7.1 不要返回局部变量的引用

```cpp
// 错误示范：返回局部变量的引用
int& func()
{
    int a = 10;   // a 是局部变量，函数结束后被销毁
    return a;      // 返回的引用指向已销毁的内存！
}

int main()
{
    int &ref = func();  // ref 引用了一块已经释放的内存
    cout << ref << endl; // 未定义行为：可能输出10，也可能输出垃圾值
    // cout << ref << endl; // 再访问一次可能就出错了
    return 0;
}
```

### 7.2 正确用法：返回静态变量或全局变量的引用

```cpp
int& func()
{
    static int a = 10;  // 静态变量：生命周期是整个程序
    a++;                // 每次调用都加1
    return a;
}

int main()
{
    int &ref = func();  // 第一次调用，a=11
    cout << ref << endl; // 11

    int &ref2 = func(); // 第二次调用，a=12
    cout << ref2 << endl; // 12

    cout << ref << endl; // 12（ref 和 ref2 引用同一个变量）
    return 0;
}
```

### 7.3 引用返回值可以作为左值

如果函数返回引用，那么函数调用可以放在赋值号左边。

```cpp
int& getElement(int arr[], int index)
{
    return arr[index];  // 返回数组元素的引用
}

int main()
{
    int arr[] = {10, 20, 30, 40, 50};

    // 函数调用作为左值（赋值号左边）
    getElement(arr, 2) = 100;  // 相当于 arr[2] = 100

    cout << "arr[2] = " << arr[2] << endl;  // 100

    return 0;
}
```

## 八、常量引用（const 引用）

用 `const` 修饰引用，表示**不能通过引用修改原变量**。

### 8.1 防止误修改

```cpp
int a = 10;
const int &ref = a;  // 常量引用

// ref = 20;  // 错误！不能通过常量引用修改值
a = 20;       // 正确！直接修改原变量是可以的
```

### 8.2 常量引用可以绑定字面量

普通引用不能绑定字面量，但常量引用可以：

```cpp
// int &ref = 10;         // 错误！普通引用不能绑定字面量
const int &ref = 10;      // 正确！常量引用可以绑定字面量

// 等价于：
// int temp = 10;
// const int &ref = temp;
```

### 8.3 函数参数中使用常量引用

如果函数不打算修改参数，用 `const` 引用既安全又高效（避免拷贝）。

```cpp
// 打印函数：只需要读取，不需要修改
void printInfo(const string &name, const int &age)
{
    cout << "姓名: " << name << ", 年龄: " << age << endl;
    // name = "xxx";  // 错误！不能修改
    // age = 0;       // 错误！不能修改
}

int main()
{
    string n = "张三";
    int a = 20;
    printInfo(n, a);       // 传变量
    printInfo("李四", 25); // 可以直接传字面量（因为是const引用）
    return 0;
}
```

## 九、引用 vs 指针对比表

| 对比项目 | 引用 | 指针 |
|---------|------|------|
| 语法 | `int &ref = a;` | `int *p = &a;` |
| 是否必须初始化 | 必须 | 可以不初始化（但不推荐） |
| 能否为空 | 不能 | 可以（空指针 `nullptr`） |
| 能否改变指向 | 不能，绑定后固定 | 可以随时指向其他变量 |
| 使用方式 | 直接当变量用 | 需要解引用 `*p` |
| 是否占用内存 | 占用（和指针一样大） | 占用（4或8字节） |
| 本质 | 编译器实现为指针常量 | 本身就是指针 |
| 是否有引用的引用 | 没有 | 有指针的指针（二级指针） |
| 语法简洁性 | 简洁，和普通变量一样 | 相对复杂 |
| 推荐场景 | 函数参数、返回值 | 动态内存、复杂数据结构 |

## 十、综合案例

```cpp
#include <iostream>
using namespace std;

// 用引用交换两个变量
void mySwap(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
}

// 用常量引用打印数组（不拷贝、不修改）
void printArray(const int arr[], int len)
{
    for (int i = 0; i < len; i++)
    {
        cout << arr[i] << " ";
    }
    cout << endl;
}

// 返回引用：让调用者可以修改数组元素
int& maxElement(int arr[], int len)
{
    int maxIndex = 0;
    for (int i = 1; i < len; i++)
    {
        if (arr[i] > arr[maxIndex])
        {
            maxIndex = i;
        }
    }
    return arr[maxIndex];
}

int main()
{
    // 1. 基本引用
    int a = 10, b = 20;
    cout << "交换前: a=" << a << ", b=" << b << endl;
    mySwap(a, b);
    cout << "交换后: a=" << a << ", b=" << b << endl;

    // 2. 常量引用打印数组
    int arr[] = {1, 3, 5, 7, 9};
    printArray(arr, 5);

    // 3. 引用返回值作为左值
    cout << "最大元素: " << maxElement(arr, 5) << endl;
    maxElement(arr, 5) = 100;  // 把最大元素改成100
    printArray(arr, 5);        // 1 3 5 7 100

    return 0;
}
```

## 总结

| 要点 | 说明 |
|------|------|
| 引用本质 | 指针常量，编译器自动解引用 |
| 必须初始化 | 创建时必须绑定到一个变量 |
| 不可改绑 | 一旦绑定，终身绑定 |
| 函数参数 | 比指针简洁，效果相同 |
| 函数返回值 | 不要返回局部变量引用，可作左值 |
| 常量引用 | 可绑字面量，防误改，推荐用于只读参数 |
