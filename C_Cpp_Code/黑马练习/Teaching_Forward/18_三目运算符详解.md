# 18 - 三目运算符详解

## 一、什么是三目运算符？

三目运算符（也叫条件运算符）是 C++ 中**唯一的三元运算符**（需要三个操作数）。
它可以看作 `if/else` 的简写形式。

语法：
```
条件 ? 值1 : 值2
```

- 条件为 **true** -> 整个表达式的结果是 **值1**
- 条件为 **false** -> 整个表达式的结果是 **值2**

---

## 二、基本用法

### 2.1 简单示例

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10, b = 20;

    // 求两个数中的较大值
    int max = (a > b) ? a : b;
    cout << "较大值：" << max << endl;   // 输出 20

    // 求两个数中的较小值
    int min = (a < b) ? a : b;
    cout << "较小值：" << min << endl;   // 输出 10

    // 直接在 cout 中使用
    cout << "a和b中较大的是: " << ((a > b) ? a : b) << endl;  // 20

    return 0;
}
```

### 2.2 执行过程图解

```
a = 10, b = 20

(a > b) ? a : b
  |
10 > 20 ? 10 : 20
  |
false ? 10 : 20
  |
选择 20（false 时选冒号后面的值）
  |
结果为 20
```

### 2.3 等价的 if/else 写法

```cpp
// 三目运算符写法
int max = (a > b) ? a : b;

// 等价于 if/else 写法
int max;
if (a > b) {
    max = a;
} else {
    max = b;
}
```

三目运算符明显更简洁，适合**简单的二选一**场景。

---

## 三、vs if/else 对比

| 特性 | 三目运算符 | if/else |
|------|-----------|---------|
| 语法简洁性 | 一行搞定 | 至少三行 |
| 可读性 | 简单场景好，复杂场景差 | 始终清晰 |
| 能否执行语句 | 只能返回值 | 可以执行多条语句 |
| 返回值 | 有返回值 | 没有返回值 |
| 适用场景 | 简单赋值 | 复杂逻辑 |

```cpp
// 适合用三目的场景
string result = (score >= 60) ? "及格" : "不及格";

// 不适合用三目的场景（太复杂，可读性差）
// 错误示范：
// string msg = (a > 0) ? ((a > 100) ? "很大" : "中等") : ((a == 0) ? "零" : "负数");

// 应该用 if/else
string msg;
if (a > 100) {
    msg = "很大";
} else if (a > 0) {
    msg = "中等";
} else if (a == 0) {
    msg = "零";
} else {
    msg = "负数";
}
```

## 四、三目运算符可以作左值（返回引用）

这是三目运算符的一个**重要特性**：当值1和值2都是**左值**（变量）时，整个表达式返回的是**引用**，可以被赋值！

### 4.1 用三目运算符赋值

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 20;

    // (a > b) ? a : b 返回的是 a 或 b 的引用
    // 所以可以给它赋值！
    ((a > b) ? a : b) = 100;

    cout << "a = " << a << endl;   // 10（没变）
    cout << "b = " << b << endl;   // 100（b 被修改了！）

    return 0;
}
```

原理解析：
- `a > b` 为 false，所以表达式返回 `b` 的**引用**
- 然后 `b = 100`，修改了 b 的值

### 4.2 等价的 if/else 写法

```cpp
// 三目作左值
((a > b) ? a : b) = 100;

// 等价于
if (a > b) {
    a = 100;
} else {
    b = 100;
}
```

### 4.3 注意：不能返回临时值的引用

```cpp
int a = 5;

// 错误！值2是字面量100（右值），不能返回引用
// (a > 10) ? a : 100 = 50;   // 编译错误！

// 正确：两边都必须是变量（左值）
int b = 100;
(a > 10) ? a : b = 50;   // OK
```

## 五、三目运算符的类型要求

值1和值2的类型必须**兼容**（可以互相转换），否则编译报错。

```cpp
#include <iostream>
using namespace std;

int main() {
    bool condition = true;

    // 类型相同：没问题
    int x = condition ? 5 : 10;          // int 和 int
    double y = condition ? 3.14 : 2.7;   // double 和 double

    // 类型不同但可以转换：编译器自动转换
    double z = condition ? 5 : 3.14;     // int 和 double -> double

    return 0;
}
```

## 六、优先级

三目运算符的优先级**非常低**，仅高于赋值运算符和逗号运算符。

```
算术 > 关系 > 逻辑 > 三目 > 赋值 > 逗号
```

### 6.1 优先级陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5, b = 10;

    // 很多人以为这行等于：cout << ((a > b) ? a : b)
    // 实际上 << 优先级高于 ?: ，所以会报错或行为异常
    // cout << a > b ? a : b;   // 错误！

    // 正确写法：加括号
    cout << ((a > b) ? a : b) << endl;   // 输出 10

    return 0;
}
```

### 6.2 赋值中的优先级

```cpp
int a = 5, b = 10;
int result;

// 这行是可以的，因为 = 优先级低于 ?:
result = a > b ? a : b;   // 等价于：result = (a > b ? a : b);
cout << result << endl;    // 10
```

## 七、嵌套三目运算符（不推荐）

三目运算符可以嵌套，但**强烈不推荐**，可读性极差。

```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 85;

    // 嵌套三目运算符：判断等级（不推荐！可读性很差）
    string grade = (score >= 90) ? "优秀" :
                   (score >= 80) ? "良好" :
                   (score >= 70) ? "中等" :
                   (score >= 60) ? "及格" : "不及格";
    cout << "等级: " << grade << endl;   // 良好

    // 推荐写法：用 if/else
    if (score >= 90) {
        grade = "优秀";
    } else if (score >= 80) {
        grade = "良好";
    } else if (score >= 70) {
        grade = "中等";
    } else if (score >= 60) {
        grade = "及格";
    } else {
        grade = "不及格";
    }
    cout << "等级: " << grade << endl;   // 良好

    return 0;
}
```

## 八、实战案例

### 案例1：判断奇偶

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = 7;
    string result = (num % 2 == 0) ? "偶数" : "奇数";
    cout << num << "是" << result << endl;
    // 输出：7是奇数
    return 0;
}
```

### 案例2：绝对值

```cpp
#include <iostream>
using namespace std;

int main() {
    int num = -15;
    int absValue = (num >= 0) ? num : -num;
    cout << "|" << num << "| = " << absValue << endl;
    // 输出：|-15| = 15
    return 0;
}
```

### 案例3：防止除以0

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 100, b = 0;
    // 安全除法：如果除数为0，结果设为0
    int result = (b != 0) ? (a / b) : 0;
    cout << "结果: " << result << endl;   // 0（因为b为0）
    return 0;
}
```

### 案例4：条件输出

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 15;
    cout << "你" << (age >= 18 ? "已成年" : "未成年") << endl;
    // 输出：你未成年
    return 0;
}
```

## 九、综合练习

```cpp
#include <iostream>
using namespace std;

int main() {
    int a, b;
    cout << "请输入两个整数: ";
    cin >> a >> b;

    int maxVal = (a > b) ? a : b;
    int minVal = (a < b) ? a : b;

    cout << "较大值: " << maxVal << endl;
    cout << "较小值: " << minVal << endl;

    // 判断正负零
    int num;
    cout << "请输入一个整数: ";
    cin >> num;

    string type = (num > 0) ? "正数" : (num < 0) ? "负数" : "零";
    cout << num << " 是 " << type << endl;

    return 0;
}
```

## 十、总结

| 知识点 | 要点 |
|--------|------|
| 语法 | `条件 ? 值1 : 值2` |
| 条件为真 | 返回值1 |
| 条件为假 | 返回值2 |
| 作左值 | 两边都是变量时，返回引用，可以赋值 |
| 优先级 | 非常低，建议**总是加括号** |
| 嵌套 | 技术上可以，但**不推荐** |
| 适用场景 | 简单的二选一赋值 |
