# 16 - 逻辑运算符详解

## 一、什么是逻辑运算符？

逻辑运算符用于**组合多个条件**，最终结果是 `bool` 类型（`true` 或 `false`）。
C++ 中有三个逻辑运算符：

| 运算符 | 名称 | 读法 | 示例 |
|--------|------|------|------|
| `!`    | 逻辑非 | "非" | `!a` |
| `&&`   | 逻辑与 | "并且" | `a && b` |
| `\|\|` | 逻辑或 | "或者" | `a \|\| b` |

---

## 二、逻辑非 `!`（单目运算符）

逻辑非的作用是**取反**：真变假，假变真。

```cpp
#include <iostream>
using namespace std;

int main() {
    bool a = true;
    bool b = false;

    cout << "!true  = " << !a << endl;   // 输出 0（false）
    cout << "!false = " << !b << endl;   // 输出 1（true）

    // 对数字取反：0 为假，非 0 为真
    int num = 0;
    cout << "!0   = " << !num << endl;   // 1（0 是假，取反为真）
    cout << "!100 = " << !100 << endl;   // 0（100 是真，取反为假）

    return 0;
}
```

**记忆技巧**：`!` 就像一个开关，按一下就翻转。

---

## 三、逻辑与 `&&`（双目运算符）

逻辑与的含义是"**并且**"——两个条件**都为真**，结果才为真。
只要有一个为假，结果就是假。

```cpp
#include <iostream>
using namespace std;

int main() {
    int age = 25;
    int score = 85;

    // 条件：年龄大于18 并且 分数大于60
    if (age > 18 && score > 60) {
        cout << "成年人且及格" << endl;   // 会执行
    }

    // 只要有一个条件不满足，就不执行
    if (age > 18 && score > 90) {
        cout << "不会执行" << endl;       // score=85 不大于90
    }

    return 0;
}
```

**生活类比**：钥匙 **并且** 密码都对，门才能打开。

---

## 四、逻辑或 `||`（双目运算符）

逻辑或的含义是"**或者**"——两个条件**有一个为真**，结果就为真。
只有两个都为假，结果才是假。

```cpp
#include <iostream>
using namespace std;

int main() {
    bool hasKey = false;
    bool hasPassword = true;

    // 有钥匙 或者 有密码，都能进门
    if (hasKey || hasPassword) {
        cout << "可以进门" << endl;   // 会执行
    }

    // 两个都不满足
    bool a = false, b = false;
    if (a || b) {
        cout << "不会执行" << endl;
    }

    return 0;
}
```

**生活类比**：公交卡 **或者** 现金，都可以坐车。

## 五、真值表（必须记住）

```
+-------+-------+------+----------+----------+
|   a   |   b   |  !a  |  a && b  | a || b   |
+-------+-------+------+----------+----------+
| true  | true  | false|  true    |  true    |
| true  | false | false|  false   |  true    |
| false | true  | true |  false   |  true    |
| false | false | true |  false   |  false   |
+-------+-------+------+----------+----------+
```

**口诀**：
- `&&`：全真为真，有假则假
- `||`：有真为真，全假则假
- `!`：真假互换

---

## 六、短路求值（重要！）

短路求值是编译器的优化策略：**一旦能确定结果，就不再计算后面的表达式**。

### 6.1 `&&` 的短路

当左边为 **false** 时，右边**不会执行**（因为结果一定是 false）。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 0;
    int b = 5;

    // a > 10 为 false，整个表达式已经确定为 false
    // 所以 b++ 不会执行！
    if (a > 10 && b++) {
        cout << "不会执行" << endl;
    }
    cout << "b = " << b << endl;   // 输出 5（b 没有自增！）

    return 0;
}
```

### 6.2 `||` 的短路

当左边为 **true** 时，右边**不会执行**（因为结果一定是 true）。

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 5;
    int b = 10;

    // a > 0 为 true，整个表达式已经确定为 true
    // 所以 b++ 不会执行！
    if (a > 0 || b++) {
        cout << "会执行" << endl;
    }
    cout << "b = " << b << endl;   // 输出 10（b 没有自增！）

    return 0;
}
```

### 6.3 短路陷阱

```cpp
// 危险！不要在逻辑表达式中放有副作用的操作
int x = 0;
if (x != 0 && 10 / x > 2) {
    // 短路保护了这里不会除以零
    // 但如果写反：(10/x > 2 && x != 0) 就会崩溃！
}
```

**最佳实践**：把容易失败的、需要保护的条件放在 `&&` 的**左边**。

## 七、逻辑运算符的优先级

优先级从高到低：

```
! (逻辑非)    -> 最高，单目运算符优先
&& (逻辑与)   -> 中间
|| (逻辑或)   -> 最低
```

**记忆口诀**：`! > && > ||`（非 > 与 > 或）

```cpp
// 等价关系演示
bool result = !true && false || true;
// 第一步：!true -> false
// 第二步：false && false -> false
// 第三步：false || true -> true
// result = true

// 建议加上括号更清晰
bool result2 = ((!true) && false) || true;
```

### 优先级陷阱

```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 1, b = 2, c = 3;

    // 很多人以为这等价于：a > b && (b > c || a > c)
    // 实际上 || 优先级低于 &&，所以等价于：
    // (a > b && b > c) || a > c
    if (a > b && b > c || a > c) {
        // ...
    }

    // 保险起见，永远加括号！
    if ((a > b) && ((b > c) || (a > c))) {
        // 意图一目了然
    }

    return 0;
}
```

## 八、综合实战案例

### 案例1：闰年判断

```cpp
#include <iostream>
using namespace std;

int main() {
    int year;
    cout << "请输入年份：";
    cin >> year;

    // 闰年条件：
    // 能被4整除 且 不能被100整除
    // 或者 能被400整除
    bool isLeap = (year % 4 == 0 && year % 100 != 0) || (year % 400 == 0);

    if (isLeap) {
        cout << year << "年是闰年" << endl;
    } else {
        cout << year << "年不是闰年" << endl;
    }

    return 0;
}
```

### 案例2：用户登录验证

```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string username = "admin";
    string password = "123456";
    bool isActive = true;

    if (username == "admin" && password == "123456" && isActive) {
        cout << "登录成功！" << endl;
    } else if (!isActive) {
        cout << "账号已被禁用" << endl;
    } else {
        cout << "用户名或密码错误" << endl;
    }

    return 0;
}
```

## 九、总结

| 知识点 | 要点 |
|--------|------|
| `!` | 取反，真变假、假变真 |
| `&&` | 全真为真，有假则假 |
| `\|\|` | 有真为真，全假则假 |
| 短路求值 | `&&`左假跳右，`\|\|`左真跳右 |
| 优先级 | `!` > `&&` > `\|\|` |
| 最佳实践 | 复杂条件**一定加括号** |
