# if 选择结构详解

## 一、什么是选择结构

选择结构（也叫分支结构）是程序的三大基本结构之一。它的作用是根据**条件**来决定执行哪一段代码。

生活中的例子：
- 如果**天气下雨**，就**带伞**，否则**不带伞**
- 如果**成绩>=60**，就**及格**，否则**不及格**

---

## 二、if 单行语句

### 2.1 基本语法

```cpp
if (条件表达式)
{
    // 条件为真时执行的代码
}
```

### 2.2 执行流程

1. 计算括号内**条件表达式**的值
2. 如果条件为**真（非0）**，执行大括号内的代码
3. 如果条件为**假（0）**，跳过大括号，继续执行后面的代码

### 2.3 代码示例

```cpp
#include <iostream>
using namespace std;

int main()
{
    int age = 18;

    // 如果年龄大于等于18岁，输出"你成年了"
    if (age >= 18)
    {
        cout << "你成年了！" << endl;
    }

    cout << "程序继续执行" << endl;

    return 0;
}
```

输出结果：
```
你成年了！
程序继续执行
```

### 2.4 注意事项：大括号省略问题

**C++中，if后面如果不加大括号，默认只控制紧跟的一条语句！**

```cpp
#include <iostream>
using namespace std;

int main()
{
    int score = 90;

    // 错误写法：没有大括号，if只控制下面一条语句
    if (score >= 60)
        cout << "恭喜你及格了！" << endl;  // 这条受if控制
        cout << "继续努力！" << endl;       // 这条不受if控制！总是会执行！

    return 0;
}
```

上面的代码中，"继续努力！"**无论分数多少都会输出**，因为它不在if的控制范围内。

**正确写法：始终加上大括号**

```cpp
#include <iostream>
using namespace std;

int main()
{
    int score = 90;

    // 正确写法：加上大括号，明确if的控制范围
    if (score >= 60)
    {
        cout << "恭喜你及格了！" << endl;
        cout << "继续努力！" << endl;
    }

    return 0;
}
```

> **最佳实践：即使if只控制一条语句，也建议始终加上大括号，避免后续修改时出错。**

## 三、if...else 语句

### 3.1 基本语法

```cpp
if (条件表达式)
{
    // 条件为真时执行的代码
}
else
{
    // 条件为假时执行的代码
}
```

### 3.2 执行流程

1. 计算条件表达式的值
2. 如果条件为**真**，执行if后面大括号的代码
3. 如果条件为**假**，执行else后面大括号的代码
4. **两者只会执行其中一个**，不会同时执行

### 3.3 代码示例

```cpp
#include <iostream>
using namespace std;

int main()
{
    int score;
    cout << "请输入你的分数：";
    cin >> score;

    // if...else 二选一
    if (score >= 60)
    {
        cout << "恭喜你，及格了！" << endl;
    }
    else
    {
        cout << "很遗憾，不及格，继续努力！" << endl;
    }

    return 0;
}
```

### 3.4 三元运算符（补充）

C++中有一个简洁的写法可以替代简单的if...else：

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 10;
    int b = 20;

    // 三元运算符：条件 ? 真值 : 假值
    // 等价于 if...else
    int max = (a > b) ? a : b;
    cout << "较大的数是：" << max << endl;

    // 等价的if...else写法：
    // if (a > b)
    //     max = a;
    // else
    //     max = b;

    return 0;
}
```

## 四、if...else if...else 多条件语句

### 4.1 基本语法

```cpp
if (条件1)
{
    // 条件1为真时执行
}
else if (条件2)
{
    // 条件1为假，且条件2为真时执行
}
else if (条件3)
{
    // 条件1、2都为假，且条件3为真时执行
}
else
{
    // 以上所有条件都为假时执行
}
```

### 4.2 执行流程

1. 从上到下依次判断每个条件
2. **一旦某个条件为真，就执行对应的代码块，然后跳出整个if结构**
3. 如果所有条件都不满足，执行else中的代码（else可以省略）

### 4.3 代码示例：分数等级判断

```cpp
#include <iostream>
using namespace std;

int main()
{
    int score;
    cout << "请输入分数（0-100）：";
    cin >> score;

    // 输入合法性检查
    if (score < 0 || score > 100)
    {
        cout << "输入的分数不合法！" << endl;
    }
    else if (score >= 90)   // 90分及以上：优秀
    {
        cout << "成绩等级：优秀（A）" << endl;
    }
    else if (score >= 80)   // 80-89分：良好
    {
        cout << "成绩等级：良好（B）" << endl;
    }
    else if (score >= 70)   // 70-79分：中等
    {
        cout << "成绩等级：中等（C）" << endl;
    }
    else if (score >= 60)   // 60-69分：及格
    {
        cout << "成绩等级：及格（D）" << endl;
    }
    else                    // 60分以下：不及格
    {
        cout << "成绩等级：不及格（F）" << endl;
    }

    return 0;
}
```

### 4.4 条件顺序很重要！

```cpp
#include <iostream>
using namespace std;

int main()
{
    int score = 85;

    // 错误写法：条件顺序不对
    if (score >= 60)    // 85 >= 60 为真，会进入这里
    {
        cout << "及格" << endl;  // 输出"及格"
    }
    else if (score >= 90)   // 永远不会执行到这里！
    {
        cout << "优秀" << endl;
    }
    else if (score >= 80)   // 永远不会执行到这里！
    {
        cout << "良好" << endl;
    }

    // 正确写法：从大到小判断
    // if (score >= 90)
    //     cout << "优秀" << endl;
    // else if (score >= 80)
    //     cout << "良好" << endl;
    // else if (score >= 60)
    //     cout << "及格" << endl;

    return 0;
}
```

> **注意：多条件if中，条件的顺序非常重要！应该把更严格的条件放在前面。**

## 五、if 的嵌套使用

### 5.1 基本语法

```cpp
if (外层条件)
{
    // 外层条件为真时执行

    if (内层条件)
    {
        // 外层和内层条件都为真时执行
    }
    else
    {
        // 外层为真，内层为假时执行
    }
}
```

### 5.2 代码示例：嵌套if

```cpp
#include <iostream>
using namespace std;

int main()
{
    int age;
    bool hasTicket;

    cout << "请输入年龄：";
    cin >> age;
    cout << "是否有票（1=有，0=没有）：";
    cin >> hasTicket;

    // 外层判断年龄
    if (age >= 18)
    {
        cout << "年龄符合要求。" << endl;

        // 内层判断是否有票
        if (hasTicket)
        {
            cout << "可以进入电影院！" << endl;
        }
        else
        {
            cout << "请先购票！" << endl;
        }
    }
    else
    {
        cout << "未成年人需要家长陪同！" << endl;

        // 内层判断
        if (hasTicket)
        {
            cout << "请在家长陪同下进入。" << endl;
        }
        else
        {
            cout << "请先购票，并由家长陪同！" << endl;
        }
    }

    return 0;
}
```

### 5.3 嵌套层次不要太深

```cpp
// 不推荐：嵌套层次太深，代码难以阅读
if (a)
{
    if (b)
    {
        if (c)
        {
            if (d)
            {
                // 代码缩进太深，难以维护
            }
        }
    }
}

// 推荐：用逻辑运算符或提前返回来减少嵌套
if (a && b && c && d)
{
    // 代码更清晰
}
```

## 六、条件表达式详解

### 6.1 什么是真，什么是假

在C++中：
- **假（false）**：值为 `0`
- **真（true）**：值为**任何非0的数**（包括负数）

```cpp
#include <iostream>
using namespace std;

int main()
{
    // 以下都是"假"
    if (0)
    {
        cout << "这行不会执行" << endl;
    }

    // 以下都是"真"
    if (1)
    {
        cout << "1为真" << endl;
    }

    if (-1)
    {
        cout << "-1也为真" << endl;
    }

    if (42)
    {
        cout << "42也为真" << endl;
    }

    // bool类型
    bool flag = true;   // true就是1
    if (flag)
    {
        cout << "flag为真" << endl;
    }

    return 0;
}
```

### 6.2 关系运算符

| 运算符 | 含义 | 示例 |
|--------|------|------|
| `==` | 等于 | `a == b` |
| `!=` | 不等于 | `a != b` |
| `>` | 大于 | `a > b` |
| `<` | 小于 | `a < b` |
| `>=` | 大于等于 | `a >= b` |
| `<=` | 小于等于 | `a <= b` |

### 6.3 逻辑运算符

| 运算符 | 含义 | 示例 | 说明 |
|--------|------|------|------|
| `&&` | 与（并且） | `a && b` | a和b都为真，结果才为真 |
| `\|\|` | 或（或者） | `a \|\| b` | a和b有一个为真，结果就为真 |
| `!` | 非（取反） | `!a` | 真变假，假变真 |

```cpp
#include <iostream>
using namespace std;

int main()
{
    int age = 25;
    double income = 8000;

    // && 逻辑与：两个条件都满足
    if (age >= 18 && income >= 5000)
    {
        cout << "符合贷款条件" << endl;
    }

    // || 逻辑或：至少一个条件满足
    if (age < 18 || age > 60)
    {
        cout << "可以享受优惠" << endl;
    }

    // ! 逻辑非：取反
    bool isStudent = false;
    if (!isStudent)
    {
        cout << "你不是学生" << endl;
    }

    return 0;
}
```

## 七、常见错误

### 7.1 错误一：`if(a=1)` vs `if(a==1)`

**这是新手最常犯的错误之一！**

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 0;

    // 错误写法：a=1 是赋值，不是比较！
    // a=1 的结果是1（非0），所以条件永远为真
    if (a = 1)
    {
        cout << "a等于1" << endl;  // 这行总是会执行！
    }

    // 正确写法：用 == 进行比较
    a = 0;
    if (a == 1)
    {
        cout << "a等于1" << endl;  // 这行不会执行
    }
    else
    {
        cout << "a不等于1" << endl;  // 这行会执行
    }

    return 0;
}
```

> **建议：把常量写在左边，如 `if(1 == a)`，这样如果不小心写成 `if(1 = a)`，编译器会报错。**

### 7.2 错误二：else 的匹配规则（就近原则）

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a = 1;
    int b = 2;

    // else 总是与最近的、未匹配的 if 配对
    if (a == 1)
        if (b == 2)
            cout << "a=1, b=2" << endl;
        else                    // 这个else匹配的是 if(b==2)，不是 if(a==1)！
            cout << "a=1, b!=2" << endl;

    // 如果想让else匹配第一个if，必须用大括号
    if (a == 1)
    {
        if (b == 2)
        {
            cout << "a=1, b=2" << endl;
        }
    }
    else    // 现在这个else匹配的是 if(a==1)
    {
        cout << "a!=1" << endl;
    }

    return 0;
}
```

### 7.3 错误三：忘记大括号导致逻辑错误

```cpp
#include <iostream>
using namespace std;

int main()
{
    int x = 5;

    // 错误：想让两条语句都在if内执行，但没加大括号
    if (x > 0)
        cout << "x是正数" << endl;
        cout << "x的值是：" << x << endl;  // 这行总是执行！

    // 正确写法
    if (x > 0)
    {
        cout << "x是正数" << endl;
        cout << "x的值是：" << x << endl;
    }

    return 0;
}
```

### 7.4 错误四：比较浮点数

```cpp
#include <iostream>
using namespace std;

int main()
{
    double a = 0.1 + 0.2;

    // 错误：浮点数有精度问题，不能直接用 == 比较
    if (a == 0.3)
    {
        cout << "相等" << endl;  // 可能不会执行！
    }

    // 正确：比较浮点数的差值是否足够小
    if (abs(a - 0.3) < 0.0001)
    {
        cout << "近似相等" << endl;  // 这个会执行
    }

    return 0;
}
```

## 八、实战案例

### 8.1 案例一：闰年判断

**闰年规则：**
1. 能被4整除但不能被100整除的年份
2. 或者能被400整除的年份

```cpp
#include <iostream>
using namespace std;

int main()
{
    int year;
    cout << "请输入一个年份：";
    cin >> year;

    // 方法一：嵌套if
    if (year % 4 == 0)
    {
        if (year % 100 == 0)
        {
            if (year % 400 == 0)
            {
                cout << year << "年是闰年" << endl;
            }
            else
            {
                cout << year << "年不是闰年" << endl;
            }
        }
        else
        {
            cout << year << "年是闰年" << endl;
        }
    }
    else
    {
        cout << year << "年不是闰年" << endl;
    }

    // 方法二：逻辑运算符（更简洁）
    if ((year % 4 == 0 && year % 100 != 0) || (year % 400 == 0))
    {
        cout << year << "年是闰年" << endl;
    }
    else
    {
        cout << year << "年不是闰年" << endl;
    }

    return 0;
}
```

### 8.2 案例二：三个数排序

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a, b, c;
    cout << "请输入三个整数（用空格分隔）：";
    cin >> a >> b >> c;

    // 用临时变量交换，使 a <= b <= c
    if (a > b)
    {
        int temp = a;
        a = b;
        b = temp;
    }
    if (a > c)
    {
        int temp = a;
        a = c;
        c = temp;
    }
    if (b > c)
    {
        int temp = b;
        b = c;
        c = temp;
    }

    cout << "从小到大排序：" << a << " " << b << " " << c << endl;

    return 0;
}
```

### 8.3 案例三：简易计算器

```cpp
#include <iostream>
using namespace std;

int main()
{
    double num1, num2;
    char op;

    cout << "请输入表达式（如 3 + 5）：";
    cin >> num1 >> op >> num2;

    if (op == '+')
    {
        cout << num1 << " + " << num2 << " = " << num1 + num2 << endl;
    }
    else if (op == '-')
    {
        cout << num1 << " - " << num2 << " = " << num1 - num2 << endl;
    }
    else if (op == '*')
    {
        cout << num1 << " * " << num2 << " = " << num1 * num2 << endl;
    }
    else if (op == '/')
    {
        // 除法要检查除数是否为0
        if (num2 != 0)
        {
            cout << num1 << " / " << num2 << " = " << num1 / num2 << endl;
        }
        else
        {
            cout << "错误：除数不能为0！" << endl;
        }
    }
    else
    {
        cout << "错误：不支持的运算符 '" << op << "'" << endl;
    }

    return 0;
}
```

## 九、总结

| 语法 | 适用场景 |
|------|----------|
| `if` | 只需要判断一个条件 |
| `if...else` | 二选一的情况 |
| `if...else if...else` | 多条件判断 |
| 嵌套`if` | 条件需要分层判断 |

**关键要点：**
1. **始终加大括号**，即使只有一条语句
2. **条件顺序**很重要，严格的条件放前面
3. `=`是赋值，`==`是比较，不要混淆
4. `else`与**最近的、未匹配的**`if`配对
5. 浮点数比较不要用`==`，要用差值判断
