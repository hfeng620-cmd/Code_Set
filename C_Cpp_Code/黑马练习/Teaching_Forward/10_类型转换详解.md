# 10 类型转换详解

## 一、什么是类型转换

- 类型转换是将一种数据类型的值转换为另一种数据类型
- C++ 中有两种类型转换方式：**隐式转换**（自动）和 **显式转换**（手动）

---

## 二、隐式转换（自动类型转换）

### 2.1 什么是隐式转换

编译器在 **不需要程序员干预** 的情况下，自动将一种类型转换为另一种类型。

```cpp
// 示例1：int -> double（小类型转大类型，安全）
int a = 10;
double b = a;   // int 自动转为 double，b = 10.0
cout << b << endl;  // 10

// 示例2：int 和 double 运算时，int 自动提升为 double
int x = 10;
double y = 3.14;
double result = x + y;  // x 先转为 10.0，再和 3.14 相加
cout << result << endl;  // 13.14

// 示例3：char -> int（char 本质就是整数）
char ch = 'A';
int num = ch;   // char 自动转为 int，num = 65
cout << num << endl;  // 65
```

### 2.2 类型大小排序（从小到大）

```
bool -> char -> short -> int -> long -> long long -> float -> double -> long double
```

- **小类型 -> 大类型**：安全转换，数据不丢失（如 int -> double）
- **大类型 -> 小类型**：危险转换，可能丢失数据（如 double -> int）

### 2.3 隐式转换的风险

```cpp
// 风险1：大类型转小类型，数据截断
double pi = 3.14159;
int a = pi;       // double -> int，小数部分被截断
cout << a << endl; // 3（丢失了 .14159）

// 风险2：溢出
int big = 300;
char ch = big;    // int -> char，300 超过 char 范围（-128~127）
cout << (int)ch << endl;  // 44（溢出了！300 % 256 = 44）

// 风险3：精度丢失
float f = 1.23456789f;  // float 精度有限
double d = f;            // float -> double，精度已经丢失了
cout << d << endl;       // 1.23457（不是 1.23456789）
```

### 2.4 隐式转换的常见场景

```cpp
// 场景1：赋值时
int a = 3.14;       // double -> int，截断
double b = 10;      // int -> double，安全

// 场景2：算术运算时
int x = 10;
double y = 3.0;
auto result = x / y;  // x 转为 double，结果是 double
cout << result << endl; // 3.33333

// 场景3：函数传参时
void printDouble(double d)
{
    cout << d << endl;
}
int num = 42;
printDouble(num);  // int 自动转为 double

// 场景4：函数返回值时
double getDouble()
{
    int a = 10;
    return a;  // int 自动转为 double，返回 10.0
}
```

## 三、显式转换（强制类型转换）

当隐式转换不能满足需求，或者你想 **明确表达转换意图** 时，使用显式转换。

### 3.1 C 风格强制转换

```cpp
// 语法：(目标类型) 表达式

double pi = 3.14159;
int a = (int)pi;        // C 风格，把 pi 强制转为 int
cout << a << endl;       // 3

// 也可以写成函数形式
int b = int(pi);         // 效果一样
cout << b << endl;       // 3

// 常见用法
int x = 10, y = 3;
double result = (double)x / y;  // 把 x 转为 double 再除
cout << result << endl;          // 3.33333（不转的话是整数除法，结果是 3）

// 不转的情况
int result2 = x / y;   // 整数除法
cout << result2 << endl; // 3（小数部分丢失）
```

### 3.2 C++ 风格强制转换（推荐）

```cpp
// 语法：static_cast<目标类型>(表达式)

double pi = 3.14159;
int a = static_cast<int>(pi);   // C++ 风格
cout << a << endl;               // 3

int x = 10, y = 3;
double result = static_cast<double>(x) / y;
cout << result << endl;           // 3.33333
```

### 3.3 C 风格 vs C++ 风格对比

| 对比项 | C 风格 `(int)a` | C++ 风格 `static_cast<int>(a)` |
|--------|-----------------|--------------------------------|
| 语法 | 简洁 | 稍长 |
| 安全性 | 不检查，编译器不报错 | 编译器会检查类型兼容性 |
| 可搜索性 | 不容易在代码中搜索 | 容易搜索（grep static_cast） |
| 推荐度 | 不推荐 | 推荐 |

```cpp
// C 风格的问题：什么都能转，不安全
class Dog {};
class Cat {};
Dog dog;
// Cat cat = (Cat)dog;        // C 风格不报错，但逻辑上完全错误！

// C++ 风格更安全：编译器会检查
// Cat cat2 = static_cast<Cat>(dog);  // 编译报错！类型不兼容
```

## 四、四种 C++ 类型转换（了解）

C++ 提供了四种命名的类型转换运算符，最常用的是 `static_cast`：

```cpp
// 1. static_cast：最常用，用于大多数类型转换
int a = 10;
double b = static_cast<double>(a);   // int -> double

// 2. dynamic_cast：用于多态类型的安全转换（学到面向对象时再了解）
// Base* base = dynamic_cast<Base*>(derived);

// 3. const_cast：去除 const 属性（很少用，有风险）
const int c = 10;
int* p = const_cast<int*>(&c);  // 去除 const

// 4. reinterpret_cast：最底层的转换（几乎不用，非常危险）
// int* p = reinterpret_cast<int*>(0x12345678);
```

> 初学阶段只需要掌握 `static_cast` 就够了。

## 五、转换规则详解

### 5.1 算术类型之间的转换规则

```cpp
// 规则1：bool -> 整型
bool b = true;
int n = b;     // true -> 1

// 规则2：char -> 整型
char ch = 'A';
int n2 = ch;   // 'A' -> 65

// 规则3：整型 -> 浮点型（可能有精度丢失）
int big = 123456789;
float f = big;  // float 精度有限，可能不精确
cout << f << endl;

// 规则4：浮点型 -> 整型（截断小数部分）
double d = 9.99;
int n3 = d;     // 9（不是四舍五入，是直接截断！）
cout << n3 << endl;

// 规则5：大整型 -> 小整型（可能溢出）
int big2 = 1000;
short s = big2;  // short 可能装不下 1000（如果 short 是 2 字节）
```

### 5.2 赋值时的截断

```cpp
// 截断：大类型赋值给小类型时发生

// 浮点 -> 整数：截断小数部分
double d1 = 3.99;
int i1 = d1;          // i1 = 3（不是 4！直接截断，不四舍五入）
cout << i1 << endl;

double d2 = -3.99;
int i2 = d2;          // i2 = -3（向零方向截断）
cout << i2 << endl;

// int -> short：可能溢出
int big = 40000;
short s = big;         // short 范围是 -32768~32767，溢出了
cout << s << endl;     // 结果不可预测

// double -> float：精度丢失
double precise = 3.14159265358979;
float rough = precise;  // float 精度只有约 6~7 位有效数字
cout << rough << endl;  // 3.14159
```

### 5.3 int 除法 vs double 除法

```cpp
int a = 10, b = 3;

// 整数除法：结果也是整数，小数部分直接丢弃
int result1 = a / b;
cout << result1 << endl;  // 3（不是 3.333...）

// 想要精确结果，至少一个操作数要转为浮点数
double result2 = (double)a / b;     // C 风格
double result3 = static_cast<double>(a) / b;  // C++ 风格
cout << result2 << endl;  // 3.33333
cout << result3 << endl;  // 3.33333

// 或者直接用 double 变量
double x = 10.0, y = 3.0;
double result4 = x / y;
cout << result4 << endl;  // 3.33333
```

## 六、常见错误

### 6.1 忘记转换导致整数除法

```cpp
int total = 100;
int count = 3;

// 错误：整数除法丢失小数
double avg1 = total / count;
cout << avg1 << endl;  // 33（不是 33.333...）

// 正确：先转换再除
double avg2 = static_cast<double>(total) / count;
cout << avg2 << endl;  // 33.3333
```

### 6.2 截断导致数据丢失

```cpp
double price = 9.99;
int intPrice = price;  // 截断为 9，不是四舍五入！
cout << intPrice << endl;  // 9

// 如果要四舍五入
int rounded = static_cast<int>(price + 0.5);
cout << rounded << endl;  // 10
```

### 6.3 溢出问题

```cpp
// char 范围：-128 ~ 127
int big = 200;
char ch = big;  // 溢出！
cout << (int)ch << endl;  // 结果不可预测
```

## 七、综合示例

```cpp
#include <iostream>
using namespace std;

int main()
{
    cout << "===== 类型转换综合示例 =====" << endl;

    // 1. 隐式转换：小 -> 大（安全）
    int intVal = 42;
    double doubleVal = intVal;  // int -> double
    cout << "\nint -> double: " << intVal << " -> " << doubleVal << endl;

    // 2. 隐式转换：大 -> 小（危险）
    double pi = 3.14159;
    int truncated = pi;  // 截断
    cout << "double -> int: " << pi << " -> " << truncated << endl;

    // 3. 显式转换：整数除法问题
    int a = 7, b = 2;
    cout << "\n整数除法: " << a << " / " << b << " = " << a / b << endl;           // 3
    cout << "浮点除法: " << a << " / " << b << " = "
         << static_cast<double>(a) / b << endl;  // 3.5

    // 4. char 和 int 互转
    char ch = 'A';
    int ascii = static_cast<int>(ch);
    cout << "\n字符 '" << ch << "' 的 ASCII 码: " << ascii << endl;

    int num = 97;
    char character = static_cast<char>(num);
    cout << "ASCII 码 " << num << " 对应的字符: '" << character << "'" << endl;

    // 5. 计算平均分（常见的坑）
    int scores[] = {85, 92, 78, 96, 88};
    int total = 0;
    for (int i = 0; i < 5; i++)
    {
        total += scores[i];
    }

    // 错误写法
    double avg1 = total / 5;
    cout << "\n错误平均分: " << avg1 << endl;  // 87（整数除法）

    // 正确写法
    double avg2 = static_cast<double>(total) / 5;
    cout << "正确平均分: " << avg2 << endl;  // 87.8

    // 6. 四舍五入
    double value = 3.7;
    int rounded = static_cast<int>(value + 0.5);
    cout << "\n" << value << " 四舍五入: " << rounded << endl;  // 4

    double value2 = 3.2;
    int rounded2 = static_cast<int>(value2 + 0.5);
    cout << value2 << " 四舍五入: " << rounded2 << endl;  // 3

    return 0;
}
```

## 八、总结

| 要点 | 说明 |
|------|------|
| 隐式转换 | 编译器自动完成，小->大安全，大->小有风险 |
| 显式转换 | 程序员手动指定，推荐用 `static_cast<>()` |
| 整数除法 | 两个 int 相除结果还是 int，小数部分丢失 |
| 浮点转整数 | 直接截断小数部分，不是四舍五入 |
| 溢出 | 大范围类型赋值给小范围类型可能溢出 |
| 原则 | 尽量避免数据丢失的转换，必须转换时用显式转换 |
