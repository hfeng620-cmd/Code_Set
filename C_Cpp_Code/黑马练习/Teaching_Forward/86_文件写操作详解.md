# 文件写操作详解

## 1. 概述

在C++中，文件操作是非常重要的功能。我们需要把程序中的数据保存到文件中，或者从文件中读取数据。
C++通过文件流（fstream）来实现文件操作，其中**写文件**使用的是 `ofstream` 类。

---

## 2. 需要包含的头文件

```cpp
#include <fstream>  // 文件操作必须包含这个头文件
```

注意：不是 `<iostream>`，而是 `<fstream>`。当然，如果你同时需要控制台输入输出，两个都要包含。

---

## 3. ofstream 类简介

`ofstream` 是 **Output File Stream** 的缩写，意思是"输出文件流"。
它专门用于**向文件写入数据**，用法和 `cout` 非常相似。

类的继承关系：
```
ios_base -> ios -> ostream -> ofstream
```

---

## 4. 写文件的基本步骤

写文件一共5个步骤：
1. 创建 `ofstream` 对象
2. 打开文件（指定文件路径）
3. 判断是否打开成功
4. 写入内容
5. 关闭文件

### 4.1 最简单的写文件示例

```cpp
#include <iostream>
#include <fstream>  // 必须包含
using namespace std;

int main() {
    // 第1步：创建 ofstream 对象
    ofstream ofs;

    // 第2步：打开文件（如果文件不存在会自动创建）
    ofs.open("test.txt");

    // 第3步：判断是否打开成功
    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    // 第4步：写入内容（用 << 运算符，就像 cout 一样）
    ofs << "Hello, World!" << endl;
    ofs << "这是第二行内容" << endl;
    ofs << "年龄：" << 18 << endl;

    // 第5步：关闭文件
    ofs.close();

    cout << "写入完成！" << endl;
    return 0;
}
```

### 4.2 用构造函数直接打开文件

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 创建对象的同时直接打开文件（构造函数传路径）
    ofstream ofs("test2.txt");

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    ofs << "直接在构造函数中打开文件" << endl;
    ofs.close();

    return 0;
}
```

## 5. 文件打开模式

打开文件时可以指定不同的模式，控制写入行为。

### 5.1 常用的打开模式

| 模式 | 说明 |
|------|------|
| `ios::out` | 输出模式（默认），覆盖原内容 |
| `ios::app` | 追加模式，在文件末尾写入 |
| `ios::trunc` | 截断模式，清空原内容 |
| `ios::binary` | 二进制模式 |

### 5.2 ios::out 模式（默认）

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 默认就是 ios::out 模式
    // 如果文件存在，会覆盖原有内容
    // 如果文件不存在，会创建新文件
    ofstream ofs("test_out.txt", ios::out);

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    ofs << "第一次写入的内容" << endl;
    ofs.close();

    // 再次打开，用 out 模式会覆盖之前的内容
    ofs.open("test_out.txt", ios::out);
    ofs << "第二次写入的内容（第一次的被覆盖了）" << endl;
    ofs.close();

    return 0;
}
```

### 5.3 ios::app 追加模式

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 先写入一些内容
    ofstream ofs("test_app.txt", ios::out);
    ofs << "第一行" << endl;
    ofs << "第二行" << endl;
    ofs.close();

    // 用 app 模式打开，在末尾追加内容
    ofs.open("test_app.txt", ios::app);
    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }
    ofs << "这是追加的内容" << endl;
    ofs << "这也是追加的内容" << endl;
    ofs.close();

    // 此时文件内容为：
    // 第一行
    // 第二行
    // 这是追加的内容
    // 这也是追加的内容

    return 0;
}
```

### 5.4 ios::trunc 截断模式

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 先创建一个有内容的文件
    ofstream ofs("test_trunc.txt");
    ofs << "原有内容1" << endl;
    ofs << "原有内容2" << endl;
    ofs.close();

    // 用 trunc 模式打开，会清空所有内容
    ofs.open("test_trunc.txt", ios::trunc);
    ofs << "新内容（原有内容已被清空）" << endl;
    ofs.close();

    return 0;
}
```

### 5.5 模式组合使用

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 用 | 运算符组合多个模式
    // ios::out | ios::app 表示输出且追加
    ofstream ofs("test_combine.txt", ios::out | ios::app);

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    ofs << "组合模式写入" << endl;
    ofs.close();

    // 二进制追加模式
    ofstream ofs2("test_binary.dat", ios::out | ios::binary);
    // ... 写入二进制数据
    ofs2.close();

    return 0;
}
```

## 6. 写入各种类型的数据

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ofstream ofs("test_types.txt");

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    // 写入字符串
    ofs << "这是一个字符串" << endl;

    // 写入整数
    int age = 25;
    ofs << "年龄：" << age << endl;

    // 写入浮点数
    double score = 95.5;
    ofs << "分数：" << score << endl;

    // 写入字符
    char ch = 'A';
    ofs << "字符：" << ch << endl;

    // 写入布尔值（会显示为0或1）
    bool flag = true;
    ofs << "布尔值：" << flag << endl;

    // 写入 string 对象
    string name = "张三";
    ofs << "姓名：" << name << endl;

    // 写入换行符的几种方式
    ofs << "方式1" << endl;      // endl 会刷新缓冲区
    ofs << "方式2\n";            // \n 只换行，不刷新缓冲区
    ofs << "方式3" << "\n";      // 同上

    // 写入多个数据（用 << 连接）
    ofs << "姓名：" << name << "，年龄：" << age << "，分数：" << score << endl;

    ofs.close();
    cout << "写入完成！" << endl;

    return 0;
}
```

## 7. 文件路径

### 7.1 相对路径

```cpp
// 相对路径：相对于当前程序运行的目录
ofstream ofs1("test.txt");           // 当前目录下的 test.txt
ofstream ofs2("data/test.txt");      // 当前目录下的 data 文件夹中的 test.txt
ofstream ofs3("../test.txt");        // 上一级目录下的 test.txt
ofstream ofs4("./output/result.txt"); // 当前目录下 output 文件夹中的 result.txt
```

### 7.2 绝对路径

```cpp
// 绝对路径：完整的文件路径
// Windows 路径注意反斜杠要转义，或者用正斜杠
ofstream ofs1("C:\\Users\\test\\Desktop\\output.txt");  // 反斜杠需要转义
ofstream ofs2("C:/Users/test/Desktop/output.txt");      // 正斜杠也可以
```

### 7.3 路径中的常见问题

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 错误示范：路径中的反斜杠没有转义
    // ofstream ofs("C:\Users\test\new.txt");  // 错误！\U \t \n 是转义字符

    // 正确写法1：用双反斜杠
    ofstream ofs1("C:\\Users\\test\\new.txt");

    // 正确写法2：用正斜杠（推荐）
    ofstream ofs2("C:/Users/test/new.txt");

    return 0;
}
```

## 8. 文件不存在时的行为

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 用 ios::out 模式打开一个不存在的文件
    // 结果：会自动创建这个文件！
    ofstream ofs("新文件.txt", ios::out);

    if (ofs.is_open()) {
        cout << "文件打开成功（不存在时会自动创建）" << endl;
        ofs << "这是新创建的文件" << endl;
        ofs.close();
    } else {
        cout << "文件打开失败" << endl;
    }

    return 0;
}
```

## 9. 完整的实际应用示例

### 9.1 将用户输入保存到文件

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    ofstream ofs("学生信息.txt", ios::app);  // 追加模式

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    string name;
    int age;
    double score;

    cout << "请输入姓名：";
    cin >> name;
    cout << "请输入年龄：";
    cin >> age;
    cout << "请输入分数：";
    cin >> score;

    // 写入文件
    ofs << "姓名：" << name << "\t";
    ofs << "年龄：" << age << "\t";
    ofs << "分数：" << score << endl;

    ofs.close();
    cout << "保存成功！" << endl;

    return 0;
}
```

### 9.2 生成CSV格式文件

```cpp
#include <iostream>
#include <fstream>
#include <string>
using namespace std;

int main() {
    // CSV文件可以用Excel打开
    ofstream ofs("data.csv");

    if (!ofs.is_open()) {
        cout << "文件打开失败！" << endl;
        return -1;
    }

    // 写入表头
    ofs << "姓名,年龄,城市,分数" << endl;

    // 写入数据
    ofs << "张三,20,北京,95" << endl;
    ofs << "李四,21,上海,88" << endl;
    ofs << "王五,22,广州,92" << endl;

    ofs.close();
    cout << "CSV文件生成完成！" << endl;

    return 0;
}
```

## 10. 常见错误和注意事项

### 10.1 忘记包含头文件

```cpp
// 错误：没有 #include <fstream>
// ofstream ofs;  // 编译错误：ofstream 未定义

// 正确：
#include <fstream>
```

### 10.2 忘记关闭文件

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream ofs("test.txt");
    ofs << "内容" << endl;
    // 忘记 ofs.close();  // 虽然程序结束时会自动关闭，但养成好习惯很重要

    // 如果不关闭文件：
    // 1. 可能导致数据丢失（缓冲区中的数据没有写入文件）
    // 2. 可能导致文件被占用，其他程序无法访问
    // 3. 可能导致资源泄漏

    ofs.close();  // 养成好习惯，用完就关
    return 0;
}
```

### 10.3 文件路径错误

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    // 路径中的文件夹不存在时，文件打开会失败
    ofstream ofs("不存在的文件夹/test.txt");

    if (!ofs.is_open()) {
        cout << "文件打开失败！请检查路径是否正确" << endl;
        return -1;
    }

    ofs.close();
    return 0;
}
```

### 10.4 权限不足

```cpp
// 可能导致打开失败的原因：
// 1. 文件被其他程序占用
// 2. 没有写入权限（如系统文件夹）
// 3. 磁盘已满
// 4. 路径格式错误

// 解决方法：总是检查 is_open()
```

### 10.5 使用 endl 和 \n 的区别

```cpp
#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream ofs1("endl测试.txt");
    ofs1 << "第一行" << endl;  // endl = 换行 + 刷新缓冲区
    ofs1 << "第二行" << endl;
    ofs1.close();

    ofstream ofs2("换行符测试.txt");
    ofs2 << "第一行\n";  // \n 只换行，不刷新缓冲区
    ofs2 << "第二行\n";
    ofs2.close();  // close() 会自动刷新缓冲区

    // 建议：
    // - 频繁写入时用 \n，性能更好
    // - 需要立即看到结果时用 endl
    // - 关闭文件前不需要手动 endl，close() 会处理

    return 0;
}
```

## 11. 总结

| 要点 | 说明 |
|------|------|
| 头文件 | `#include <fstream>` |
| 写文件类 | `ofstream` |
| 打开文件 | `ofs.open("路径")` 或构造函数传参 |
| 判断打开 | `ofs.is_open()` |
| 写入内容 | `ofs << 数据` |
| 关闭文件 | `ofs.close()` |
| 默认模式 | `ios::out`，覆盖原内容 |
| 追加模式 | `ios::app`，在末尾添加 |
| 自动创建 | 文件不存在时会自动创建 |

记住这5个步骤：**创建对象 -> 打开文件 -> 判断成功 -> 写入内容 -> 关闭文件**
