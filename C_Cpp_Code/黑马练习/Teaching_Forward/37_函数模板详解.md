# 函数模板详解

## 一、为什么需要函数模板

### 1.1 问题引入

假设我们要写一个交换两个变量值的函数，不同类型的变量需要写多个版本：

```cpp
#include <iostream>
using namespace std;

// 交换int
void swapInt(int &a, int &b) {
    int temp = a;
    a = b;
    b = temp;
}

// 交换double
void swapDouble(double &a, double &b) {
    double temp = a;
    a = b;
    b = temp;
}

// 交换char
void swapChar(char &a, char &b) {
    char temp = a;
    a = b;
    b = temp;
}

int main() {
    int a = 1, b = 2;
    swapInt(a, b);
    cout << "a=" << a << ", b=" << b << endl;

    double c = 1.5, d = 2.5;
    swapDouble(c, d);
    cout << "c=" << c << ", d=" << d << endl;

    return 0;
}
```

**问题：这些函数的逻辑完全一样，只是类型不同！重复写代码既麻烦又容易出错。**

### 1.2 解决方案：函数模板

> **函数模板（Function Template）可以创建一个通用函数，支持多种数据类型。**
> 程序员只需要写一次代码，编译器会根据传入的参数类型自动生成对应版本的函数。

```cpp
#include <iostream>
using namespace std;

// 定义函数模板
template<typename T>    // 声明模板，T 是类型参数（可以取任意名字）
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
}

int main() {
    int a = 1, b = 2;
    mySwap(a, b);           // 编译器自动推导 T = int
    cout << "a=" << a << ", b=" << b << endl;

    double c = 1.5, d = 2.5;
    mySwap(c, d);           // 编译器自动推导 T = double
    cout << "c=" << c << ", d=" << d << endl;

    char e = 'A', f = 'B';
    mySwap(e, f);           // 编译器自动推导 T = char
    cout << "e=" << e << ", f=" << f << endl;

    return 0;
}
```

**运行结果：**
```
a=2, b=1
c=2.5, d=1.5
e=B, f=A
```

## 二、函数模板的语法

### 2.1 基本语法格式

```cpp
// 语法格式：
// template<typename T>     或   template<class T>
// 返回值类型 函数名(参数列表) { 函数体 }

// template  - 关键字，表示"模板"
// typename  - 关键字，表示后面是一个类型名称（也可以用 class）
// T         - 类型参数名，代表某种数据类型（可以取任意名字）

// 以下两种写法完全等价：
template<typename T>
void func1(T a) { }

template<class T>
void func2(T a) { }
```

### 2.2 多个类型参数

```cpp
#include <iostream>
using namespace std;

// 可以定义多个类型参数
template<typename T1, typename T2>
void showTwo(T1 a, T2 b) {
    cout << "第一个值：" << a << "，类型大小：" << sizeof(T1) << endl;
    cout << "第二个值：" << b << "，类型大小：" << sizeof(T2) << endl;
    cout << "-------------------" << endl;
}

int main() {
    showTwo(10, 3.14);          // T1=int, T2=double
    showTwo("Hello", 'A');      // T1=const char*, T2=char
    showTwo(100, true);         // T1=int, T2=bool
    return 0;
}
```

### 2.3 模板的完整写法

```cpp
#include <iostream>
using namespace std;

// 模板函数：求两个数中的较大值
template<typename T>
T getMax(T a, T b) {
    return (a > b) ? a : b;
}

// 模板函数：打印数组
template<typename T>
void printArray(T arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i] << " ";
    }
    cout << endl;
}

// 模板函数：求绝对值
template<typename T>
T myAbs(T x) {
    return (x < 0) ? -x : x;
}

int main() {
    cout << "最大值：" << getMax(10, 20) << endl;
    cout << "最大值：" << getMax(3.14, 2.72) << endl;
    cout << "最大值：" << getMax('A', 'Z') << endl;

    cout << "---" << endl;

    int nums[] = {1, 2, 3, 4, 5};
    printArray(nums, 5);

    double vals[] = {1.1, 2.2, 3.3};
    printArray(vals, 3);

    cout << "---" << endl;

    cout << "|-5| = " << myAbs(-5) << endl;
    cout << "|-3.14| = " << myAbs(-3.14) << endl;

    return 0;
}
```

## 三、模板的两种调用方式

### 3.1 自动类型推导

```cpp
#include <iostream>
using namespace std;

template<typename T>
void func(T a) {
    cout << "T的类型大小：" << sizeof(T) << "，值：" << a << endl;
}

int main() {
    // 自动类型推导：编译器根据实参自动推导 T 的类型
    func(10);       // T = int
    func(3.14);     // T = double
    func('A');      // T = char
    func(true);     // T = bool
    return 0;
}
```

**注意：自动推导必须能推导出一致的类型！**

```cpp
template<typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    // cout << add(1, 3.14) << endl;   // 错误！T不能同时是int和double
    cout << add(1, 2) << endl;         // OK：T = int
    cout << add(1.5, 2.5) << endl;     // OK：T = double
    return 0;
}
```

### 3.2 显式指定类型

```cpp
#include <iostream>
using namespace std;

template<typename T>
void printValue(T a) {
    cout << "值：" << a << "，类型大小：" << sizeof(T) << endl;
}

template<typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    // 显式指定类型：在函数名后面用 <类型> 指定
    printValue<int>(10);           // 显式指定 T = int
    printValue<double>(3.14);      // 显式指定 T = double
    printValue<char>('A');         // 显式指定 T = char

    cout << "---" << endl;

    // 显式指定可以解决类型推导不一致的问题
    cout << add<double>(1, 3.14) << endl;   // OK：T = double

    return 0;
}
```

## 四、模板的推导限制与注意事项

### 4.1 自动推导不会进行隐式类型转换

```cpp
#include <iostream>
using namespace std;

template<typename T>
void func(T a, T b) {
    cout << "a=" << a << ", b=" << b << endl;
}

int main() {
    func(10, 20);          // OK：都是int
    func(1.5, 2.5);        // OK：都是double

    // func(10, 20.5);     // 错误！自动推导时，T不能同时是int和double
    func<double>(10, 20.5); // OK：显式指定T=double，10会隐式转换为10.0

    return 0;
}
```

### 4.2 模板参数必须能够匹配

```cpp
#include <iostream>
using namespace std;

template<typename T>
void compare(T a, T b) {
    if (a > b) cout << a << " > " << b << endl;
    else if (a < b) cout << a << " < " << b << endl;
    else cout << a << " == " << b << endl;
}

int main() {
    compare(10, 20);       // OK
    compare(3.14, 2.72);   // OK

    // 对于自定义类型（类），需要重载 > 和 < 运算符才能用于模板
    return 0;
}
```

## 五、模板函数与普通函数的调用规则

**当普通函数和模板函数同时存在时，编译器有明确的调用优先级。**

```cpp
#include <iostream>
using namespace std;

// 普通函数
int add(int a, int b) {
    cout << "调用了普通函数" << endl;
    return a + b;
}

// 函数模板
template<typename T>
T add(T a, T b) {
    cout << "调用了函数模板" << endl;
    return a + b;
}

int main() {
    // 情况1：如果普通函数完全匹配，优先调用普通函数
    cout << add(1, 2) << endl;          // 调用普通函数

    // 情况2：如果普通函数不匹配，使用模板函数
    cout << add(1.5, 2.5) << endl;      // 调用模板函数

    // 情况3：可以强制调用模板函数（使用空模板参数列表）
    cout << add<>(1, 2) << endl;        // 强制调用模板函数

    // 情况4：显式指定类型
    cout << add<double>(1, 2) << endl;  // 调用模板函数，指定T=double

    return 0;
}
```

**运行结果：**
```
调用了普通函数
3
调用了函数模板
4
调用了函数模板
3
调用了函数模板
3
```

**调用规则总结：**
```
1. 如果普通函数完全匹配 -> 优先调用普通函数
2. 如果普通函数需要类型转换 -> 优先调用模板函数（模板更精确）
3. 使用 <> 空模板参数 -> 强制调用模板函数
4. 使用 <类型> 显式指定 -> 强制调用模板函数
```

## 六、函数模板的重载

```cpp
#include <iostream>
using namespace std;

// 模板版本1：交换两个值
template<typename T>
void mySwap(T &a, T &b) {
    T temp = a;
    a = b;
    b = temp;
    cout << "调用了两参数版本" << endl;
}

// 模板版本2：交换数组（重载，参数个数不同）
template<typename T>
void mySwap(T a[], T b[], int size) {
    for (int i = 0; i < size; i++) {
        T temp = a[i];
        a[i] = b[i];
        b[i] = temp;
    }
    cout << "调用了数组版本" << endl;
}

int main() {
    int x = 1, y = 2;
    mySwap(x, y);
    cout << "x=" << x << ", y=" << y << endl;

    int arr1[] = {1, 2, 3};
    int arr2[] = {4, 5, 6};
    mySwap(arr1, arr2, 3);

    for (int i = 0; i < 3; i++) {
        cout << arr1[i] << " ";
    }
    cout << endl;

    return 0;
}
```

## 七、模板的局限性与解决方案

### 7.1 模板的局限性

```
模板不是万能的！某些类型的操作可能不支持。

例如：
1. 数组之间的赋值（不能直接 a = b）
2. 自定义类型之间的比较（需要重载运算符）
3. 某些类型不支持特定操作

解决方案：模板具体化（特化）
```

### 7.2 具体化（特化）模板

> **当模板无法满足某些类型的特殊需求时，可以为特定类型提供专门的实现，这叫做模板具体化（Specialization）。**

```cpp
#include <iostream>
#include <cstring>
using namespace std;

// 通用模板
template<typename T>
T myMax(T a, T b) {
    cout << "调用通用模板：";
    return (a > b) ? a : b;
}

// 具体化版本：为 const char* 类型提供特殊实现
// 因为 const char* 比较的是地址，不是字符串内容！
template<>
const char* myMax<const char*>(const char* a, const char* b) {
    cout << "调用具体化模板：";
    return (strcmp(a, b) > 0) ? a : b;
}

int main() {
    cout << myMax(10, 20) << endl;
    cout << myMax(3.14, 2.72) << endl;

    // 调用具体化模板（字符串比较内容而非地址）
    const char* s1 = "Hello";
    const char* s2 = "World";
    cout << myMax(s1, s2) << endl;

    return 0;
}
```

**运行结果：**
```
调用通用模板：20
调用通用模板：3.14
调用具体化模板：World
```

### 7.3 具体化的语法

```cpp
// 具体化模板的语法：
// template<> 返回值类型 函数名<具体类型>(参数列表) { ... }
// 注意：template后面是空的尖括号 <>

#include <iostream>
#include <string>
using namespace std;

// 通用模板：比较是否相等
template<typename T>
bool isEqual(T a, T b) {
    cout << "通用模板：";
    return a == b;
}

// 具体化：为 string 类型提供特殊实现（不区分大小写比较）
template<>
bool isEqual<string>(string a, string b) {
    cout << "具体化模板：";
    // 转换为小写后比较
    for (auto &c : a) c = tolower(c);
    for (auto &c : b) c = tolower(c);
    return a == b;
}

int main() {
    cout << boolalpha;
    cout << isEqual(10, 10) << endl;
    cout << isEqual(string("Hello"), string("hello")) << endl;
    return 0;
}
```

**运行结果：**
```
通用模板：true
具体化模板：true
```

## 八、模板的典型应用

```cpp
#include <iostream>
using namespace std;

// 应用1：通用排序函数（冒泡排序）
template<typename T>
void bubbleSort(T arr[], int size) {
    for (int i = 0; i < size - 1; i++) {
        for (int j = 0; j < size - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                T temp = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = temp;
            }
        }
    }
}

// 应用2：通用打印数组
template<typename T>
void printArray(T arr[], int size) {
    for (int i = 0; i < size; i++) {
        cout << arr[i];
        if (i < size - 1) cout << ", ";
    }
    cout << endl;
}

// 应用3：通用查找函数
template<typename T>
int findElement(T arr[], int size, T target) {
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) return i;
    }
    return -1;
}

int main() {
    int nums[] = {5, 3, 8, 1, 9, 2};
    int n = sizeof(nums) / sizeof(nums[0]);
    cout << "排序前："; printArray(nums, n);
    bubbleSort(nums, n);
    cout << "排序后："; printArray(nums, n);

    cout << "---" << endl;

    double vals[] = {3.14, 1.41, 2.72, 0.58};
    int m = sizeof(vals) / sizeof(vals[0]);
    cout << "排序前："; printArray(vals, m);
    bubbleSort(vals, m);
    cout << "排序后："; printArray(vals, m);

    cout << "---" << endl;

    cout << "查找8的位置：" << findElement(nums, n, 8) << endl;
    cout << "查找100的位置：" << findElement(nums, n, 100) << endl;

    return 0;
}
```

## 九、注意事项总结

| 序号 | 知识点 | 说明 |
|------|--------|------|
| 1 | `template<typename T>` | 声明模板，typename 可替换为 class |
| 2 | 自动推导 | 编译器根据实参推导 T 的类型 |
| 3 | 显式指定 | `func<类型>(参数)` 手动指定类型 |
| 4 | 推导必须一致 | 自动推导时所有 T 必须是同一类型 |
| 5 | 普通函数优先 | 普通函数完全匹配时优先于模板 |
| 6 | `<>` 强制调用模板 | `func<>(a, b)` 强制使用模板版本 |
| 7 | 具体化/特化 | `template<>` 为特定类型提供特殊实现 |
| 8 | 不能隐式转换 | 自动推导不会做隐式类型转换 |

## 十、练习题

1. 写一个函数模板 `myMin`，返回两个数中的较小值。
2. 写一个函数模板 `countEqual`，统计数组中等于某个值的元素个数。
3. 为 `const char*` 类型提供 `myMin` 的具体化版本，按字符串长度比较。

```cpp
// 练习1 参考答案：
template<typename T>
T myMin(T a, T b) {
    return (a < b) ? a : b;
}

// 练习2 参考答案：
template<typename T>
int countEqual(T arr[], int size, T target) {
    int count = 0;
    for (int i = 0; i < size; i++) {
        if (arr[i] == target) count++;
    }
    return count;
}

// 练习3 参考答案：
#include <cstring>
template<>
const char* myMin<const char*>(const char* a, const char* b) {
    return (strlen(a) < strlen(b)) ? a : b;
}
```
