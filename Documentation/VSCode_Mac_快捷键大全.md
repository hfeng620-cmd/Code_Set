# VS Code Mac 快捷键大全

## 基本编辑
- `Cmd + X` - 剪切当前行
- `Cmd + C` - 复制当前行
- `Cmd + V` - 粘贴
- `Cmd + Z` - 撤销
- `Cmd + Shift + Z` - 重做
- `Cmd + A` - 全选
- `Cmd + S` - 保存文件
- `Cmd + Shift + S` - 另存为
- `Cmd + W` - 关闭当前标签页
- `Cmd + K W` - 关闭所有标签页
- `Cmd + N` - 新建文件
- `Cmd + O` - 打开文件
- `Cmd + P` - 快速打开文件
- `Cmd + Shift + P` - 命令面板

## 选择和导航
- `Cmd + D` - 选择当前单词的下一个匹配项
- `Cmd + K Cmd + D` - 跳转到下一个差异
- `Cmd + Shift + L` - 选择所有匹配项
- `Cmd + F2` - 选择所有出现的当前单词
- `Cmd + G` - 跳转到下一行
- `Cmd + Shift + G` - 跳转到上一行
- `Cmd + L` - 选择当前行
- `Cmd + Shift + Left/Right` - 选择到行首/行尾
- `Cmd + Shift + Up/Down` - 选择到上一行/下一行

## 查找和替换
- `Cmd + F` - 当前文件内查找
- `Cmd + H` - 当前文件内替换
- `Cmd + Shift + F` - 全局查找
- `Cmd + Shift + H` - 全局替换
- `Alt + Enter` - 选择所有匹配项
- `F3` - 查找下一个
- `Shift + F3` - 查找上一个

## 代码编辑
- `Cmd + /` - 注释/取消注释行
- `Alt + Up/Down` - 向上/下移动行
- `Shift + Alt + Up/Down` - 向上/下复制行
- `Tab` - 缩进
- `Shift + Tab` - 减少缩进
- `Cmd + K Cmd + C` - 添加行注释
- `Cmd + K Cmd + U` - 移除行注释
- `Cmd + K Cmd + K` - 切换行注释

## 多光标操作
- `Cmd + Click` - 在点击位置添加光标
- `Cmd + Alt + Up/Down` - 在上一行/下一行添加光标
- `Cmd + Shift + L` - 在选中项的所有匹配处添加光标
- `Cmd + U` - 撤销上一个光标操作
- `Cmd + Alt + Left/Right` - 跳转到上一个/下一个光标位置

## 窗口和面板
- `Cmd + \` - 拆分编辑器
- `Cmd + 1/2/3` - 切换到第1/2/3个编辑器组
- `Cmd + K Cmd + Left/Right` - 在编辑器组间移动
- `Cmd + B` - 切换侧边栏显示/隐藏
- `Ctrl + `` ` `` - 切换终端
- `Ctrl + J` - 切换面板显示/隐藏
- `Ctrl + Shift + E` - 显示资源管理器
- `Ctrl + Shift + F` - 显示搜索
- `Ctrl + Shift + G` - 显示Git
- `Ctrl + Shift + D` - 显示调试
- `Ctrl + Shift + X` - 显示扩展

## 调试
- `F5` - 启动/继续调试
- `F9` - 切换断点
- `F10` - 单步跳过
- `F11` - 单步进入
- `Shift + F11` - 单步跳出
- `Shift + F5` - 停止调试
- `Ctrl + F5` - 启动不调试

## Git操作
- `Cmd + Shift + G` - 打开Git面板
- `Cmd + Enter` - 提交更改
- `Cmd + K Cmd + W` - 丢弃更改
- `Cmd + K P` - 推送到远程仓库
- `Cmd + K G` - 拉取远程仓库

## 终端
- `Ctrl + `` ` `` - 打开/关闭集成终端
- `Cmd + `` ` `` - 新建终端
- `Cmd + Shift + `` ` `` - 分割终端
- `Cmd + C` - 复制终端内容
- `Cmd + V` - 粘贴到终端

## 其他实用快捷键
- `Cmd + K Z` - 进入Zen模式
- `Cmd + K M` - 选择文件语言模式
- `Cmd + K L` - 格式化文档
- `Cmd + K F` - 格式化选中内容
- `Cmd + K B` - 切换面包屑
- `Cmd + K U` - 切换大小写
- `Cmd + K S` - 保存所有文件
- `Cmd + K Q` - 关闭快速提示
- `Cmd + K Cmd + X` - 删除行尾空格
- `Cmd + K Cmd + O` - 在新窗口中打开文件夹

## 导航和书签
- `Cmd + T` - 显示所有符号
- `Cmd + G` - 跳转到行
- `Cmd + Shift + O` - 跳转到符号
- `Cmd + Shift + :` - 跳转到行和列
- `F12` - 跳转到定义
- `Cmd + F12` - 跳转到实现
- `Alt + F12` - 显示定义预览
- `Shift + F12` - 查找所有引用
- `Cmd + K Cmd + I` - 显示悬停信息

- `F2` - 重命名符号
- `Cmd + .` - 快速修复
- `Cmd + Shift + K` - 删除行
- `Cmd + K Cmd + C` - 添加行注释
- `Cmd + K Cmd + U` - 移除行注释
- `Cmd + K Cmd + K` - 切换行注释
