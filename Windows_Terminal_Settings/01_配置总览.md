# Windows Terminal 配置总览

这套终端效果由 4 层组成：

1. Windows Terminal
用于窗口、标签页、透明度、配色方案、快捷键、默认启动的 Shell。

2. PowerShell 7
实际运行的终端 Shell。当前默认使用 `pwsh.exe -NoLogo`。

3. oh-my-posh
负责彩色分段提示符，也就是命令行左侧那条带颜色的 prompt。

4. Nerd Font
负责图标、菱形分隔符、Powerline 符号的正常显示。当前使用 `JetBrainsMono NFM`。

## 当前用到了什么

- 终端程序：Windows Terminal
- 默认 Shell：PowerShell 7
- Prompt 美化：oh-my-posh 29.10.0
- 字体：JetBrainsMono NFM
- Windows Terminal 配色方案：NeonNoir
- oh-my-posh 主题文件：`D:\OneDrive\文档\AutoHotkey\terminal\neon-synthwave.omp.json`
- PowerShell 配置文件：`C:\Users\Administrator\Documents\PowerShell\Microsoft.PowerShell_profile.ps1`
- Windows Terminal 设置文件：`C:\Users\Administrator\AppData\Local\Packages\Microsoft.WindowsTerminal_8wekyb3d8bbwe\LocalState\settings.json`

## 这套效果怎么弄出来

先装基础组件：

- 安装 Windows Terminal
- 安装 PowerShell 7
- 安装 oh-my-posh
- 安装 Nerd Font

再做 3 个配置：

1. 在 Windows Terminal 中把默认 Profile 设为 PowerShell 7。
2. 在 Windows Terminal 中设置字体、透明度、配色方案、快捷键。
3. 在 PowerShell 的 `$PROFILE` 中加载 oh-my-posh，并指定自定义主题文件。

## 当前关键配置点

### 1. Windows Terminal

- 默认 Profile：PowerShell 7
- 字体：`JetBrainsMono NFM`
- 字号：`12`
- 颜色方案：`NeonNoir`
- 透明度：`90`
- Acrylic：`true`
- 光标：`bar`
- 滚动条：隐藏
- 内边距：`12, 10, 12, 10`

### 2. PowerShell 7

PowerShell 启动后会加载 `$PROFILE`，里面初始化：

- PSReadLine 预测和按键
- 自定义函数 `ll`、`la`、`grep`
- 从系统代理同步到环境变量
- oh-my-posh prompt

### 3. oh-my-posh

当前使用自定义主题 `neon-synthwave.omp.json`，主要提供：

- 左侧 session 段
- OS 图标段
- 路径段
- Git 段
- 执行耗时段
- 管理员/root 段
- 状态段
- 右侧 shell 和时间段

### 4. 字体

之所以要 Nerd Font，是因为提示符里有这些图标和分隔符：

- Powerline 分隔符
- 文件夹图标
- Git 图标
- 状态图标
- Shell 图标

如果不用 Nerd Font，很多符号会显示成方块或者错位。

## 建议的恢复顺序

如果以后要在另一台机器复刻这套配置，按这个顺序最稳：

1. 安装 Windows Terminal
2. 安装 PowerShell 7
3. 安装 oh-my-posh
4. 安装 Nerd Font
5. 复制 `settings.json`
6. 复制 PowerShell profile
7. 复制 `neon-synthwave.omp.json`
8. 重开 Windows Terminal
