# VSCode Shortcut Notes

适用环境：VSCode 默认快捷键，Linux

如果以后忘了完整快捷键列表，直接按：

```text
Ctrl+K Ctrl+S
```

如果想看命令面板，直接按：

```text
Ctrl+Shift+P
```

## Vim Style Explorer Move

你当前机器上还自定义了资源管理器中的 `j/k`：

| 功能 | 快捷键 |
|---|---|
| 文件树中向下选择 | `j` |
| 文件树中向上选择 | `k` |

这两个键只会在左侧文件资源管理器聚焦时生效，不会影响你在编辑器里正常输入。

## Useful Notes

- 右上角没有三角按钮，不代表不能运行；`F5` 和 `Ctrl+F5` 更稳定。
- 不同扩展会新增自己的快捷键，但上面这些是最常用、最值得先记住的一批。
- 如果要查看全部快捷键，按 `Ctrl+K Ctrl+S`。
- 如果要执行任意命令，按 `Ctrl+Shift+P` 打开命令面板。
- 如果快捷键冲突，去 `Ctrl+K Ctrl+S` 搜索命令名，直接改。
